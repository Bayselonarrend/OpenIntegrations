# Прочитать строку 
 Читает данные из указанного соединения в виде строки



`Функция ПрочитатьСтроку(Знач Соединение, Знач Кодировка = "UTF-8", Знач Маркер = "", Знач Таймаут = 5000) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Соединение | --tcp | Произвольный | &#x2714; | Соединение, см. ОткрытьСоединение |
  | Кодировка | --enc | Строка | &#x2716; | Кодировка преобразования данных в строку |
  | Маркер | --marker | Строка, ДвоичныеДанные | &#x2716; | Маркер конца сообщения. Пусто > без маркера |
  | Таймаут | --timeout | Число | &#x2716; | Таймаут ожидания данных (мс). 0 > без ограничений |

  
  Возвращаемое значение:   Строка - Полученные данные в виде строки

<br/>

:::tip
При закрытии соединения, ошибке или обнаружении EOF чтение завершается в любом случае

Параметры с типом данных Двоичные данные могут принимать также пути к файлам на диске и URL
:::
