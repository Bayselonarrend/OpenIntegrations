# Открыть соединение 
 Создает TCP соединение



`Функция ОткрытьСоединение(Знач Адрес) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Адрес | --address | Строка | &#x2714; | Адрес и порт для подключения |

  
  Возвращаемое значение:   Неопределено, Произвольный -  Возвращает объект TCP клиента при успешном подключении или неопределено

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Адрес      = "45.79.112.203:4242";
    Соединение = OPI_TCP.ОткрытьСоединение(Адрес);

    OPI_TCP.ЗакрытьСоединение(Соединение);
```
    
:::caution
**NOCLI!:** данный метод недоступен для использования в CLI версии
:::
<br/>
