# Закрыть соединение 
 Явно закрывает созданное ранее соединение



`Функция ЗакрытьСоединение(Знач Соединение) ЭкспортВозврат Соединение.Disconnect();КонецФункции`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Соединение | --tcp | Произвольный | &#x2714; | Соединение, см. ОткрытьСоединение |

  
  Возвращаемое значение:   Булево - всегда возвращает Истина

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Адрес      = "45.79.112.203:4242";
    Соединение = OPI_TCP.ОткрытьСоединение(Адрес);

    Результат = OPI_TCP.ЗакрытьСоединение(Соединение);
```
    



```json title="Результат"
true
```
