# Отправить строку 
 Отправляет данные в виде строки через указанное соединение



`Функция ОтправитьСтроку(Знач Соединение, Знач Данные, Знач Кодировка = "UTF-8", Знач Таймаут = 5000) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Соединение | - | Произвольный | &#x2714; | Соединение, см. ОткрытьСоединение |
  | Данные | - | Строка | &#x2714; | Данные для отправки в виде строки |
  | Кодировка | - | Строка | &#x2716; | Кодировка для записи исходящей строки в поток |
  | Таймаут | - | Число | &#x2716; | Таймаут ожидания записи (мс). 0 > без ограничений |

  
  Возвращаемое значение:   Булево - Признак успешного выполнения

<br/>


:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
