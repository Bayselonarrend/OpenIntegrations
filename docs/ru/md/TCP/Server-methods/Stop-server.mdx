# Остановить сервер 
 Завершает работу сервера



`Функция ОстановитьСервер(Знач ОбъектСервера) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ОбъектСервера | - | Произвольный | &#x2714; | Объект компоненты запущенного сервера |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - Результат выполнения


:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Порт   = 9877;
    Сервер = OPI_TCP.ЗапуститьСервер(Порт);

    Результат = OPI_TCP.ОстановитьСервер(Сервер);
```
