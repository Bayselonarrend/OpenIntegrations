# Закрыть входящее соединение 
 Закрывает входящее соединение сервера по ID



`Функция ЗакрытьВходящееСоединение(Знач ОбъектСервера, Знач IDСоединения) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ОбъектСервера | - | Произвольный | &#x2714; | Объект компоненты запущенного сервера |
  | IDСоединения | - | Строка | &#x2714; | Идентификатор соединения |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - Результат выполнения


:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Порт   = 9881;
    Сервер = OPI_TCP.ЗапуститьСервер(Порт);

    // Подключаем клиента
    Адрес  = "127.0.0.1:" + Порт;
    Клиент = OPI_TCP.ОткрытьСоединение(Адрес);

    Если Не OPI_TCP.ЭтоОбъектКлиента(Клиент) Тогда
        ВызватьИсключение "Не удалось подключиться к серверу";
    КонецЕсли;

    Сообщение = "Hello!" + Символы.ПС;

    OPI_TCP.ОтправитьСтроку(Клиент, Сообщение);

    // Получаем ID соединения
    ПервоеСообщение = OPI_TCP.ПолучитьДанныеОчередногоСоединения(Сервер, 5000);
    IDСоединения    = ПервоеСообщение["connectionId"];

    // Закрываем соединение со стороны сервера
    Результат = OPI_TCP.ЗакрытьВходящееСоединение(Сервер, IDСоединения);
```
