# Это объект сервера 
 Проверяет, что значение является объектом внешней компоненты сервера TCP



`Функция ЭтоОбъектСервера(Знач Значение) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Значение | - | Произвольный | &#x2714; | Значение для проверки |

  
  Возвращаемое значение:   Булево -  Это коннектор


:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Порт   = 9884;
    Сервер = OPI_TCP.ЗапуститьСервер(Порт);

    Результат = OPI_TCP.ЭтоОбъектСервера(Сервер);
```
    



```json title="Результат"
true
```
