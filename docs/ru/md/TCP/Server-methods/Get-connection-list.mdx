# Получить список соединений 
 Получает список соединений



`Функция ПолучитьСписокСоединений(Знач ОбъектСервера) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ОбъектСервера | - | Произвольный | &#x2714; | Объект компоненты запущенного сервера |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - Результат выполнения

:::tip
В списке выводятся активные соединения и закрытые соединения с необработанными данными в буфере
:::

:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Порт   = 9883;
    Сервер = OPI_TCP.ЗапуститьСервер(Порт);

    // Подключаем несколько клиентов
    Адрес   = "127.0.0.1:" + Порт;
    Клиент1 = OPI_TCP.ОткрытьСоединение(Адрес);
    Клиент2 = OPI_TCP.ОткрытьСоединение(Адрес);

    Если Не OPI_TCP.ЭтоОбъектКлиента(Клиент1) Или Не OPI_TCP.ЭтоОбъектКлиента(Клиент2) Тогда
        ВызватьИсключение "Не удалось подключиться к серверу";
    КонецЕсли;

    OPI_TCP.ОтправитьСтроку(Клиент1, "Client 1" + Символы.ПС);
    OPI_TCP.ОтправитьСтроку(Клиент2, "Client 2" + Символы.ПС);

    // Получаем сообщения для инициализации соединений
    OPI_TCP.ПолучитьДанныеОчередногоСоединения(Сервер, 5000);
    OPI_TCP.ПолучитьДанныеОчередногоСоединения(Сервер, 5000);

    // Получаем список соединений
    Результат = OPI_TCP.ПолучитьСписокСоединений(Сервер);
```
