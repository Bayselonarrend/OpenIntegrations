# Завершить получение 
 Закрывает получение данных для указанного соединения



`Функция ЗавершитьПолучение(Знач ОбъектСервера, Знач IDСоединения) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ОбъектСервера | - | Произвольный | &#x2714; | Объект компоненты запущенного сервера |
  | IDСоединения | - | Строка | &#x2714; | Идентификатор соединения |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - Результат выполнения


:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ПортЗапуска   = 9877;
    ОбъектСервера = OPI_TCP.ЗапуститьСервер(ПортЗапуска);

    // Подключаемся к запущенному серверу
    АдресПодключения  = "127.0.0.1:9877";
    ОбъектКлиента     = OPI_TCP.ОткрытьСоединение(АдресПодключения);
    Если Не OPI_TCP.ЭтоОбъектКлиента(ОбъектКлиента) Тогда
        ВызватьИсключение OPI_Инструменты.JSONСтрокой(ОбъектКлиента);
    КонецЕсли;

    АктивныеСоединения = OPI_TCP.ПолучитьСписокСоединений(ОбъектСервера);
    IDСоединения    = АктивныеСоединения["connections"][0]["connectionId"];

    // Завершаем получение со стороны сервера
    Результат = OPI_TCP.ЗавершитьПолучение(ОбъектСервера, IDСоединения);
```
