# Открыть соединение 
 Создает TCP соединение



`Функция ОткрытьСоединение(Знач Адрес, Знач Tls = "", Знач Прокси = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Адрес | - | Строка | &#x2714; | Адрес и порт для подключения |
  | Tls | - | Структура Из КлючИЗначение | &#x2716; | Настройки TLS, если необходимо. См. [`ПолучитьНастройкиTls`](/docs/TCP/Client-methods/Get-tls-settings) |
  | Прокси | - | Структура Из КлючИЗначение | &#x2716; | Настройки прокси, если необходимо. См. [`ПолучитьНастройкиПрокси`](/docs/TCP/Client-methods/Get-proxy-settings) |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение, Произвольный -  Возвращает объект TCP клиента при успешном подключении или информацию об ошибке


:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Адрес      = "45.79.112.203:4242";
    Соединение = OPI_TCP.ОткрытьСоединение(Адрес);

    OPI_TCP.ЗакрытьСоединение(Соединение);

    Адрес = "tcpbin.com:4243";
    Tls   = OPI_TCP.ПолучитьНастройкиTLS(Истина);

    ПроксиПользователь = "proxyuser";
    ПроксиПароль       = "12we...";
    ПроксиАдрес        = "127.0.0.1";
    ПроксиПорт         = "1080";

    Прокси = OPI_TCP.ПолучитьНастройкиПрокси(ПроксиАдрес, ПроксиПорт, "socks5", ПроксиПользователь, ПроксиПароль);
    Tls    = OPI_TCP.ПолучитьНастройкиTLS(Истина);

    Соединение = OPI_TCP.ОткрытьСоединение(Адрес, TLS, Прокси);
```
    



