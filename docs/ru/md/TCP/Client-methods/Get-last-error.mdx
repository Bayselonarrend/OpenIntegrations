# Получить последнюю ошибку 
 Получает информацию о последней ошибке в соединении



`Функция ПолучитьПоследнююОшибку(Знач Соединение) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Соединение | - | Произвольный | &#x2714; | Соединение, См. [`ОткрытьСоединение`](/docs/TCP/Client-methods/Create-connection) |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение, Неопределено -  Информация об ошибке или неопределено, если ошибки нет


:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Адрес      = "45.79.112.203:4242";
    Соединение = OPI_TCP.ОткрытьСоединение(Адрес);
    Данные     = "Hello server!" + Символы.ПС;

    Отправка  = OPI_TCP.ОтправитьСтроку(Соединение, Данные);
```
    



```json title="Результат"
null
```
