# Получить последнюю ошибку
 Получает информацию о последней ошибке в соединении



`Функция ПолучитьПоследнююОшибку(Знач Соединение) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Соединение | --tcp | Произвольный | &#x2714; | Соединение, см. ОткрытьСоединение |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение, Неопределено -  Информация об ошибке или неопределено, если ошибки нет
