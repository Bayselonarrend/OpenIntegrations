# Прочитать строку 
 Читает данные из указанного соединения в виде строки



`Функция ПрочитатьСтроку(Знач Соединение, Знач Кодировка = "UTF-8", Знач Маркер = "", Знач Таймаут = 5000) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Соединение | - | Произвольный | &#x2714; | Соединение, См. [`ОткрытьСоединение`](/docs/TCP/Client-methods/Create-connection) |
  | Кодировка | - | Строка | &#x2716; | Кодировка преобразования данных в строку |
  | Маркер | - | Строка, ДвоичныеДанные | &#x2716; | Маркер конца сообщения. Пусто > без маркера |
  | Таймаут | - | Число | &#x2716; | Таймаут ожидания данных (мс). 0 > без ограничений |

  
  Возвращаемое значение:   Строка - Полученные данные в виде строки

:::tip
При закрытии соединения, ошибке или обнаружении EOF чтение завершается в любом случае

Параметры с типом данных Двоичные данные могут принимать также пути к файлам на диске и URL
:::

:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Адрес      = "45.79.112.203:4242";
    Соединение = OPI_TCP.ОткрытьСоединение(Адрес);
    Данные     = "Hello server!" + Символы.ПС;

    OPI_TCP.ОтправитьСтроку(Соединение, Данные);

    // Маркер конца сообщения для исключения ожидания конца таймаута
    Маркер    = Символы.ПС;
    Результат = OPI_TCP.ПрочитатьСтроку(Соединение, , Маркер);
    // ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

    OPI_TCP.ЗакрытьСоединение(Соединение);
```
    



```json title="Результат"
"Hello server!\n"
```
