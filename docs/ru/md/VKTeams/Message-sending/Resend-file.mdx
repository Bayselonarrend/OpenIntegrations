# Переслать файл
 Отправляет ранее загруженный файл по его ID



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПереслатьФайл(Знач Токен, Знач IDЧата, Знач IDФайла, Знач Текст = "", Знач Разметка = "MarkdownV2") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | IDЧата | --chatid | Строка, Число | &#x2714; | ID чата для отправки файла |
  | IDФайла | --fileid | Строка, Число | &#x2714; | ID Файла для отправки |
  | Текст | --text | Строка | &#x2716; | Подпись к файлу |
  | Разметка | --parsemod | Строка | &#x2716; | Вид разметки для текста сообщения: MarkdownV2 или HTML |


<div className="return-value-note">

  </div>

  | Параметр | Описание |
  |---|---|
  | proxy | ИнтернетПрокси или структура полей Протокол, Сервер, Порт, Пользователь, Пароль, ИспользоватьАутентификациюОС |
  | timeout | Таймаут выполнения запроса |
  | adv_response | Оформляет ответ в виде полной структуры с полями code, body и headers |


  </TabItem>
</Tabs>

:::tip
Метод в документации API: [GET /messages/sendFile](https://teams.vk.com/botapi/#/messages/get_messages_sendFile)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен   = "001.3501506236.091...";
    IDЧата  = "AoLI0egLWBSLR1Ngn2w";
    IDФайла = "f7ey3o4dcncSTDBRC7c3fM6904a4ed1bd";
    Текст   = "Подпись к файлу";

    Результат = OPI_VKTeams.ПереслатьФайл(Токен, IDЧата, IDФайла, Текст);
```
