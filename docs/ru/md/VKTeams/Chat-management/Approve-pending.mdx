# Одобрить заявку на вступление
 Подтверждает вступление пользователя в закрытый чат



`Функция ОдобритьЗаявкуНаВступление(Знач Токен, Знач IDЧата, Знач IDПользователя = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | IDЧата | --chatid | Строка, Число | &#x2714; | ID чата |
  | IDПользователя | --userid | Строка, Число | &#x2716; | ID пользователя. Ответ на все заявки, если не заполнено |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от VK Teams

<br/>

:::tip
Для вызова этого метода бот должен быть администратором в чате.<br/>
Метод в документации API: [GET ​​/chats/resolvePending](https://teams.vk.com/botapi/#/chats/get_chats_resolvePending)<br/>
<br/>
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен        = "001.3501506236.091...";
    IDЧата       = "689203963@chat.agent";
    Пользователь = 1011987091;

    Результат = OPI_VKTeams.ОдобритьЗаявкуНаВступление(Токен, IDЧата, Пользователь);

    Результат = OPI_VKTeams.ОдобритьЗаявкуНаВступление(Токен, IDЧата);
```
