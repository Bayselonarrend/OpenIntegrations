# Загрузить коды активации товара
 Загружает коды активации для цифрового товара или услуги



`Функция ЗагрузитьКодыАктивацииТовара(Знач IDКлиента, Знач КлючAPI, Знач IDТовара, Знач Коды) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | IDКлиента | --clientid | Строка | &#x2714; | Идентификатор клиента |
  | КлючAPI | --apikey | Строка | &#x2714; | API ключ |
  | IDТовара | --productid | Число | &#x2714; | ID товара |
  | Коды | --codes | Строка, Массив Из Строка | &#x2714; | Коды активации цифрового товара |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Ozon Seller API

<br/>

:::tip
Код активации привязывается к карточке цифрового товара<br/>
Метод проверки статуса добавления - ПолучитьСтатусДобавленияКодов()<br/>
Метод в документации API: [post /v1/product/upload_digital_codes](https://docs.ozon.ru/api/seller/#operation/ProductAPI_UploadDigitalCode)<br/>
<br/>
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    IDКлиента = "2128753";
    КлючAPI   = "7cc90d26-33e4-499b...";
    IDТовара  = "1235184552";

    Коды = Новый Массив;
    Коды.Добавить("11111111");
    Коды.Добавить("22222222");

    Результат = OPI_Ozon.ЗагрузитьКодыАктивацииТовара(IDКлиента, КлючAPI, IDТовара, Коды);
```
