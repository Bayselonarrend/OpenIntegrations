# Изменить тему форума
 Создает новую тему в группе с включенным функционалом тем



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ИзменитьТемуФорума(Знач Токен, Знач IDЧата, Знач IDТемы, Знач Заголовок = Неопределено, Знач IDИконки = Неопределено) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | IDЧата | --forum | Строка, Число | &#x2714; | ID чата создания темы |
  | IDТемы | --topic | Строка, Число | &#x2714; | ID темы |
  | Заголовок | --title | Строка | &#x2716; | Новый заголовок |
  | IDИконки | --icon | Строка | &#x2716; |  См. [`ПолучитьСписокИконокАватаров`](/docs/Telegram/Forum-topic-management/Get-avatar-icon-list) |

  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

*Для текущего метода нет дополнительных параметров расширенного вызова.*


  </TabItem>
</Tabs>
  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Telegram

:::tip
Метод в документации API: [editForumTopic](https://core.telegram.org/bots/api#editforumtopic)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен       = "6129457865:AAFyzNYOAFbu...";
    Чат         = "-1001971186208";
    Тема        = "12491";
    НовоеИмя    = "Новый тестовый заголовок";
    НоваяИконка = "5310132165583840589";

    Результат = OPI_Telegram.ИзменитьТемуФорума(Токен, Чат, Тема, НовоеИмя, НоваяИконка);
```
