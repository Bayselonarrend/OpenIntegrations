# Закрыть тему форума
 Закрывает тему для новых сообщений



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ЗакрытьТемуФорума(Знач Токен, Знач IDЧата, Знач IDТемы = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | IDЧата | --forum | Строка, Число | &#x2714; | ID чата темы |
  | IDТемы | --topic | Строка, Число | &#x2716; | ID темы |


<div className="return-value-note">
