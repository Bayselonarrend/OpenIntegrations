# Открепить сообщение
 Открепляет сообщение в шапке чата



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ОткрепитьСообщение(Знач Токен, Знач IDЧата, Знач IDСообщения) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | IDЧата | --chat | Строка, Число | &#x2714; | ID целевого чата |
  | IDСообщения | --message | Строка, Число | &#x2714; | ID целевого сообщения |

  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | proxy | ИнтернетПрокси или структура полей Протокол, Сервер, Порт, Пользователь, Пароль, ИспользоватьАутентификациюОС |
  | timeout | Таймаут выполнения запроса |
  | adv_response | Оформляет ответ в виде полной структуры с полями code, body и headers |


  </TabItem>
</Tabs>
  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Telegram

:::tip
Метод в документации API: [unpinChatMessage](https://core.telegram.org/bots/api#unpinchatmessage)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен       = "6129457865:AAFyzNYOAFbu...";
    IDКанала    = "@testsichee";
    IDСообщения = "22536";

    Результат = OPI_Telegram.ОткрепитьСообщение(Токен, IDКанала, IDСообщения);
```
