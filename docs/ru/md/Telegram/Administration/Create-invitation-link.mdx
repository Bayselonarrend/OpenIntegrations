# Создать ссылку-приглашение
 Создает ссылку для вступления в закрытый чат



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция СоздатьСсылкуПриглашение(Знач Токен, Знач IDЧата, Знач Заголовок = "", Знач ДатаИстечения = "", Знач ЛимитПользователей = 0) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | IDЧата | --chat | Строка, Число | &#x2714; | ID целевого чата или IDЧата*IDТемы |
  | Заголовок | --title | Строка | &#x2716; | Заголовок приглашения |
  | ДатаИстечения | --expire | Дата | &#x2716; | Дата окончания жизни ссылки (бессрочно, если не указано) |
  | ЛимитПользователей | --limit | Число | &#x2716; | Лимит пользователей (бесконечно, если не указано) |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - сериализованный JSON ответа от Telegram
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | proxy | ИнтернетПрокси или структура полей Протокол, Сервер, Порт, Пользователь, Пароль, ИспользоватьАутентификациюОС |
  | timeout | Таймаут выполнения запроса |
  | adv_response | Оформляет ответ в виде полной структуры с полями code, body и headers |
  | retries | Число попыток отправки при коде `5**` или внутренних ошибках клиента |
  | dontwait | Создает фоновое задание и возвращает его объект (только для 1С и OneScript) |


  </TabItem>
</Tabs>

:::tip
Метод в документации API: [createChatInviteLink](https://core.telegram.org/bots/api#createchatinvitelink)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен       = "6129457865:AAFyzNYOAFbu...";
    IDКанала    = "@testsichee";
    Сутки       = 86400;
    ТекущаяДата = OPI_Инструменты.ПолучитьТекущуюДату();

    Заголовок     = "Ссылка " + Строка(ТекущаяДата);
    Истечение     = ТекущаяДата + Сутки;
    UnixИстечение = OPI_Инструменты.UNIXTime(Истечение);

    Результат = OPI_Telegram.СоздатьСсылкуПриглашение(Токен, IDКанала, Заголовок, Истечение, 200);
```
