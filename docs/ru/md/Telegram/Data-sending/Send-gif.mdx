# Отправить гифку
 Отправляет гифку в чат или канал



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ОтправитьГифку(Знач Токен, Знач IDЧата, Знач Текст, Знач Гифка, Знач Клавиатура = "", Знач Разметка = "Markdown") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | IDЧата | --chat | Строка, Число | &#x2714; | ID целевого чата или IDЧата*IDТемы |
  | Текст | --text | Строка | &#x2714; | Текст сообщения |
  | Гифка | --gif | ДвоичныеДанные, Строка | &#x2714; | Файл гифки |
  | Клавиатура | --keyboard | Строка, Структура Из КлючИЗначение | &#x2716; | Клавиатура. См. [`СформироватьКлавиатуруПоМассивуКнопок`](/docs/Telegram/Data-sending/Form-keyboard-from-button-array) |
  | Разметка | --parsemode | Строка | &#x2716; | Вид обработки текста (HTML, Markdown, MarkdownV2) |

Метод в документации API: [sendAnimation](https://core.telegram.org/bots/api#sendanimation)

Параметры с типом данных Двоичные данные могут принимать также пути к файлам на диске и URL
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен    = "6129457865:AAFyzNYOAFbu...";
    IDЧата   = "461699897";
    IDКанала = "@testsichee";
    Текст    = "Строковое значение";
    Гифка    = "https://hut.openintegrations.dev/test_data/animation.gif";

    ГифкаПуть = ПолучитьИмяВременногоФайла("gif");
    КопироватьФайл(Гифка, ГифкаПуть);

    ГифкаДД = Новый ДвоичныеДанные(ГифкаПуть);

    // В чат, по URL
    Результат = OPI_Telegram.ОтправитьГифку(Токен, IDЧата, Текст, Гифка);

    // В канал, на диске
    Результат = OPI_Telegram.ОтправитьГифку(Токен, IDКанала, Текст, ГифкаПуть);

    // В канал, из двоичных данных
    Результат = OPI_Telegram.ОтправитьГифку(Токен, IDКанала, Текст, ГифкаДД);
```
