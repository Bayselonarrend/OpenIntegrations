# Переслать сообщение
 Пересылает сообщение между чатами или в рамках одного чата



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПереслатьСообщение(Знач Токен, Знач IDОригинала, Знач ОткудаID, Знач КудаID) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | IDОригинала | --message | Строка, Число | &#x2714; | ID оригинального сообщения |
  | ОткудаID | --from | Строка, Число | &#x2714; | ID чата оригинального сообщения |
  | КудаID | --to | Строка, Число | &#x2714; | ID целевого чата или IDЧата*IDТемы |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - сериализованный JSON ответа от Telegram
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | proxy | ИнтернетПрокси или структура полей Протокол, Сервер, Порт, Пользователь, Пароль, ИспользоватьАутентификациюОС |
  | timeout | Таймаут выполнения запроса |
  | adv_response | Оформляет ответ в виде полной структуры с полями code, body и headers |


  </TabItem>
</Tabs>

:::tip
Метод в документации API: [forwardMessage](https://core.telegram.org/bots/api#forwardmessage)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен       = "6129457865:AAFyzNYOAFbu...";
    IDЧата      = "461699897";
    IDКанала    = "@testsichee";
    IDСообщения = "22536";

    Результат = OPI_Telegram.ПереслатьСообщение(Токен, IDСообщения, IDКанала, IDЧата);
```
