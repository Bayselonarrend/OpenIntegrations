# Скачать файл
 Получает файл по ID



`Функция СкачатьФайл(Знач Токен, Знач Идентификатор, Знач ПутьСохранения = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Идентификатор | --object | Строка | &#x2714; | Идентификатор файла |
  | ПутьСохранения | --out | Строка | &#x2716; | Путь сохранения файла |

  
  Возвращаемое значение:   ДвоичныеДанные,Строка - Двоичные данные или путь к файлу при указании параметра ПутьСохранения

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен         = "ya29.a0AZYkNZiCaf3vW8I05cl_i6pLKCRWSjff1YhC7Yi0SgLO50zysjwrX_9QPi1WejmqgGdFjcbaOJKsPO3tvT7gB712mg1gxgxiL1pCUVthx...";
    Идентификатор = "1CK_FNwlq0eNfmse2YyeXaNStG80Lu2wI";

    Результат = OPI_GoogleDrive.СкачатьФайл(Токен, Идентификатор);
```
