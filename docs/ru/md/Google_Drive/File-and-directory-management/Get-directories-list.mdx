# Получить список каталогов
 Получает список каталогов диска



`Функция ПолучитьСписокКаталогов(Знач Токен, Знач ИмяСодержит = "", Знач Подробно = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | ИмяСодержит | --querry | Строка | &#x2716; | Отбор по имени |
  | Подробно | --depth | Булево | &#x2716; | Добавляет список файлов к полям каталога |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - Массив соответствий каталогов

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Имя   = "Тестовая папка";
    Токен = "ya29.a0AQQ_BDTgQbn9c5rXhPK3FCakms_7oKkdIWE1CCQFnSPftb2RxvloVqnYOcnsAKhXrD0Gq8Tz2Kgn5IBXG5E25uksZfvvTtXob-jFcuj_9LZYs9La4HrcFk766...";

    Результат = OPI_GoogleDrive.ПолучитьСписокКаталогов(Токен, Имя, Истина);
```
    

 <Tabs>
  
    <TabItem value="bash" label="Bash" default>
        ```bash
            oint gdrive ПолучитьСписокКаталогов \
              --token "***" \
              --querry "Тестовая папка" \
              --depth true
        ```
    </TabItem>
  
    <TabItem value="bat" label="CMD/Bat" default>
        ```batch
            oint gdrive ПолучитьСписокКаталогов ^
              --token "***" ^
              --querry "Тестовая папка" ^
              --depth true
        ```
    </TabItem>
