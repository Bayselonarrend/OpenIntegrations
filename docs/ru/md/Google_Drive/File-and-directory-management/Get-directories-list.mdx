# Получить список каталогов
 Получает список каталогов диска



`Функция ПолучитьСписокКаталогов(Знач Токен, Знач ИмяСодержит = "", Знач Подробно = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | ИмяСодержит | --querry | Строка | &#x2716; | Отбор по имени |
  | Подробно | --depth | Булево | &#x2716; | Добавляет список файлов к полям каталога |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - Массив соответствий каталогов

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Имя   = "Тестовая папка";
    Токен = "ya29.a0AS3H6Nw6rTG3pFXv17ZLXkvuvtNl_DHHXbxe2yDRIB09Sq-BA7xPQj9V7OKAaAiHRZAsVqI0UD6026Q_3gklXjU-kam0kzFCe2adXI3lS...";

    Результат = OPI_GoogleDrive.ПолучитьСписокКаталогов(Токен, Имя, Истина);
```
