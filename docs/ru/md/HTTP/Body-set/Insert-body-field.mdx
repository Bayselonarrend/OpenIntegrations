# Вставить поле тела 
 Вставляет новое поле в коллекцию Form или Json тела



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ВставитьПолеТела(Знач КлючПоля, Знач ЗначениеПоля, Знач ТипЗначения = Неопределено) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | КлючПоля | - | Строка | &#x2714; | Ключ вставляемого поля |
  | ЗначениеПоля | - | Произвольный | &#x2714; | Значение поля |
  | ТипЗначения | - | Строка | &#x2716; | Имя типа для попытки автоматического приведения |

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL = "https://bin.openintegrations.dev";
    URL = URL + "/post";

    Результат = OPI_ЗапросыHTTP.НовыйЗапрос()
        .Инициализировать(URL)
        .ИнициализироватьJsonТело()
        .ВставитьПолеТела("Поле1", 10) // <---
        .ВставитьПолеТела("Поле2", "Текст")
        .ОбработатьЗапрос("POST")
        .ВернутьОтветКакJSONКоллекцию();
```
