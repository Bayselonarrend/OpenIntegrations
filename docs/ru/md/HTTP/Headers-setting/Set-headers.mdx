# Установить заголовки 
 Устанавливает коллекцию заголовков запросов



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция УстановитьЗаголовки(Знач Значение, Знач ПолнаяЗамена = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Значение | - | Произвольный | &#x2714; | Структура или соответствие заголовков запроса |
  | ПолнаяЗамена | - | Булево | &#x2716; | Очищает все добавленные ранее заголовки перед установкой |

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL = "https://bin.openintegrations.dev";
    URL = URL + "/get";

    Заголовки = Новый Соответствие;
    Заголовки.Вставить("X-Header1", "Value1");
    Заголовки.Вставить("X-Header2", "Value2");

    Результат = OPI_ЗапросыHTTP.НовыйЗапрос()
        .Инициализировать()
        .УстановитьURL(URL)
        .УстановитьЗаголовки(Заголовки) // <---
        .ОбработатьЗапрос("GET")
        .ВернутьОтветКакJSONКоллекцию();
```
