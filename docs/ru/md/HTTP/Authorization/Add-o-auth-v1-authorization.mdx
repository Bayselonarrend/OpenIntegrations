# Добавить OAuth v1 авторизацию 
 Добавляет данные для OAuth v1 авторизации



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ДобавитьOAuthV1Авторизацию(Знач Token, Знач Secret, Знач ConsumerKey, Знач ConsumerSecret, Знач Версия) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Token | - | Строка | &#x2714; | Token для авторизации |
  | Secret | - | Строка | &#x2714; | Secret для авторизации |
  | ConsumerKey | - | Строка | &#x2714; | Consumer key для авторизации |
  | ConsumerSecret | - | Строка | &#x2714; | Consumer secret для авторизации |
  | Версия | - | Строка | &#x2714; | Версия API |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  ОбработкаОбъект.OPI_HTTPКлиент -  Этот же объект обработки
  </div>
</div>

  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

*Для текущего метода нет дополнительных параметров расширенного вызова.*


  </TabItem>
</Tabs>

:::tip
По умолчанию, для создания подписи используется HMAC-SHA256. Для изменения алгоритма можно воспользоваться функцией `УстановитьАлгоритмOAuthV1`
:::

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL = "https://bin.openintegrations.dev";
    URL = URL + "/get";

    Токен              = "***";
    Секрет             = "***";
    КлючПользователя   = "***";
    СекретПользователя = "***";
    Версия             = "1.0";

    Результат = OPI_ЗапросыHTTP.НовыйЗапрос()
        .Инициализировать(URL)
        .ДобавитьOAuthV1Авторизацию(Токен, Секрет, КлючПользователя, СекретПользователя, Версия) // <---
        .ОбработатьЗапрос("GET")
        .ВернутьОтветКакJSONКоллекцию();
```
