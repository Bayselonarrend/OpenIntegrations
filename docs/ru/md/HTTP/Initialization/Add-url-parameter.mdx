# Добавить параметр URL 
 Добавляет query параметр URL в запрос



`Функция ДобавитьПараметрURL(Знач Имя, Знач Значение, Знач ИгнорироватьПустой = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Имя | - | Строка | &#x2714; | Ключ параметра запроса |
  | Значение | - | Произвольный | &#x2714; | Значение параметра запроса |
  | ИгнорироватьПустой | - | Булево | &#x2716; | Не добавлять параметр, если передано пустое значение |

  
  Возвращаемое значение:   ОбработкаОбъект.OPI_HTTPКлиент -  Этот же объект обработки

:::tip
Значение параметра будет преобразовано в строку
:::

:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
