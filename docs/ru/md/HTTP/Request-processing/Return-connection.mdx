# Вернуть соединение 
 Возвращает объект текущего HTTP соединения



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ВернутьСоединение(Принудительно = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Принудительно | - | Булево | &#x2716; | Ложь > Вместо соединения будет возвращен объект обработки, если в ней были ошибки |
