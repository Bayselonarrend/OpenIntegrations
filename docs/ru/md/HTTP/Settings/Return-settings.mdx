# Вернуть настройки




`Функция ВернутьНастройки(Знач Отбор = Неопределено) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Отбор | --filter | Строка, Массив Из Строка | &#x2716; | Отбор по имени или именам настроек для получения |

  
  Возвращаемое значение:   Произвольный - Структура настроек, значение конкретной настройки или объект обработки (при ошибке)



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Результат = OPI_ЗапросыHTTP.НовыйЗапрос()
        .Инициализировать()
        .ВернутьНастройки();
```
