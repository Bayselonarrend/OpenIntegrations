# Использовать сжатие gzip 
 Включает или отключает заголовок получения данных от сервера в сжатом виде



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ИспользоватьСжатиеGzip(Знач Флаг) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Флаг | - | Булево | &#x2714; | Признак использования gzip |

  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

*Для текущего метода нет дополнительных параметров расширенного вызова.*


  </TabItem>
</Tabs>
  
  Возвращаемое значение:   ОбработкаОбъект.OPI_HTTPКлиент -  Этот же объект обработки

:::tip
По умолчанию ответ от сервера запрашивается со сжатием gzip
:::

:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL = "https://bin.openintegrations.dev";
    URL = URL + "/post";

    Картинка = "https://hut.openintegrations.dev/test_data/picture.jpg"; // URL, Путь или Двоичные данные

    Результат = OPI_ЗапросыHTTP.НовыйЗапрос()
        .Инициализировать(URL)
        .УстановитьДвоичноеТело(Картинка)
        .ИспользоватьСжатиеGzip(Ложь) // <---
        .ОбработатьЗапрос("POST", Ложь)
        .ВернутьЗапрос();
```
    



```json title="Результат"
НЕ JSON: HTTPЗапрос
```
