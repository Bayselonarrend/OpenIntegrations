# Копировать модель
 Копирует существующую модель



`Функция КопироватьМодель(Знач URL, Знач Модель, Знач Имя, Знач ДопЗаголовки = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера Ollama |
  | Модель | --model | Строка | &#x2714; | Имя существующей модели |
  | Имя | --name | Строка | &#x2714; | Имя новой модели |
  | ДопЗаголовки | --headers | Соответствие Из КлючИЗначение | &#x2716; | Доп заголовки запроса, если необходимо |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - Результат обработки

<br/>

:::tip
Метод в документации API: [Copy a Model](https://github.com/ollama/ollama/blob/main/docs/api.md#copy-a-model)
:::
<br/>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL   = "https://api.athenaeum.digital/ollama";
    Токен = "10KO..."; // Авторизация - не часть API Ollama

    Модель = "mario";
    Имя    = "mario2";

    ДопЗаголовки = Новый Соответствие;
    ДопЗаголовки.Вставить("Authorization", СтрШаблон("Bearer %1", Токен));

    Результат = OPI_Ollama.КопироватьМодель(URL, Модель, Имя, ДопЗаголовки);
```
