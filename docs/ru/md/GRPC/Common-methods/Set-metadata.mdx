# Установить метаданные 
 Устанавливает новые метаданные соединения



`Функция УстановитьМетаданные(Знач Соединение, Знач Метаданные) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Соединение | - | Произвольный | &#x2714; | Существующее соединение |
  | Метаданные | - | Структура Из КлючИЗначение | &#x2714; | Список метаданных |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - Результат обработки

:::tip
Заменяет метаданные, установленные при создании соединения
:::

:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
