# Открыть соединение 
 Открывает gRPC соединение



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ОткрытьСоединение(Знач Параметры, Знач Tls = Неопределено) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Параметры | - | Строка | &#x2714; | Параметры соединения. См. [`ПолучитьПараметрыСоединения`](/docs/GRPC/Common-methods/Get-connection-parameters) |
  | Tls | - | Структура Из КлючИЗначение | &#x2716; | Настройки TLS, если необходимо. См. [`ПолучитьНастройкиTls`](/docs/GRPC/Common-methods/Get-tls-settings) |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Произвольный - Объект коннектора или соответствие с информацией об ошибке
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | addin_mode | Ручной выбор режима подключения внешней компоненты (для 1С): Изолированно, НеИзолированно |


  </TabItem>
</Tabs>

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Адрес = "https://grpcb.in:9001";

    Proto1 = "https://hut.openintegrations.dev/test_data/grpcbin_with_import.proto"; // Строка, путь к файлу или URL
    Proto2 = "https://hut.openintegrations.dev/test_data/mt.proto";     // Строка, путь к файлу или URL

    Схемы = Новый Соответствие;
    Схемы.Вставить("main.proto"    , Proto1); // Основной
    Схемы.Вставить("my_types.proto", Proto2); // Для импорта в основной

    Мета      = Новый Структура("somekey", "somevalue");
    Параметры = OPI_GRPC.ПолучитьПараметрыСоединения(Адрес, Схемы, Мета);
    Tls       = OPI_GRPC.ПолучитьНастройкиTls(Истина);

    Результат = OPI_GRPC.ОткрытьСоединение(Параметры, Tls);
```
