# Получить параметры соединения
 Формирует структуру параметров для создания соединения



`Функция ПолучитьПараметрыСоединения(Знач Адрес, Знач Proto = Неопределено, Знач Метаданные = Неопределено) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Адрес | --addr | Строка | &#x2714; | Адрес для подключения (с протоколом) |
  | Proto | --proto | Произвольный | &#x2716; | Схема или структура схем: Ключ > имя файла (для import), Значение > схема строкой или путь к файлу |
  | Метаданные | --meta | Структура Из КлючИЗначение | &#x2716; | Структура метаданных, если необходимо |

  
  Возвращаемое значение:   Структура Из КлючИЗначение - Параметры соединения

:::tip
Имя файла Proto используется для возможности дальнейшего указания этого Proto в import другого Proto Оно не обязано являтся настоящим путем к файлу. Proto, не учавствующие в import других Proto, могут иметь произвольное имя файла

Если значение Proto не является объектом (коллекцией ключ и значение), то оно будет интерпретировано как `{"main.proto": &lt;переданное значение&gt;}`
:::
