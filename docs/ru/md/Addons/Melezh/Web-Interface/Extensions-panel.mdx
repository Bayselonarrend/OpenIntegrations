Далее идет **панель расширений**

## Общий вид

![Панель расширений](../../../../static/img/Docs/Melezh/ru/21.png)

На панели расширений представлен список подключенных пользовательских `.os` модулей, путь к ним нам на диске и количество методов, которые были корректно интерпретированы как функции для использования в обработчиках

В верхнем правом углу страницы присутствуют три кнопки действий:

![Панель расширений](../../../../static/img/Docs/Melezh/ru/22.png)

+ Первая кнопка позволяет вызвать окно создания нового расширения

+ Вторая кнопка отвечает за обновление кэша расширений на сервере. Так как список расширений и их параметров формируется один раз при запуске сервера, обновить его после внесения изменений можно только при помощи этой кнопки или перезапуском самого сервера

+ Третья кнпока позволяет обновить представленный на панели список актуальными данными без перезагрузки страницы
