# Включить регламентное задание
 Включает регламентное задание по ID



`Функция ВключитьРегламентноеЗадание(Знач Проект, Знач IDЗадания) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Проект | --proj | Строка | &#x2714; | Путь к файлу проекта |
  | IDЗадания | --task | Строка | &#x2714; | ID задания |

  
  Возвращаемое значение:   Структура Из КлючИЗначение - Результат переключения

<br/>
