# Получить регламентное задание
 Получает информацию о задании по ID



`Функция ПолучитьРегламентноеЗадание(Знач Проект, Знач IDЗадания) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Проект | --proj | Строка | &#x2714; | Путь к файлу проекта |
  | IDЗадания | --task | Строка | &#x2714; | ID задания |

  
  Возвращаемое значение:   Структура Из КлючИЗначение - Информация об обработчике

<br/>



    

 <Tabs>
  
    <TabItem value="bash" label="Bash" default>
        ```bash
            melezh  ПолучитьРегламентноеЗадание \
              --proj "/proj.melezh" \
              --task 1
        ```
    </TabItem>
  
    <TabItem value="bat" label="CMD/Bat" default>
        ```batch
            melezh  ПолучитьРегламентноеЗадание ^
              --proj "/proj.melezh" ^
              --task 1
        ```
    </TabItem>
</Tabs>
