# Получить список регламентных заданий
 Получает список регламентных заданий в проекте



`Функция ПолучитьСписокРегламентныхЗаданий(Знач Проект) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Проект | --proj | Строка | &#x2714; | Путь к файлу проекта |

  
  Возвращаемое значение:   Структура Из КлючИЗначение - Список заданий

<br/>



    

 <Tabs>
  
    <TabItem value="bash" label="Bash" default>
        ```bash
            melezh  ПолучитьСписокРегламентныхЗаданий \
              --proj "/proj.melezh"
        ```
    </TabItem>
  
    <TabItem value="bat" label="CMD/Bat" default>
        ```batch
            melezh  ПолучитьСписокРегламентныхЗаданий ^
              --proj "/proj.melezh"
        ```
    </TabItem>
</Tabs>
