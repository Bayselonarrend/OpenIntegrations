# Изменить регламентное задание
 Изменяет расписание выбранного регламентного задания



`Функция ИзменитьРегламентноеЗадание(Знач Проект, Знач IDЗадания, Знач Расписание = "", Знач КлючОбработчика = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Проект | --proj | Строка | &#x2714; | Путь к файлу проекта |
  | IDЗадания | --task | Строка | &#x2714; | ID задания |
  | Расписание | --cron | Строка | &#x2716; | Расписание в расширенном cron формате |

  
  Возвращаемое значение:   Структура Из КлючИЗначение - Результат изменения задания

<br/>



    

 <Tabs>
  
    <TabItem value="bash" label="Bash" default>
        ```bash

              --proj "/proj.melezh" \
        ```
