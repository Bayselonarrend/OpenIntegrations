# Добавить регламентное задание
 Добавляет новый обработчик в проект



`Функция ДобавитьРегламентноеЗадание(Знач Проект, Знач КлючОбработчика, Знач Расписание) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Проект | --proj | Строка | &#x2714; | Путь к файлу проекта |
  | КлючОбработчика | --handler | Строка | &#x2714; | Ключ обработчика |
  | Расписание | --cron | Строка | &#x2714; | Расписание в расширенном cron формате |

  
  Возвращаемое значение:   Структура Из КлючИЗначение - Результат добавления задания

<br/>

:::tip
Формат расписания:<br/>sec  min   hour   day of month   month   day of week   year<br/>0   30   9,12,15     1,15       May-Aug  Mon,Wed,Fri  2018/2
:::
<br/>


    

 <Tabs>
  
    <TabItem value="bash" label="Bash" default>
        ```bash
            melezh  ДобавитьРегламентноеЗадание \
              --proj "/proj.melezh" \
              --handler "my-key" \
              --cron "0 7 10 * * * *"
        ```

    </TabItem>
