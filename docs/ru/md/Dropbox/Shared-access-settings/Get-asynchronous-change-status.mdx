# Получить статус асинхронного изменения
 Получает статус асинхронной работы по изменению доступов



`Функция ПолучитьСтатусАсинхронногоИзменения(Знач Токен, Знач IDРаботы) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | IDРаботы | --job | Строка | &#x2714; | ID асинхронной работы |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Dropbox

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен    = "sl.CCRidcEOx5_Sft41hhKNkIVvFAu8ZT1CoP6wh40MOAccwPZhLOI2YXXzbpEypt0YZf_3ZGYM9H...";
    IDРаботы = "dbjid:AABrLP8RbSbI6B8BWACJPQsnAaAcyWovxYuyf_nxYnxOjOZU84YRi5uSjh6XreAnG5YP2Iq72GCT8yTeT1m444xz";

    Результат = OPI_Dropbox.ПолучитьСтатусАсинхронногоИзменения(Токен, IDРаботы);
```
