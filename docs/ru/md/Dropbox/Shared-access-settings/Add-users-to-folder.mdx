# Добавить пользователей к папке
 Предоставляет стороннии пользователям доступ к каталогу



`Функция ДобавитьПользователейКПапке(Знач Токен, Знач IDПапки, Знач АдресаПочты, Знач ТолькоПросмотр = Истина) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | IDПапки | --folder | Строка | &#x2714; | ID публичного каталога (shared folder ID) |
  | АдресаПочты | --emails | Строка, Массив Из Строка | &#x2714; | Список адресов почты добавляемых пользователей |
  | ТолькоПросмотр | --readonly | Булево | &#x2716; | Запрещает редактирование файла для стороннего пользователя |

  
  Возвращаемое значение:   Неопределено - пустой ответ
