# Получить список версий объекта
 Получает список версий (ревизий) объекта



`Функция ПолучитьСписокВерсийОбъекта(Знач Токен, Знач Путь, Знач Количество = 10) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Путь | --path | Строка | &#x2714; | Путь к объекту |
  | Количество | --amount | Строка, Число | &#x2716; | Число последних версий объекта для отображения |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Dropbox

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен = "sl.CBw46cIJLPk0Xg3MVRbs5VgDgSZozffBD6ltE5NDB8X7jDaZlIrHZmPEx60hZbiIUDVhfEjuNc...";
    Путь  = "/New/pic.png";

    Результат = OPI_Dropbox.ПолучитьСписокВерсийОбъекта(Токен, Путь, 1);
```
