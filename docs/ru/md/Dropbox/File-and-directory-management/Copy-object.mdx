# Копировать объект
 Копирует файл или каталог по выбранному пути



`Функция КопироватьОбъект(Знач Токен, Знач Откуда, Знач Куда) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Откуда | --from | Строка | &#x2714; | Путь к объекту оригинала |
  | Куда | --to | Строка | &#x2714; | Целевой путь для нового объекта |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Dropbox
