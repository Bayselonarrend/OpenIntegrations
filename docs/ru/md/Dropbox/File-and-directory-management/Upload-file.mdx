# Загрузить файл
 Загружает файл на облачный диск



`Функция ЗагрузитьФайл(Знач Токен, Знач Файл, Знач Путь, Знач Перезаписывать = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Файл | --file | Строка, ДвоичныеДанные | &#x2714; | Данные файл для загрузки |
  | Путь | --path | Строка | &#x2714; | Путь сохранения на Dropbox |
  | Перезаписывать | --overwrite | Булево | &#x2716; | Перезаписывать файл при конфликте путей |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Dropbox

<br/>

:::tip
Параметры с типом данных Двоичные данные могут принимать также пути к файлам на диске и URL
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Путь     = "/New/pic.png";
    Токен    = "sl.CB3Ppv-heMGAtz7VGnrX2QCBUahgZ7iiChxnPAHWs9vqQ5ZcBLmznTMZ8h3IvAuhPOSm89QiaV...";
    Картинка = "https://api.athenaeum.digital/test_data/picture.jpg";

    КартинкаПуть = ПолучитьИмяВременногоФайла("png");
    КопироватьФайл(Картинка, КартинкаПуть);

    Результат = OPI_Dropbox.ЗагрузитьФайл(Токен, КартинкаПуть, Путь, Истина);
```
