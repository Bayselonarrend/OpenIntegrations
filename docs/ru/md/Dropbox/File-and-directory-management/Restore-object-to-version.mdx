# Восстановить объект к версии
 Восстанавливает состояние объекта к необходимой версии (ревизии)



`Функция ВосстановитьОбъектКВерсии(Знач Токен, Знач Путь, Знач Версия) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Путь | --path | Строка | &#x2714; | Путь к объекту |
  | Версия | --rev | Строка | &#x2714; | ID версии (ревизии) для востановления |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Dropbox

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Версия    = "626a6f2fcb4f2bd841553";
    Токен     = "sl.CAirrApMP3MmqfxAgYhlitMkH3J7CmcSnaOjCUghRVLtFIhPZ0jzduJbWPea8qBaOeeB-L_PZ6...";
    Путь      = "/New/pic.png";

    Результат = OPI_Dropbox.ВосстановитьОбъектКВерсии(Токен, Путь, Версия);
```
