# Получить превью
 Получает PDF или HTML превью объекта (только для документов)



`Функция ПолучитьПревью(Знач Токен, Знач Путь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Путь | --path | Строка | &#x2714; | Путь к объекту |

  
  Возвращаемое значение:   ДвоичныеДанные - превью документа
