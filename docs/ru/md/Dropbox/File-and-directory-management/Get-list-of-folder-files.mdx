# Получить список файлов папки
 Получает список первых файлов каталога или продолжает получение следующих при указании курсора



`Функция ПолучитьСписокФайловПапки(Знач Токен, Знач Путь = "", Знач Подробно = Ложь, Знач Курсор = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Путь | --path | Строка | &#x2716; | Путь к каталогу. Необязателен, если указан курсор |
  | Подробно | --detail | Булево | &#x2716; | Добавляет дополнительные поля информации для медиафайлов |
  | Курсор | --cursor | Строка | &#x2716; | Курсор из предыдущего запроса для получения следующего набора файлов |

  
  Возвращаемое значение:   HTTPОтвет - Получить список файлов папки
