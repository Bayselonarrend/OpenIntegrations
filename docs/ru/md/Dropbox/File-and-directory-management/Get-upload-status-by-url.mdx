# Получить статус загрузки по URL
 Получает статус загрузки файла по URL



`Функция ПолучитьСтатусЗагрузкиПоURL(Знач Токен, Знач IDРаботы) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | IDРаботы | --job | Строка | &#x2714; | ID асинхронной работы из ответа ЗагрузитьФайлПоURL |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Dropbox

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен    = "sl.CBG8hWTAeQUMIDvdg2G2nf_pdX3rktbvuu5E7lmsOmK-0-uELkt5I_IEtTRyVkGJgDwCS1LCL2...";
    ИДРаботы = "cLu5aH_-YR0AAAAAAAAAAQ";
    Статус   = "in_progress";

    Пока Статус = "in_progress" Цикл

        Результат = OPI_Dropbox.ПолучитьСтатусЗагрузкиПоURL(Токен, ИДРаботы);
        Статус    = Результат[".tag"];

        OPI_Инструменты.Пауза(5);

    КонецЦикла;
```
