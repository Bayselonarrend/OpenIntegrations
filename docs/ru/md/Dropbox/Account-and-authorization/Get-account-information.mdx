# Получить информацию об аккаунте
 Получает информацию об аккаунте



`Функция ПолучитьИнформациюОбАккаунте(Знач Токен, Знач Аккаунт = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Аккаунт | --account | Строка | &#x2716; | ID аккаунта. Текущий аккаунт токена, если не заполнено |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Dropbox

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен = "sl.B_P83AtWK3iWmiaVVtO7xtmH-oAB1JFAuYM0O4cp24jpCem-7yD8_3nHrlvjtoWiI7T4kRh7Yx...";

    Результат = OPI_Dropbox.ПолучитьИнформациюОбАккаунте(Токен);
```
