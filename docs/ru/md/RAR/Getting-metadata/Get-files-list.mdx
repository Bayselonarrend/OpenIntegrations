# Получить список файлов
 Получает иерархический список файлов и каталогов архива



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьСписокФайлов(Знач Архив, Знач Пароль = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Архив | --src | Строка, ДвоичныеДанные | &#x2714; | Путь к архиву на диске или двоичные данные |
  | Пароль | --password | Строка | &#x2716; | Пароль архива, если требуется |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - Информация о файлах в архиве
  </div>
</div>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его данные (только для 1С и OneScript) |


  </TabItem>

:::tip

:::

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ПутьАрхива = Параметры["RAR_ArchivePath"];

    Результат = OPI_RAR.ПолучитьСписокФайлов(ПутьАрхива);
```
