# Открыть соединение 
 Создает подключение к указанной базе



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ОткрытьСоединение(Знач СтрокаПодключения = "", Знач Tls = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | СтрокаПодключения | - | Строка | &#x2716; | Строка подключения. См. [`СформироватьСтрокуПодключения`](/docs/MySQL/Common-methods/Generate-connection-string) |
  | Tls | - | Структура Из КлючИЗначение | &#x2716; | Настройки TLS, если необходимо. См. [`ПолучитьНастройкиTls`](/docs/MySQL/Common-methods/Get-tls-settings) |

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Адрес  = "127.0.0.1";
    Логин  = "bayselonarrend";
    Пароль = "12we...";
    База   = "";

    TLS = Истина;
    Порт = 3306;
    СтрокаПодключения = OPI_MySQL.СформироватьСтрокуПодключения(Адрес, База, Логин, Пароль, Порт);

    Если TLS Тогда
        НастройкиTLS = OPI_MySQL.ПолучитьНастройкиTLS(Истина);
    Иначе
        НастройкиTLS = Неопределено;
    КонецЕсли;

    Результат = OPI_MySQL.ОткрытьСоединение(СтрокаПодключения, НастройкиTLS);
```
