# Сформировать строку подключения
 Формирует строку подключения из переданных данных



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция СформироватьСтрокуПодключения(Знач Адрес, Знач База = "", Знач Логин = "", Знач Пароль = "", Знач Порт = "3306") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Адрес | --addr | Строка | &#x2714; | IP адрес или доменное имя сервера |
  | База | --db | Строка | &#x2716; | Имя базы данных для подключения |
  | Логин | --login | Строка | &#x2716; | Логин пользователя mysql |
  | Пароль | --pass | Строка | &#x2716; | Пароль пользователя mysql |
  | Порт | --port | Строка | &#x2716; | Порт подключения |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>

  Строка -  Строка подключения к базе MySQL
  </div>
</div>
  
  </TabItem>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его данные (только для 1С и OneScript) |
