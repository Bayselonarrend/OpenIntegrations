# Разархивировать файлы
 Распаковывает выбранные файлы из 7z-архива по списку путей



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция РазархивироватьФайлы(Знач Архив, Знач Пути, Знач КаталогНазначения = "", Знач Пароль = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Архив | --src | Строка, ДвоичныеДанные | &#x2714; | Путь к архиву на диске или двоичные данные |
  | Пути | --paths | Массив Из Строка | &#x2714; | Полные пути файлов внутри архива |
  | КаталогНазначения | --dest | Строка | &#x2716; | Путь каталога для распаковки. В соответствие, если не заполнено |
  | Пароль | --password | Строка | &#x2716; | Пароль архива, если требуется |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - Информация о выполнении или описание выбранных файлов
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его данные (только для 1С и OneScript) |


  </TabItem>
</Tabs>

:::tip
Параметры с типом данных Двоичные данные могут принимать также пути к файлам на диске и URL
:::
