# Изменить архив
 Добавляет, заменяет и удаляет файлы в существующем 7z-архиве



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ИзменитьАрхив(Знач Архив, Знач ДобавляемыеФайлы = Неопределено, Знач УдаляемыеПути = Неопределено, Знач Настройки = Неопределено) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Архив | --src | Строка, ДвоичныеДанные | &#x2714; | Путь к архиву на диске или двоичные данные |
  | ДобавляемыеФайлы | --additions | Соответствие Из КлючИЗначение | &#x2716; | Ключ: путь в архиве; значение: путь к файлу или двоичные данные |
  | УдаляемыеПути | --deletions | Массив Из Строка | &#x2716; | Пути файлов в архиве для удаления |
  | Настройки | --settings | Соответствие Из КлючИЗначение | &#x2716; | Настройки изменения. См. [`ПолучитьСтруктуруИзмененияАрхива`](/docs/7z/Archiving/Get-archive-modification-structure) |

:::tip

<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ПутьАрхива      = Параметры["SevenZ_ArchivePath"];
    ПутьНовогоФайла = Параметры["SevenZ_NewFilePath"];

    ДобавляемыеФайлы = Новый Соответствие;
    ДобавляемыеФайлы.Вставить("extra/info.txt", ПутьНовогоФайла);
    ДобавляемыеФайлы.Вставить("readme.txt"    , ПолучитьДвоичныеДанныеИзСтроки("Replaced readme content", "UTF-8"));

    УдаляемыеПути = Новый Массив;
    УдаляемыеПути.Добавить("config/settings.json");
    УдаляемыеПути.Добавить("missing/path.txt");

    Результат = OPI_7z.ИзменитьАрхив(ПутьАрхива, ДобавляемыеФайлы, УдаляемыеПути);
```
