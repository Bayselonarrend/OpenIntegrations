# Восстановить счетчик
 Восстанавливает ранее удаленный счетчик по ID



`Функция ВосстановитьСчетчик(Знач Токен, Знач IDСчетчика) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен авторизации |
  | IDСчетчика | --counter | Строка, Число | &#x2714; | ID счетчика для восстановления |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Yandex

<br/>

:::tip
Метод в документации API: [Восстановление счетчика](https://yandex.ru/dev/metrika/ru/management/openapi/counter/undeleteCounter)
:::
<br/>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен      = "y0_AgAAAABdylaOAAy9KgAAAAEXh6i...";
    IDСчетчика = "103080464";

    Результат = OPI_YandexMetrika.ВосстановитьСчетчик(Токен, IDСчетчика);
```
