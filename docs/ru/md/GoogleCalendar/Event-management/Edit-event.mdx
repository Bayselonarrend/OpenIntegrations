# Изменить событие
 Изменяет существующее событие



`Функция ИзменитьСобытие(Знач Токен, Знач Календарь, Знач ОписаниеСобытия, Знач Событие) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Календарь | --calendar | Строка | &#x2714; | ID календаря |
  | ОписаниеСобытия | --props | Строка | &#x2714; | Новое описание события |
  | Событие | --event | Строка | &#x2714; | ID события |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Google



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен     = "ya29.a0ATkoCc7NIaMoQyko9kNNsGA_NcNp2ra5WH144UV1BR2r9JdyTQocbBSQiR9QcLAI2btfyyE1OMQUkgMHs9xVvwkMlDaQ8dqWiQIftCNobgkG15t59vzvmxN2o...";
    Календарь = "3020e4442e28bc7824257a6e027e4f63a40c310587e1f95fab6bac0a96027c08@group.calendar.google.com";
    Событие   = "plk7hgjdaq2v308h52jv1o6o90";
    Описание  = "Новое описание события";

    ОписаниеСобытия = Новый Соответствие;
    ОписаниеСобытия.Вставить("Описание", Описание);

    Результат = OPI_GoogleCalendar.ИзменитьСобытие(Токен, Календарь, ОписаниеСобытия, Событие);
```
