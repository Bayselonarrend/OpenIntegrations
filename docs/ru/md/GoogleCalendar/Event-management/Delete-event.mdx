# Удалить событие
 Удаляет событие по ID



`Функция УдалитьСобытие(Знач Токен, Знач Календарь, Знач Событие) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Календарь | --calendar | Строка | &#x2714; | ID календаря |
  | Событие | --event | Строка | &#x2714; | ID события |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Google

<br/>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен     = "ya29.a0AQQ_BDT_CMAJqFIcOfzUrSDYRQ2mZ6TLDIGN-tv27R3mPj8iFcUgZ-x-aipMSCOTXMfqqTzT2aRN4TCe8MFrBD2tInauCN39ueJtc4LHbj71tbNlq9LdkejgF...";
    Календарь = "cece9cf2ef513ea9209ec0e4b5029809c52153f74ad18d53e40d362bcb1f297a@group.calendar.google.com";
    Событие   = "ccknhhgag1168lmsuqdkn9n9gk";

    Результат = OPI_GoogleCalendar.УдалитьСобытие(Токен, Календарь, Событие);
```
