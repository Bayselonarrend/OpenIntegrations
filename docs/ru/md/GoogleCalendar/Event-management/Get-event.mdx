# Получить событие
 Получает событие по ID



`Функция ПолучитьСобытие(Знач Токен, Знач Календарь, Знач Событие) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Календарь | --calendar | Строка | &#x2714; | ID календаря |
  | Событие | --event | Строка | &#x2714; | ID события |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Google



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен     = "ya29.a0Aa7pCA_GKuTstHik6bj1MJWZFLq6Rw5mQEZ6UpvNp9rgAbC-k5fpzhJPU_c4cEvVXwDo6GAXMYcBuPpzfZoPNP2pKbXWwL-bwnzotcuHaeGL9ZoSxPM95kRPm...";
    Календарь = "820a0b504e49dc31b520578e608186d90ce12d730bb5b46b48f372df1a63cd0b@group.calendar.google.com";
    Событие   = "nfvagukkh3e3juj875j87q06r4";

    Результат = OPI_GoogleCalendar.ПолучитьСобытие(Токен, Календарь, Событие);
```
