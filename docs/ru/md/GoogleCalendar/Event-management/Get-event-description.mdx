# Получить описание события
 Получает пустой макет для создания события



`Функция ПолучитьОписаниеСобытия(Знач Пустая = Ложь) Экспорт`

<Tabs>
  <TabItem value="params" label="Параметры" default>

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Пустая | --empty | Булево | &#x2716; | Истина > структура с пустыми значениями, Ложь > в значениях будут описания полей |

  
  </TabItem>
  <TabItem value="extended" label="Расширенный вызов">

*Для текущего метода нет дополнительных параметров расширенного вызова.*


  </TabItem>
</Tabs>
  
  Возвращаемое значение:  Соответствие Из КлючИЗначение - Пустой макет события



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Пустая    = Ложь;
    Результат = OPI_GoogleCalendar.ПолучитьОписаниеСобытия(Пустая);
```
