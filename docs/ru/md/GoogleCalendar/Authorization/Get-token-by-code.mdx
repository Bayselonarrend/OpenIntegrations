# Получить токен по коду
 Получает токен по коду из авторизации в браузере



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьТокенПоКоду(Знач ClientID, Знач ClientSecret, Знач Code) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ClientID | --id | Строка | &#x2714; | Client ID |
  | ClientSecret | --secret | Строка | &#x2714; | Client secret |
  | Code | --code | Строка | &#x2714; | Code из браузера |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - сериализованный JSON ответа от Google
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его данные (только для 1С и OneScript) |


  </TabItem>
</Tabs>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ClientID     = "345925493105-641q49rl4oflrtr46fkgqf5upiltr4bg.apps.googleusercontent.com";
    ClientSecret = "GOCSPX-v--MJyEjtV...";
    Code         = "4/0AeaYSHBDZAbAtChyjMxPKa0DXIilJYYcVCgKYzA_CPmCPSt1mg_KqxI4w5CP3lTlYibpGg";

    Результат = OPI_GoogleCalendar.ПолучитьТокенПоКоду(ClientID, ClientSecret, Code);
```
