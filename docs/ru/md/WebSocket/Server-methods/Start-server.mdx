# Запустить сервер 
 Запускает ожидание сообщений на указанном порту



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ЗапуститьСервер(Знач Порт, Знач РазмерПула = 100, Знач Логирование = Неопределено) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Порт | - | Число | &#x2714; | Порт сервера |
  | РазмерПула | - | Число | &#x2716; | Максимальное число одновременно поддерживаемых соединений |
  | Логирование | - | Структура Из КлючИЗначение | &#x2716; | Настройки логирования. См. [`ПолучитьНастройкиЛогирования`](/docs/WebSocket/Common/Get-logging-settings) |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Произвольный - Объект сервера или соответствие с информацией об ошибке
  </div>
</div>
