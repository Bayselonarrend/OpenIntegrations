# Отправить данные 
 Отправляет данные в указанное соединение



`Функция ОтправитьДанные(Знач ОбъектСервера, Знач IDСоединения, Знач Данные) Экспорт`

<Tabs>
  <TabItem value="params" label="Параметры" default>

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ОбъектСервера | - | Произвольный | &#x2714; | Объект компоненты запущенного сервера |
  | IDСоединения | - | Строка | &#x2714; | Идентификатор соединения |
  | Данные | - | Строка, ДвоичныеДанные | &#x2714; | Данные для отправки |

  
  </TabItem>
  <TabItem value="extended" label="Расширенный вызов">

*Для текущего метода нет дополнительных параметров расширенного вызова.*


  </TabItem>
</Tabs>
  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - Результат выполнения

:::tip
Параметры с типом данных Двоичные данные могут принимать также пути к файлам на диске и URL
:::

:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
