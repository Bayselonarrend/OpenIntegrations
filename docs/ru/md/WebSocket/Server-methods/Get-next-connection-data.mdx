# Получить данные очередного соединения 
 Получает данные из буфера очередного активного соединения в очереди, где есть данные входящие данные



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьДанныеОчередногоСоединения(Знач ОбъектСервера, Знач Таймаут = 1000) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ОбъектСервера | - | Произвольный | &#x2714; | Объект компоненты запущенного сервера |
  | Таймаут | - | Число | &#x2716; | Период ожидания новых данных, если очередь пуста (в мс) |
  | МаксимальныйРазмер | - | Число | &#x2714; | Максимальный размер данных для получения |

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    // Запуск сервера
    ПортЗапуска   = 9894;
    ОбъектСервера = OPI_WebSocket.ЗапуститьСервер(ПортЗапуска);

    // Подключение к серверу при помощи клиента
    АдресПодключения = "ws://127.0.0.1:9894";
    ОбъектКлиента    = OPI_WebSocket.ОткрытьСоединение(АдресПодключения);

    Если Не OPI_WebSocket.ЭтоОбъектКлиента(ОбъектКлиента) Тогда
        ВызватьИсключение OPI_Инструменты.JSONСтрокой(ОбъектКлиента);
    Иначе
        // Отправка с клиента
        Сообщение = "Hello from client!";
        Отправка  = OPI_WebSocket.ОтправитьТекстовоеСообщение(ОбъектКлиента, Сообщение);
    КонецЕсли;

    // Получение на сервере
    Результат = OPI_WebSocket.ПолучитьДанныеОчередногоСоединения(ОбъектСервера, 5000);
```
