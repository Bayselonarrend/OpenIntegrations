# Получить настройки TLS 
 Формирует настройки для использования TLS при выполнении запросов



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьНастройкиTls(Знач ОтключитьПроверкуСертификатов, Знач ПутьКСертификату = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ОтключитьПроверкуСертификатов | - | Булево | &#x2714; | Позволяет работать с некорректными сертификатами, в т.ч. самоподписанными |
  | ПутьКСертификату | - | Строка | &#x2716; | Путь к корневому PEM файлу сертификата, если его нет в системном хранилище |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Структура Из КлючИЗначение - Структура настроек TLS соединения
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

*Для текущего метода нет дополнительных параметров расширенного вызова.*


  </TabItem>
</Tabs>

:::tip
Настройки Tls могут быть установлены только в момент создания соединения
:::
