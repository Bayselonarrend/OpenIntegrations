# Закрыть соединение 
 Явно закрывает созданное ранее соединение



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ЗакрытьСоединение(Знач Соединение, Знач Код = 1000, Знач Причина = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |

  | Код | - | Число | &#x2716; | Код состояния |
  | Причина | - | Строка | &#x2716; | Текстовое сообщение при закрытии |
