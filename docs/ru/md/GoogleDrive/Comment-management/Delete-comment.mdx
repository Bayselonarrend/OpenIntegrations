# Удалить комментарий
 Удаляет комментарий по ID



`Функция УдалитьКомментарий(Знач Токен, Знач ИДОбъекта, Знач ИДКомментария) Экспорт`

<Tabs>
  <TabItem value="params" label="Параметры" default>

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | ИДОбъекта | --object | Строка | &#x2714; | Идентификатор файла или каталога размещения комментария |
  | ИДКомментария | --comment | Строка | &#x2714; | Идентификатор комментария |

  
  </TabItem>
  <TabItem value="extended" label="Расширенный вызов">

  | Параметр | Описание |
  |---|---|
  | proxy | ИнтернетПрокси или структура полей Протокол, Сервер, Порт, Пользователь, Пароль, ИспользоватьАутентификациюОС |
  | timeout | Таймаут выполнения запроса |
  | adv_response | Оформляет ответ в виде полной структуры с полями code, body и headers |


  </TabItem>
</Tabs>
  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Google
