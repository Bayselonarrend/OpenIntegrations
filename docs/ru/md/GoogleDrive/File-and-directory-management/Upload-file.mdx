# Загрузить файл
 Загружает файл на диск



`Функция ЗагрузитьФайл(Знач Токен, Знач Файл, Знач Описание) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Файл | --file | ДвоичныеДанные, Строка | &#x2714; | Загружаемый файл |
  | Описание | --props | Соответствие Из КлючИЗначение | &#x2714; |  См. [`ПолучитьОписаниеФайла`](/docs/GoogleDrive/File-and-directory-management/Get-file-description) |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Google

:::tip
Параметры с типом данных Двоичные данные могут принимать также пути к файлам на диске и URL
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен    = "ya29.a0ATi6K2sWW_31WLH0d9aK0qk0HJ-UfyvZDcLeBbF5bLU7QH09Bx2pXAupIw40USQVaxXqsovItVSt9swWuPdBDSA0eFaVPE0CbaU2OLQlyrnITI1VVXgZ_Gx2R...";
    Каталог  = "1RA5EV-HEr2SDzooZLmZ2h6HnnmZ0EHzP";
    Картинка = "https://hut.openintegrations.dev/test_data/picture.jpg";      // URL, Двоичные данные или Путь к файлу

    Пустая   = Ложь;
    Описание = OPI_GoogleDrive.ПолучитьОписаниеФайла(Пустая);

    Описание.Вставить("Родитель", Каталог);

    Результат = OPI_GoogleDrive.ЗагрузитьФайл(Токен, Картинка, Описание);
```
