# Изменить свойства страницы
 Изменяет свойства существующей страницы



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ИзменитьСвойстваСтраницы(Знач Токен, Знач Страница, Знач Данные = "", Знач Иконка = "", Знач Обложка = "", Знач Архивирована = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Страница | --page | Строка | &#x2714; | ID изменяемой страницы |
  | Данные | --data | Соответствие Из КлючИЗначение | &#x2716; | Соответствие изменяемых параметров |
  | Иконка | --icon | Строка | &#x2716; | URL картинки иконки страницы |
  | Обложка | --cover | Строка | &#x2716; | URL картинки обложки страницы |
  | Архивирована | --archive | Булево | &#x2716; | Архивировать страницу или нет (булево) |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - сериализованный JSON ответа от Notion
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его объект (только для 1С и OneScript) |


  </TabItem>
</Tabs>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен        = "secret_9RsfMrRMqZwqp0Zl0B...";
    Страница     = "29d82aa7-fc4e-81bd-8fc1-e5b7d59b73fa";
    Иконка       = "https://hut.openintegrations.dev/test_data/picture.jpg";
    Обложка      = "https://hut.openintegrations.dev/test_data/picture2.jpg";
    Архивировать = Ложь;

    Свойства = Новый Соответствие;
    Свойства.Вставить("Активен", Ложь);
    Свойства.Вставить("Почта"  , "vector@mail.ru");

    Результат = OPI_Notion.ИзменитьСвойстваСтраницы(Токен, Страница, Свойства, Иконка, Обложка, Архивировать);
```
