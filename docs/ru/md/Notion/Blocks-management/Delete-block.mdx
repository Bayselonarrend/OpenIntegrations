# Удалить блок
 Удаляет блок по ID



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция УдалитьБлок(Знач Токен, Знач ИДБлока) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | ИДБлока | --block | Строка | &#x2714; | ID блока |
