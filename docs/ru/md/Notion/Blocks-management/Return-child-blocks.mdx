# Вернуть дочерние блоки
 Возвращает список дочерних блоков блока-родителя



`Функция ВернутьДочерниеБлоки(Знач Токен, Знач ИДБлока) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | ИДБлока | --block | Строка | &#x2714; | ID блока родителя |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Notion



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен = "secret_9RsfMrRMqZwqp0Zl0B...";
    Блок  = "29d82aa7-fc4e-8196-bfa0-e3dd8f77f4ee";

    Результат = OPI_Notion.ВернутьДочерниеБлоки(Токен, Блок);
```
