# Получить лог 
 Получает данные лога соединения (при включенном логировании в память)



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьЛог(Знач Соединение, Знач КакСтрока = Ложь, Знач ЧислоСобытий = 100) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Соединение | - | Произвольный | &#x2714; | Объект компоненты с открытым соединением или портом |
  | КакСтрока | - | Булево | &#x2716; | Истина > возвращает лог одной строкой, Ложь > как массив |
  | ЧислоСобытий | - | Число | &#x2716; | Число последних событий для получения. 0 > без ограничений |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Строка, Соответствие Из КлючИЗначение - Лог как строка или соответствие с полным результатом выполнения
  </div>
</div>
  
  </TabItem>

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ФайлЛога             = ПолучитьИмяВременногоФайла("txt");
    НастройкиЛогирования = OPI_ZeroMQ.ПолучитьНастройкиЛогирования(Истина, 100, ФайлЛога);

    Порт          = 5555;
    ОбъектСервера = OPI_ZeroMQ.ОткрытьПортRep(Порт);

    Если Не OPI_ZeroMQ.ЭтоОбъектКоннектора(ОбъектСервера) Тогда
        ВызватьИсключение OPI_Инструменты.JSONСтрокой(ОбъектСервера);
    КонецЕсли;

    Адрес      = "tcp://127.0.0.1:5555";
    Соединение = OPI_ZeroMQ.СоздатьСоединениеReq(Адрес, НастройкиЛогирования);

    Если Не OPI_ZeroMQ.ЭтоОбъектКоннектора(Соединение) Тогда
        ВызватьИсключение OPI_Инструменты.JSONСтрокой(Соединение);
    КонецЕсли;

    Сообщение = СтрШаблон("ZMQ_LOG_%1", Формат(ТекущаяДата(), "ДФ=yyyyMMddhhmmss"));
    Данные = ПолучитьДвоичныеДанныеИзСтроки(Сообщение);

    OPI_ZeroMQ.ОтправитьДанные(Соединение, Данные, 3000);
    OPI_ZeroMQ.ПолучитьДанные(ОбъектСервера, 3000);

    Результат = OPI_ZeroMQ.ПолучитьЛог(Соединение);
```
