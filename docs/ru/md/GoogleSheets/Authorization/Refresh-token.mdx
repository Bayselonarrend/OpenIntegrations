# Обновить токен
 Обновляет токен по Refresh token



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ОбновитьТокен(Знач ClientID, Знач ClientSecret, Знач RefreshToken) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ClientID | --id | Строка | &#x2714; | Client ID |
  | ClientSecret | --secret | Строка | &#x2714; | Client secret |
  | RefreshToken | --refresh | Строка | &#x2714; | Refresh token |
