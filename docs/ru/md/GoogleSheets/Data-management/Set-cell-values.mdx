# Установить значения ячеек
 Устанавливает значения ячеек листа



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция УстановитьЗначенияЯчеек(Знач Токен, Знач Книга, Знач СоответствиеЗначений, Знач Лист = "", Знач ОсновноеИзмерение = "COLUMNS") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Книга | --spreadsheet | Строка | &#x2714; | ID книги |
  | СоответствиеЗначений | --data | Соответствие Из КлючИЗначение | &#x2714; | Данные заполнения, где ключ это имя ячейки вида A1 |
  | Лист | --sheetname | Строка | &#x2716; | Имя листа (первый лист по умолчанию) |
  | ОсновноеИзмерение | --dim | Строка | &#x2716; | Основное измерение при заполнении диапазона массивом |
