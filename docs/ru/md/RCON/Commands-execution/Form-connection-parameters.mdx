# Сформировать параметры соединения
 Формирует коллекцию параметров соединения



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция СформироватьПараметрыСоединения(Знач URL, Знач Пароль, Знач ТаймаутЧтения = 30, Знач ТаймаутЗаписи = 30) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера |
  | Пароль | --pass | Строка | &#x2714; | Пароль для подключения |
  | ТаймаутЧтения | --rtout | Число | &#x2716; | Таймаут ожидания ответа (в секундах) |
  | ТаймаутЗаписи | --wtout | Число | &#x2716; | Таймаут ожидания отправки запроса (в секундах) |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Структура Из КлючИЗначение - Структура параметров соединения
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его объект (только для 1С и OneScript) |


  </TabItem>
</Tabs>
