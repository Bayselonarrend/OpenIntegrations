# Получить список опубликованных объектов.
 Получает список опубликованных объектов



`Функция ПолучитьСписокОпубликованныхОбъектов(Знач Токен, Знач Количество = 0, Знач СмещениеОтНачала = 0) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Количество | --amount | Число | &#x2716; | Количество возвращаемых объектов |
  | СмещениеОтНачала | --offset | Число | &#x2716; | Смещение для получение объектов не из начала списка |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Yandex

<br/>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен            = "y0_AgAAAABdylaOAAs0QgAAAAD5i-a...";
    Количество       = 2;
    СмещениеОтНачала = 1;

    Результат = OPI_YandexDisk.ПолучитьСписокОпубликованныхОбъектов(Токен, Количество, СмещениеОтНачала);
```
    

 <Tabs>
  
    <TabItem value="bash" label="Bash" default>
        ```bash
            oint yadisk ПолучитьСписокОпубликованныхОбъектов \
              --token "***" \
              --amount 2 \
              --offset 1
        ```
    </TabItem>
