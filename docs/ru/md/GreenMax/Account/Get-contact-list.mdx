# Получить список контактов
 Получает список контактов аккаунта



`Функция ПолучитьСписокКонтактов(Знач ПараметрыДоступа, Знач Количество = Неопределено) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ПараметрыДоступа | --access | Структура Из КлючИЗначение | &#x2714; | Параметры доступа. См. [`СформироватьПараметрыДоступа`](/docs/GreenMax/Account/Form-access-parameters) |
  | Количество | --count | Число | &#x2716; | Количество контактов для получения. Все, если не указано |

  
  Возвращаемое значение:   Массив Из Соответствие - сериализованный JSON ответа от Green API

:::tip
Метод в документации API: [GetContacts](https://green-api.com/v3/docs/api/service/GetContacts/)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ApiUrl           = "https://3100.api.green-api.com/v3";
    MediaUrl         = "https://3100.api.green-api.com/v3";
    IdInstance       = "31003...";
    ApiTokenInstance = "17915d8e0b4b4975a183478da...";

    Количество = 1;

    ПараметрыДоступа = OPI_GreenMax.СформироватьПараметрыДоступа(ApiUrl, MediaUrl, IdInstance, ApiTokenInstance);
    Результат        = OPI_GreenMax.ПолучитьСписокКонтактов(ПараметрыДоступа, Количество);
```
