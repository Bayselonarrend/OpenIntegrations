# Отправить текстовое сообщение
 Отправляет текстовое сообщение в выбранный чат



`Функция ОтправитьТекстовоеСообщение(Знач ПараметрыДоступа, Знач IDЧата, Знач Текст, Знач ВремяНабора = 0) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ПараметрыДоступа | --access | Структура Из КлючИЗначение | &#x2714; | Параметры доступа. См. [`СформироватьПараметрыДоступа`](/docs/GreenMax/Account/Form-access-parameters) |
  | IDЧата | --chat | Строка | &#x2714; | Идентификатор чата |
  | Текст | --text | Строка | &#x2714; | Текст сообщения |
  | ВремяНабора | --typing | Число | &#x2716; | Время показа признака набора текста перед отправкой (в мс.) |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Green API

:::tip
Метод в документации API: [SendMessage](https://green-api.com/v3/docs/api/sending/SendMessage/)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ApiUrl           = "https://3100.api.green-api.com/v3";
    MediaUrl         = "https://3100.api.green-api.com/v3";
    IdInstance       = "31003...";
    ApiTokenInstance = "17915d8e0b4b4975a183478da...";

    Текст  = "Текст сообщения";
    IDЧата = 12345678;
    Набор  = 2000;

    ПараметрыДоступа = OPI_GreenMax.СформироватьПараметрыДоступа(ApiUrl, MediaUrl, IdInstance, ApiTokenInstance);
    Результат        = OPI_GreenMax.ОтправитьТекстовоеСообщение(ПараметрыДоступа, IDЧата, Текст, Набор);
```
