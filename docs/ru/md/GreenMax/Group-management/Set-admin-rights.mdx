# Назначить права администратора
 Назначает права администратора указанному участнику группового чата



`Функция НазначитьПраваАдминистратора(Знач ПараметрыДоступа, Знач IDЧата, Знач IDУчастника) Экспорт`

<Tabs>
  <TabItem value="params" label="Параметры" default>

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ПараметрыДоступа | --access | Структура Из КлючИЗначение | &#x2714; | Параметры доступа. См. [`СформироватьПараметрыДоступа`](/docs/GreenMax/Account/Form-access-parameters) |
  | IDЧата | --chat | Строка | &#x2714; | ID группового чата |
  | IDУчастника | --member | Строка | &#x2714; | ID участника |

  
  </TabItem>
  <TabItem value="extended" label="Расширенный вызов">

  | Параметр | Описание |
  |---|---|
  | proxy | ИнтернетПрокси или структура полей Протокол, Сервер, Порт, Пользователь, Пароль, ИспользоватьАутентификациюОС |
  | timeout | Таймаут выполнения запроса |
  | adv_response | Оформляет ответ в виде полной структуры с полями code, body и headers |


  </TabItem>
</Tabs>
  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Green API

:::tip
Метод в документации API: [SetGroupAdmin](https://green-api.com/v3/docs/api/groups/SetGroupAdmin/)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ApiUrl           = "https://3100.api.green-api.com/v3";
    MediaUrl         = "https://3100.api.green-api.com/v3";
    IdInstance       = "31003...";
    ApiTokenInstance = "17915d8e0b4b4975a183478da...";

    IDЧата      = 12345678;
    IDУчастника = 87654321;

    ПараметрыДоступа = OPI_GreenMax.СформироватьПараметрыДоступа(ApiUrl, MediaUrl, IdInstance, ApiTokenInstance);
    Результат        = OPI_GreenMax.НазначитьПраваАдминистратора(ПараметрыДоступа, IDЧата, IDУчастника);
```
