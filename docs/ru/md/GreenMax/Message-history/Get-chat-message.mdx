# Получить сообщение чата
 Получает информацию о сообщении чата по ID



`Функция ПолучитьСообщениеЧата(Знач ПараметрыДоступа, Знач IDЧата, Знач IDСообщения) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ПараметрыДоступа | --access | Структура Из КлючИЗначение | &#x2714; | Параметры доступа. См. [`СформироватьПараметрыДоступа`](/docs/GreenMax/Account/Form-access-parameters) |
  | IDЧата | --chat | Строка | &#x2714; | ID чата |
  | IDСообщения | --message | Строка | &#x2714; | ID сообщения |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Green API

:::tip
Метод в документации API: [GetMessage](https://green-api.com/v3/docs/api/journals/GetMessage/)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ApiUrl           = "https://3100.api.green-api.com/v3";
    MediaUrl         = "https://3100.api.green-api.com/v3";
    IdInstance       = "31003...";
    ApiTokenInstance = "17915d8e0b4b4975a183478da...";

    IDЧата      = 12345678;
    IDСообщения = "2761480287806";

    ПараметрыДоступа = OPI_GreenMax.СформироватьПараметрыДоступа(ApiUrl, MediaUrl, IdInstance, ApiTokenInstance);
    Результат        = OPI_GreenMax.ПолучитьСообщениеЧата(ПараметрыДоступа, IDЧата, IDСообщения);
```
