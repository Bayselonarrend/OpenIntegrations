# Создать элемент
 Создает новый тестовый элемент



`Функция СоздатьЭлемент(Знач URL, Знач Токен, Знач Проект, Знач СтруктураЭлемента, Знач Родитель = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | Проект | --proj | Строка | &#x2714; | ID проекта |
  | СтруктураЭлемента | --params | Структура Из КлючИЗначение | &#x2714; | Параметры элемента. См. [`ПолучитьСтруктуруЭлемента`](/docs/ReportPortal/Results-management/Get-item-structure) |
  | Родитель | --parent | Строка | &#x2716; | ID родительского элемента, если необходимо |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal
