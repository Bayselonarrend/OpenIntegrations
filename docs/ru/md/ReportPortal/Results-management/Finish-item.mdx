# Завершить элемент
 Завершает работу созданного ранее тестового элемента



`Функция ЗавершитьЭлемент(Знач URL, Знач Токен, Знач Проект, Знач IDЭлемента, Знач СтруктураЗавершения) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | Проект | --proj | Строка | &#x2714; | ID проекта |
  | IDЭлемента | --id | Строка | &#x2714; | ID элемента |
  | СтруктураЗавершения | --params | Структура Из КлючИЗначение | &#x2714; | Параметры завершения. См. [`ПолучитьСтруктуруЗавершенияЭлемента`](/docs/ReportPortal/Results-management/Get-item-completion-structure) |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL        = "tests.openintegrations.dev";
    Токен      = "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VyX25hbWUiOiJzdXBlcmFkbWluIiwic2NvcGVzIjpbInVpIl0sImV4cCI6MTc2MjAwMDI3NiwiaWF0IjoxNzYxOTEzODc2LCJqdGkiOiI0NTI3YTc5OS0wN2...";
    IDЭлемента = "";
    Проект     = "Test";

    ДатаЗавершения = Дата("20260101200000");

    СтруктураЗавершения = Новый Структура;
    СтруктураЗавершения.Вставить("endTime", ДатаЗавершения);
    СтруктураЗавершения.Вставить("status" , "passed");

    Результат = OPI_ReportPortal.ЗавершитьЭлемент(URL, Токен, Проект, IDЭлемента, СтруктураЗавершения);
```
