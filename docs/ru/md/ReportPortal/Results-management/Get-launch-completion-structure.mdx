# Получить структуру завершения запуска
 Получает структуру параметров завершения тестового сеанса



`Функция ПолучитьСтруктуруЗавершенияЗапуска(Знач Время, Знач Статус = "", Знач Описание = "", Знач Артибуты = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Время | --end | Дата | &#x2714; | Дата и время завершения запуска |
  | Статус | --status | Строка | &#x2716; | Статус заверщения: passed, failed, stopped, skipped, interrupted, cancelled |
  | Описание | --descr | Строка | &#x2716; | Описание запуска. Перезаписывает указанное при создании |
  | Артибуты | --attr | Строка | &#x2716; | Атрибуты запуска. Перезаписывает указанное при создании |

  
  Возвращаемое значение:   Структура Из КлючИЗначение - Структура полей
