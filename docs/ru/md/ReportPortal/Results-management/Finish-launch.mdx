# Завершить запуск
 Завершает запущенный ранее сеанс выполнения тестов



`Функция ЗавершитьЗапуск(Знач URL, Знач Токен, Знач Проект, Знач IDЗапуска, Знач СтруктураЗавершения) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | Проект | --proj | Строка | &#x2714; | ID проекта |
  | IDЗапуска | --id | Строка | &#x2714; | ID запуска |
  | СтруктураЗавершения | --params | Структура Из КлючИЗначение | &#x2714; | Параметры завершения. См. [`ПолучитьСтруктуруЗавершенияЗапуска`](/docs/ReportPortal/Results-management/Get-launch-completion-structure) |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal
