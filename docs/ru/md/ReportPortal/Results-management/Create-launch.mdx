# Создать запуск
 Запускает новый сеанс выполнения тестов



`Функция СоздатьЗапуск(Знач URL, Знач Токен, Знач Проект, Знач СтруктураЗапуска) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | Проект | --proj | Строка | &#x2714; | ID проекта |
  | СтруктураЗапуска | --params | Структура Из КлючИЗначение | &#x2714; | Параметры запуска. См. [`ПолучитьСтруктуруЗапуска`](/docs/ReportPortal/Results-management/Get-launch-structure) |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal
