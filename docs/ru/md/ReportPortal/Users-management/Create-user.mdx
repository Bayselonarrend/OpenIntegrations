# Создать пользователя
 Создает нового пользователя



`Функция СоздатьПользователя(Знач URL, Знач Токен, Знач СтруктураПользователя) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | СтруктураПользователя | --params | Структура Из КлючИЗначение | &#x2714; | Данные пользователя. См. [`ПолучитьСтруктуруПользователя`](/docs/ReportPortal/Users-management/Get-user-fields-structure) |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal
