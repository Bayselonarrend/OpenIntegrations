# Получить структуру пользователя
 Получает структуру полей описания пользователя



`Функция ПолучитьСтруктуруПользователя(Знач Пустая = Ложь, Знач КакСоответствие = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Пустая | --empty | Булево | &#x2716; | Истина > структура с пустыми значениями, Ложь > в значениях будут описания полей |
  | КакСоответствие | --map | Булево | &#x2716; | Истина > возвращает поля фильтра как соответствие |

  
  Возвращаемое значение:   Структура Из КлючИЗначение - Структура полей



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Результат = OPI_ReportPortal.ПолучитьСтруктуруПользователя();
```
