# Получить структуру лога
 Получает структуру параметров для записи лога



`Функция ПолучитьСтруктуруЛога(Знач IDЗапуска, Знач IDЭлемента, Знач Время, Знач Текст = "", Знач Уровень = "info") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | IDЗапуска | --launch | Строка | &#x2714; | ID запуска, к которому принадлежит тестовый элемент |
  | IDЭлемента | --id | Строка | &#x2714; | ID тестового элемента |
  | Время | --time | Дата | &#x2714; | Дата и время записи лога |
  | Текст | --text | Строка | &#x2716; | Текстовая информация лога |
  | Уровень | --level | Строка | &#x2716; | Уровень информации: error, warn, info, debug, trace, fatal, unknown |

  
  Возвращаемое значение:   Структура Из КлючИЗначение - Структура полей
