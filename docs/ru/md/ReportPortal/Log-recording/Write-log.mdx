# Записать лог
 Добавляет информацию о выполнении для тестового элемента



`Функция ЗаписатьЛог(Знач URL, Знач Токен, Знач Проект, Знач СтруктураЛога) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | Проект | --proj | Строка | &#x2714; | ID проекта |
  | СтруктураЛога | --params | Структура Из КлючИЗначение | &#x2714; | Информация о выполнении. См. [`ПолучитьСтруктуруЛога`](/docs/ReportPortal/Log-recording/Get-log-structure) |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL        = "tests.openintegrations.dev";
    Токен      = "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VyX25hbWUiOiJzdXBlcmFkbWluIiwic2NvcGVzIjpbInVpIl0sImV4cCI6MTc3MjgzMDA3MSwiaWF0IjoxNzcyNzQzNjcxLCJqdGkiOiI3YmRjMDY4Ny0yY2...";
    IDЗапуска  = "4854a293-05e3-4ed5-8903-c397427aeb21";
    IDЭлемента = "83b69c21-82c3-4024-b27f-954d3098657e";
    Проект     = "Test";
    Время      = Дата("20260101100010");
    Текст      = "Some log 1";

    СтруктураЛога = OPI_ReportPortal.ПолучитьСтруктуруЛога(IDЗапуска, IDЭлемента, Время, Текст);
```
