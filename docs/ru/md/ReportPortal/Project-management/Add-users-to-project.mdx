# Добавить пользователей в проект
 Добавляет пользователей в указанный проект



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ДобавитьПользователейВПроект(Знач URL, Знач Токен, Знач ИмяПроекта, Знач СписокПользователей) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | ИмяПроекта | --proj | Строка | &#x2714; | Имя проекта |
  | СписокПользователей | --users | Соответствие Из КлючИЗначение | &#x2714; | Пользователи: Ключ > имя, Значение > роль |


<div className="return-value-note">
