# Исключить пользователей из проекта
 Исключает пользователей из указанного проекта



`Функция ИсключитьПользователейИзПроекта(Знач URL, Знач Токен, Знач ИмяПроекта, Знач МассивПользователей) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | ИмяПроекта | --proj | Строка | &#x2714; | Имя проекта |
  | МассивПользователей | --users | Массив Из Строка | &#x2714; | Массив имен пользователей |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL        = "tests.openintegrations.dev";
    Токен      = "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VyX25hbWUiOiJzdXBlcmFkbWluIiwic2NvcGVzIjpbInVpIl0sImV4cCI6MTc2MjAwMDI3NiwiaWF0IjoxNzYxOTEzODc2LCJqdGkiOiI0NTI3YTc5OS0wN2...";
    ИмяПроекта = "TestProject";

    МассивПользователей = Новый Массив;
    МассивПользователей.Добавить("default");

    Результат = OPI_ReportPortal.ИсключитьПользователейИзПроекта(URL, Токен, ИмяПроекта, МассивПользователей);
```
