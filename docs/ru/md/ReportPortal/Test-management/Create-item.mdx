# Создать элемент
 Создает новый тестовый элемент



`Функция СоздатьЭлемент(Знач URL, Знач Токен, Знач Проект, Знач СтруктураЭлемента, Знач Родитель = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | Проект | --proj | Строка | &#x2714; | ID проекта |
  | СтруктураЭлемента | --params | Структура Из КлючИЗначение | &#x2714; | Параметры элемента. См. [`ПолучитьСтруктуруЭлемента`](/docs/ReportPortal/Test-management/Get-item-structure) |
  | Родитель | --parent | Строка | &#x2716; | ID родительского элемента, если необходимо |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL       = "tests.openintegrations.dev";
    Токен     = "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VyX25hbWUiOiJzdXBlcmFkbWluIiwic2NvcGVzIjpbInVpIl0sImV4cCI6MTc2MjAwMDI3NiwiaWF0IjoxNzYxOTEzODc2LCJqdGkiOiI0NTI3YTc5OS0wN2...";
    IDЗапуска = "";
    Проект    = "Test";

    ДатаНачала = Дата("20260101100000");

    СтруктураЭлемента = Новый Структура;
    СтруктураЭлемента.Вставить("name"       , "Test");
    СтруктураЭлемента.Вставить("startTime"  , ДатаНачала);
    СтруктураЭлемента.Вставить("type"       , "test");
    СтруктураЭлемента.Вставить("description", "Test launch");
    СтруктураЭлемента.Вставить("launchUuid" , IDЗапуска);

    Результат = OPI_ReportPortal.СоздатьЭлемент(URL, Токен, Проект, СтруктураЭлемента);
```
