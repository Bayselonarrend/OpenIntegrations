# Создать запуск
 Запускает новый сеанс выполнения тестов



`Функция СоздатьЗапуск(Знач URL, Знач Токен, Знач Проект, Знач СтруктураЗапуска) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | Проект | --proj | Строка | &#x2714; | ID проекта |
  | СтруктураЗапуска | --params | Структура Из КлючИЗначение | &#x2714; | Параметры запуска. См. [`ПолучитьСтруктуруЗапуска`](/docs/ReportPortal/Test-management/Get-launch-structure) |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL    = "tests.openintegrations.dev";
    Токен  = "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VyX25hbWUiOiJzdXBlcmFkbWluIiwic2NvcGVzIjpbInVpIl0sImV4cCI6MTc2MjAwMDI3NiwiaWF0IjoxNzYxOTEzODc2LCJqdGkiOiI0NTI3YTc5OS0wN2...";
    Проект = "Test";

    ДатаНачала = Дата("20260101100000");

    СтруктураЗапуска = Новый Структура;
    СтруктураЗапуска.Вставить("name"       , "Test");
    СтруктураЗапуска.Вставить("startTime"  , ДатаНачала);
    СтруктураЗапуска.Вставить("description", "Test launch");

    Результат = OPI_ReportPortal.СоздатьЗапуск(URL, Токен, Проект, СтруктураЗапуска);
```
