# Удалить элемент
 Удаляет тестовый элемент по ID



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция УдалитьЭлемент(Знач URL, Знач Токен, Знач Проект, Знач IDЭлемента) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | Проект | --proj | Строка | &#x2714; | ID проекта |
  | IDЭлемента | --id | Число | &#x2714; | ID тестового элемента |


<div className="return-value-note">

  </div>

:::
