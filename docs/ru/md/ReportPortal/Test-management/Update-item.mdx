# Изменить элемент
 Изменяет данные тестового элемента по ID



`Функция ИзменитьЭлемент(Знач URL, Знач Токен, Знач Проект, Знач IDЭлемента, Знач СтруктураЭлемента) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | Проект | --proj | Строка | &#x2714; | ID проекта |
  | IDЭлемента | --id | Число | &#x2714; | ID тестового элемента |
  | СтруктураЭлемента | --params | Структура Из КлючИЗначение | &#x2714; | Параметры элемента. См. [`ПолучитьСтруктуруЭлемента`](/docs/ReportPortal/Test-management/Get-item-structure) |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal

:::tip
Числовой ID элемента можно получить по UUID при помощи функции `ПолучитьЭлемент`
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL        = "tests.openintegrations.dev";
    Токен      = "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VyX25hbWUiOiJzdXBlcmFkbWluIiwic2NvcGVzIjpbInVpIl0sImV4cCI6MTc2MjAwMDI3NiwiaWF0IjoxNzYxOTEzODc2LCJqdGkiOiI0NTI3YTc5OS0wN2...";
    IDЭлемента = "";
    Проект     = "Test";

    СтруктураЗапуска = Новый Структура;
    СтруктураЗапуска.Вставить("description", "New description");

    Результат = OPI_ReportPortal.ИзменитьЭлемент(URL, Токен, Проект, IDЭлемента, СтруктураЗапуска);
```
