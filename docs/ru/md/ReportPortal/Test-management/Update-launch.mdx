# Изменить запуск
 Изменяет данные запуска сеанса тестирования



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ИзменитьЗапуск(Знач URL, Знач Токен, Знач Проект, Знач IDЗапуска, Знач СтруктураЗапуска) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | Проект | --proj | Строка | &#x2714; | ID проекта |
  | IDЗапуска | --id | Число | &#x2714; | ID запуска |
  | СтруктураЗапуска | --params | Структура Из КлючИЗначение | &#x2714; | Параметры запуска. См. [`ПолучитьСтруктуруЗапуска`](/docs/ReportPortal/Test-management/Get-launch-structure) |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | proxy | ИнтернетПрокси или структура полей Протокол, Сервер, Порт, Пользователь, Пароль, ИспользоватьАутентификациюОС |
  | timeout | Таймаут выполнения запроса |
  | adv_response | Оформляет ответ в виде полной структуры с полями code, body и headers |


  </TabItem>
</Tabs>

:::tip
Числовой ID запуска можно получить по UUID при помощи функции `ПолучитьЗапуск`
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL       = "tests.openintegrations.dev";
    Токен     = "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VyX25hbWUiOiJzdXBlcmFkbWluIiwic2NvcGVzIjpbInVpIl0sImV4cCI6MTc2MjAwMDI3NiwiaWF0IjoxNzYxOTEzODc2LCJqdGkiOiI0NTI3YTc5OS0wN2...";
    IDЗапуска = "";
    Проект    = "Test";

    СтруктураЗапуска = Новый Структура;
    СтруктураЗапуска.Вставить("description", "New description");

    Результат = OPI_ReportPortal.ИзменитьЗапуск(URL, Токен, Проект, IDЗапуска, СтруктураЗапуска);
```
