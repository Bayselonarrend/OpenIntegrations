# Получить постоянный токен
 Создает постоянный токен пользователя ReportPortal



`Функция ПолучитьПостоянныйТокен(Знач URL, Знач Токен, Знач IDПользователя, Знач ИмяКлюча) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | IDПользователя | --user | Строка | &#x2714; | ID пользователя |
  | ИмяКлюча | --name | Строка | &#x2714; | Имя токена |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL            = "tests.openintegrations.dev";
    Токен          = "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VyX25hbWUiOiJzdXBlcmFkbWluIiwic2NvcGVzIjpbInVpIl0sImV4cCI6MTc2NTQ4MjQ4OSwiaWF0IjoxNzY1Mzk2MDg5LCJqdGkiOiJiMTdmN2JkMS1hYj...";
    IDПользователя = "1";
    ИмяКлюча       = "test_key";

    Результат = OPI_ReportPortal.ПолучитьПостоянныйТокен(URL, Токен, IDПользователя, ИмяКлюча);
```
