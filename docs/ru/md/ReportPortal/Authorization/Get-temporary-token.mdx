# Получить временный токен
 Получает временный токен на основе логина и пароля



`Функция ПолучитьВременныйТокен(Знач URL, Знач Логин, Знач Пароль) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Логин | --login | Строка | &#x2714; | Логин пользователя |
  | Пароль | --pass | Строка | &#x2714; | Пароль пользователя |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal

<br/>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL    = "tests.openintegrations.dev";
    Логин  = "superadmin";
    Пароль = "Rit2...";

    Результат = OPI_ReportPortal.ПолучитьВременныйТокен(URL, Логин, Пароль);
```
