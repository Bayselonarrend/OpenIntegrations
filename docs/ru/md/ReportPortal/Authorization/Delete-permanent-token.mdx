# Удалить постоянный токен
 Удаляет ранее созданный токен пользователя



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция УдалитьПостоянныйТокен(Знач URL, Знач Токен, Знач IDПользователя, Знач IDКлюча) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | IDПользователя | --user | Строка | &#x2714; | ID пользователя |
  | IDКлюча | --id | Строка | &#x2714; | ID токена |
