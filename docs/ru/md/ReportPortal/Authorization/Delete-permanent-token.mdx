# Удалить постоянный токен
 Удаляет ранее созданный токен пользователя



`Функция УдалитьПостоянныйТокен(Знач URL, Знач Токен, Знач IDПользователя, Знач IDКлюча) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера ReportPortal |
  | Токен | --token | Строка | &#x2714; | Токен доступа |
  | IDПользователя | --user | Строка | &#x2714; | ID пользователя |
  | IDКлюча | --id | Строка | &#x2714; | ID токена |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от ReportPortal



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL            = "tests.openintegrations.dev";
    Токен          = "test-key_vrOtLPcVRa-X93ekUYBuPhHpoiO...";
    IDПользователя = "1";
    IDКлюча        = "1...";

    Результат = OPI_ReportPortal.УдалитьПостоянныйТокен(URL, Токен, IDПользователя, IDКлюча);
```
