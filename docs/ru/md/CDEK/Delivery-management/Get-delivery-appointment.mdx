# Получить договоренность о доставке
 Получает информацию об оговоренных с клиентом дате, времени и адресе доставки



`Функция ПолучитьДоговоренностьОДоставке(Знач Токен, Знач UUID, Знач ТестовыйAPI = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен авторизации |
  | UUID | --uuid | Строка | &#x2714; | UUID договоренности |
  | ТестовыйAPI | --testapi | Булево | &#x2716; | Флаг использования API для тестовых запросов |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от CDEK

<br/>

:::tip
Договоренность должна быть предварительно создана. См. ЗафиксироватьДоговоренностьОДоставке

Метод в документации API: [Информация о договоренности о доставке](https://api-docs.cdek.ru/36981626.html)
:::
<br/>
