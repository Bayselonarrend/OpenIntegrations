# Получить договоренность о доставке
 Получает информацию об оговоренных с клиентом дате, времени и адресе доставки



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьДоговоренностьОДоставке(Знач Токен, Знач UUID, Знач ТестовыйAPI = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен авторизации |
  | UUID | --uuid | Строка | &#x2714; | UUID договоренности |
  | ТестовыйAPI | --testapi | Булево | &#x2716; | Флаг использования API для тестовых запросов |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - сериализованный JSON ответа от CDEK
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его объект (только для 1С и OneScript) |


  </TabItem>
</Tabs>

:::tip
Договоренность должна быть предварительно создана. См. ЗафиксироватьДоговоренностьОДоставке

Метод в документации API: [Информация о договоренности о доставке](https://api-docs.cdek.ru/36981626.html)
:::
<br/>
