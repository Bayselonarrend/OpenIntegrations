# Получить заявку на вызов курьера
 Получает заявку на вызов курьера по UUID



`Функция ПолучитьЗаявкуНаВызовКурьера(Знач Токен, Знач UUID, Знач ТестовыйAPI = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен авторизации |
  | UUID | --uuid | Строка | &#x2714; | UUID заявки |
  | ТестовыйAPI | --testapi | Булево | &#x2716; | Флаг использования API для тестовых запросов |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от CDEK

:::tip
Заявка должна быть предварительно создана. См. СоздатьЗаявкуНаВызовКурьера

Метод в документации API: [Информация о заявке](https://api-docs.cdek.ru/29948360.html)
:::
<br/>
