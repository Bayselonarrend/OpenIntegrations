# Получить чек заказа
 Получает информацию о чеке по заказу



`Функция ПолучитьЧекЗаказа(Знач Токен, Знач UUID, Знач ТестовыйAPI = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен авторизации |
  | UUID | --uuid | Строка | &#x2714; | UUID заказа |
  | ТестовыйAPI | --testapi | Булево | &#x2716; | Флаг использования API для тестовых запросов |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от CDEK

<br/>

:::tip
Метод в документации API: [Информация о чеке](https://api-docs.cdek.ru/68257388.html)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен = ПараметрыФункции["CDEK_Token"];
    UUID  = ПараметрыФункции["CDEK_OrderUUID"];

    Результат = OPI_CDEK.ПолучитьЧекЗаказа(Токен, UUID, Истина);
```
