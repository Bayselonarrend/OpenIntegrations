# Получить чек заказа
 Получает информацию о чеке по заказу



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьЧекЗаказа(Знач Токен, Знач UUID, Знач ТестовыйAPI = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен авторизации |
  | UUID | --uuid | Строка | &#x2714; | UUID заказа |
  | ТестовыйAPI | --testapi | Булево | &#x2716; | Флаг использования API для тестовых запросов |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - сериализованный JSON ответа от CDEK
  </div>
</div>

  |---|---|
  | dontwait | Создает фоновое задание и возвращает его объект (только для 1С и OneScript) |
