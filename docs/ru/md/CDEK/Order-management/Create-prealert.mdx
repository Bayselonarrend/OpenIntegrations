# Создать преалерт
 Создает преалерт для списка заказов



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция СоздатьПреалерт(Знач Токен, Знач МассивUUID, Знач ДатаПередачи, Знач ПВЗ, Знач ТестовыйAPI = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен авторизации |
  | МассивUUID | --uuids | Строка, Массив Из Строка | &#x2714; | Один или массив UUID заказов |
  | ДатаПередачи | --date | Дата | &#x2714; | Планируемая дата передачи заказов в СДЭК |
  | ПВЗ | --point | Строка | &#x2714; | Код ПВЗ, в который планируется передать заказы |
  | ТестовыйAPI | --testapi | Булево | &#x2716; | Флаг использования API для тестовых запросов |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - сериализованный JSON ответа от CDEK
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | proxy | `ИнтернетПрокси` или структура полей `Протокол`, `Сервер`, `Порт`, `Пользователь`, `Пароль`, `ИспользоватьАутентификациюОС` |
  | timeout | Таймаут выполнения запроса |
  | adv_response | Оформляет ответ в виде полной структуры HTTP-ответа с полями `code`, `body` и `headers` |
  | retries | Число попыток отправки HTTP-запроса при коде `5**` или внутренних ошибках клиента |
  | dontwait | Создает фоновое задание и возвращает его данные (только для 1С и OneScript) |


  </TabItem>
</Tabs>

:::tip
Метод в документации API: [Регистрация преалерта](https://api-docs.cdek.ru/78934763.html)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен        = ПараметрыФункции["CDEK_Token"];
    UUID         = ПараметрыФункции["CDEK_OrderUUID"];
    ПВЗ          = "NSK27";
    ДатаПередачи = OPI_Инструменты.ПолучитьТекущуюДату() + 60 * 60 * 24;

    Результат = OPI_CDEK.СоздатьПреалерт(Токен, UUID, ДатаПередачи, ПВЗ, Истина);
```
