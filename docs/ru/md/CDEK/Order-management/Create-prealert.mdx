# Создать преалерт
 Создает преалерт для списка заказов



`Функция СоздатьПреалерт(Знач Токен, Знач МассивUUID, Знач ДатаПередачи, Знач ПВЗ, Знач ТестовыйAPI = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен авторизации |
  | МассивUUID | --uuids | Строка, Массив Из Строка | &#x2714; | Один или массив UUID заказов |
  | ДатаПередачи | --date | Дата | &#x2714; | Планируемая дата передачи заказов в СДЭК |
  | ПВЗ | --point | Строка | &#x2714; | Код ПВЗ, в который планируется передать заказы |
  | ТестовыйAPI | --testapi | Булево | &#x2716; | Флаг использования API для тестовых запросов |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от CDEK

:::tip
Метод в документации API: [Регистрация преалерта](https://api-docs.cdek.ru/78934763.html)
:::
