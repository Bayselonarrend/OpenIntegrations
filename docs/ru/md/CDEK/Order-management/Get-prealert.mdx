# Получить преалерт
 Получает ранее созданный преалерт



`Функция ПолучитьПреалерт(Знач Токен, Знач UUID, Знач ТестовыйAPI = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен авторизации |
  | UUID | --uuid | Строка | &#x2714; | UUID преалерта |
  | ТестовыйAPI | --testapi | Булево | &#x2716; | Флаг использования API для тестовых запросов |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от CDEK

:::tip
Преалерт должен быть предварительно создан. См. СоздатьПреалерт

Метод в документации API: [Информация о преалерте](https://api-docs.cdek.ru/78934799.html)
:::
