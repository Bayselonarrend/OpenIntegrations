# Получить список офисов
 Получает список офисов с фильтром или без



`Функция ПолучитьСписокОфисов(Знач Токен, Знач Фильтр = "", Знач ТестовыйAPI = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен авторизации |
  | Фильтр | --filter | Структура Из КлючИЗначение | &#x2716; | Фильтр офисов. См. ПолучитьОписаниеФильтраОфисов |
  | ТестовыйAPI | --testapi | Булево | &#x2716; | Флаг использования API для тестовых запросов |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от CDEK

<br/>

:::tip
Метод в документации API: [Список офисов](https://api-docs.cdek.ru/36982648.html)
:::
<br/>
