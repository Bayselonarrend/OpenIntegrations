# Получить список регионов
 Получает список доступных регионов



`Функция ПолучитьСписокРегионов(Знач Токен, Знач Страны = Неопределено, Знач Страница = 0, Знач Язык = "rus", Знач ТестовыйAPI = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен авторизации |
  | Страны | --countries | Массив Из Строка | &#x2716; | Массив кодов стран в формате ISO_31661_alpha2 для отбора |
  | Страница | --page | Число | &#x2716; | Страница выборки результата |
  | Язык | --lang | Строка | &#x2716; | Язык: rus, eng, zho |
  | ТестовыйAPI | --testapi | Булево | &#x2716; | Флаг использования API для тестовых запросов |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от CDEK

:::tip
Метод в документации API: [Список регионов](https://api-docs.cdek.ru/33829418.html)
:::
