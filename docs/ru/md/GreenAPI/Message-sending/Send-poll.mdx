# Отправить опрос
 Отправляет опрос с вариантами ответов в выбранный чат



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ОтправитьОпрос(Знач ПараметрыДоступа, Знач IDЧата, Знач Текст, Знач Варианты, Знач МножественныйВыбор = Ложь, Знач IDЦитируемого = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ПараметрыДоступа | --access | Структура Из КлючИЗначение | &#x2714; | Параметры доступа. См. [`СформироватьПараметрыДоступа`](/docs/GreenAPI/Account/Form-access-parameters) |
  | IDЧата | --chat | Строка | &#x2714; | Идентификатор чата |
  | Текст | --text | Строка | &#x2714; | Текст сообщения |
  | Варианты | --options | Массив Из Строка | &#x2714; | Варианты ответов |
  | МножественныйВыбор | --multi | Булево | &#x2716; | Позволяет выбирать несколько вариантов ответа |
  | IDЦитируемого | --quoted | Строка | &#x2716; | ID цитируемого сообщения, если необходимо |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - сериализованный JSON ответа от Green API
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | proxy | ИнтернетПрокси или структура полей Протокол, Сервер, Порт, Пользователь, Пароль, ИспользоватьАутентификациюОС |
  | timeout | Таймаут выполнения запроса |
  | adv_response | Оформляет ответ в виде полной структуры с полями code, body и headers |
  | retries | Число попыток отправки при коде `5**` или внутренних ошибках клиента |
  | dontwait | Создает фоновое задание и возвращает его объект (только для 1С и OneScript) |


  </TabItem>
</Tabs>

:::tip
Метод в документации API: [SendPoll](https://green-api.com/docs/api/sending/SendPoll/)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ApiUrl           = "https://7105.api.greenapi.com";
    MediaUrl         = "https://7105.media.greenapi.com";
    IdInstance       = "71051...";
    ApiTokenInstance = "425010d90e114aa6b78f0969e...";

    IDЧата = "120363410406221140@g.us";
    Текст  = "Какой ваш любимый цвет?";

    Варианты = Новый Массив;
    Варианты.Добавить("Красный");
    Варианты.Добавить("Желтый");
    Варианты.Добавить("Зеленый");

    ПараметрыДоступа = OPI_GreenAPI.СформироватьПараметрыДоступа(ApiUrl, MediaUrl, IdInstance, ApiTokenInstance);
    Результат        = OPI_GreenAPI.ОтправитьОпрос(ПараметрыДоступа, IDЧата, Текст, Варианты);
```
