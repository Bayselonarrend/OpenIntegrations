# Получить описание контакта
 Получает описание контакта для функции ОтправитьКонстакт



`Функция ПолучитьОписаниеКонтакта(Знач Телефон, Знач Имя = "", Знач Фамилия = "", Знач Отчество = "", Знач Компания = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Телефон | --phone | Число | &#x2714; | Телефон в международном формате без знака плюс |
  | Имя | --name | Строка | &#x2716; | Имя контакта |
  | Фамилия | --surname | Строка | &#x2716; | Фамилия контакта |
  | Отчество | --midname | Строка | &#x2716; | Отчество контакта |
  | Компания | --company | Строка | &#x2716; | Название компании контакта |

  
  Возвращаемое значение:   Структура Из КлючИЗначение -  Описание контакта



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Телефон  = 79001234568;
    Имя      = "Артем";
    Фамилия  = "Евпаторийский";
    Отчество = "Петрович";
    Компания = "Велосипед";

    Результат = OPI_GreenAPI.ПолучитьОписаниеКонтакта(Телефон, Имя, Фамилия, Отчество, Компания);
```
