# Получить описание локации
 Получает описание локации для отправки в ОтправитьЛокацию



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьОписаниеЛокации(Знач Широта, Знач Долгота, Знач Адрес = "", Знач Название = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Широта | --lat | Число | &#x2714; | Географическая широта |
  | Долгота | --long | Число | &#x2714; | Географическая долгота |
  | Адрес | --addr | Строка | &#x2716; | Адрес локации |
  | Название | --name | Строка | &#x2716; | Название локации |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Структура Из КлючИЗначение - Описание локации
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его объект (только для 1С и OneScript) |


  </TabItem>
</Tabs>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Широта   = 53.908522;
    Долгота  = 27.574821;
    Адрес    = "Площадь Победы, Минск";
    Название = "пл. Победы";

    Результат = OPI_GreenAPI.ПолучитьОписаниеЛокации(Широта, Долгота, Адрес, Название);
```
    

 <Tabs>
  
    <TabItem value="bash" label="Bash" default>
