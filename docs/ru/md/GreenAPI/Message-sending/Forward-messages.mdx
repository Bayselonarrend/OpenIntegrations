# Переслать сообщения
 Пересылает сообщения из одного чата в другой



`Функция ПереслатьСообщения(Знач ПараметрыДоступа, Знач Откуда, Знач Куда, Знач Сообщения) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ПараметрыДоступа | --access | Структура Из КлючИЗначение | &#x2714; | Параметры доступа. См. [`СформироватьПараметрыДоступа`](/docs/GreenAPI/Account/Form-access-parameters) |
  | Откуда | --from | Строка | &#x2714; | ID чата источника сообщения |
  | Куда | --to | Строка | &#x2714; | ID чата приемника сообщения |
  | Сообщения | --msgs | Строка, Массив Из Строка | &#x2714; | Сообщения или одно сообщения для отправки |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Green API

:::tip
Метод в документации API: [ForwardMessages](https://green-api.com/docs/api/sending/ForwardMessages/)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ApiUrl           = "https://7105.api.greenapi.com";
    MediaUrl         = "https://7105.media.greenapi.com";
    IdInstance       = "71051...";
    ApiTokenInstance = "425010d90e114aa6b78f0969e...";

    Откуда = "11001234567@c.us";
    Куда   = "120363410406221140@g.us";

    Сообщение = "3EB05EA43320C6336EF896";

    ПараметрыДоступа = OPI_GreenAPI.СформироватьПараметрыДоступа(ApiUrl, MediaUrl, IdInstance, ApiTokenInstance);
    Результат        = OPI_GreenAPI.ПереслатьСообщения(ПараметрыДоступа, Откуда, Куда, Сообщение);
```
