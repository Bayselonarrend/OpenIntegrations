# Отправить файл по URL
 Отправляет файл по URL в выбранный чат



`Функция ОтправитьФайлПоURL(Знач ПараметрыДоступа, Знач IDЧата, Знач URLФайла, Знач ИмяФайла, Знач Описание = "", Знач IDЦитируемого = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ПараметрыДоступа | --access | Структура Из КлючИЗначение | &#x2714; | Параметры доступа. См. [`СформироватьПараметрыДоступа`](/docs/GreenAPI/Account/Form-access-parameters) |
  | IDЧата | --chat | Строка | &#x2714; | Идентификатор чата |
  | URLФайла | --url | Строка | &#x2714; | URL файла для отправки |
  | ИмяФайла | --filename | Строка | &#x2714; | Имя загружаемого файла с раширением |
  | Описание | --caption | Строка | &#x2716; | Текст сообщения под файлом |
  | IDЦитируемого | --quoted | Строка | &#x2716; | ID цитируемого сообщения, если необходимо |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Green API

:::tip
Метод в документации API: [SendFileByUrl](https://green-api.com/docs/api/sending/SendFileByUrl/)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ApiUrl           = "https://7105.api.greenapi.com";
    MediaUrl         = "https://7105.media.greenapi.com";
    IdInstance       = "71051...";
    ApiTokenInstance = "425010d90e114aa6b78f0969e...";

    Файл     = "https://hut.openintegrations.dev/test_data/picture.jpg";
    ИмяФайла = "photo.jpg";
    IDЧата   = "120363410406221140@g.us";
    Описание = "Описание файла";

    ПараметрыДоступа = OPI_GreenAPI.СформироватьПараметрыДоступа(ApiUrl, MediaUrl, IdInstance, ApiTokenInstance);
    Результат        = OPI_GreenAPI.ОтправитьФайлПоURL(ПараметрыДоступа, IDЧата, Файл, ИмяФайла, Описание);
```
