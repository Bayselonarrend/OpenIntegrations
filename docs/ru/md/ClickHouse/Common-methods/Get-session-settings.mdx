# Получить настройки сессии
 Формирует структуру настроек сессии выполнения запроса



`Функция ПолучитьНастройкиСессии(Знач IDСессии = Неопределено, Знач ПроверятьСессию = Неопределено, Знач Таймаут = 60) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | IDСессии | --id | Строка | &#x2716; | ID сессии. Новый уникальный идентификатор, если не указано (будет создана новая сессия) |
  | ПроверятьСессию | --check | Булево | &#x2716; | Проверять существование сессии. Истина, когда ID указан и ложь, когда нет, если не указано иное |
  | Таймаут | --timeout | Число | &#x2716; | Время жизни сессии в секундах |

  
  Возвращаемое значение:   Структура Из КлючИЗначение - Структура настроек сессии



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    IDСессии  = Строка(Новый УникальныйИдентификатор);
    Проверять = Истина;
    Таймаут   = 120;

    Результат = OPI_ClickHouse.ПолучитьНастройкиСессии(IDСессии, Проверять, Таймаут);
```
