# Установить Webhook
 Устанавливает Webhook для приема сообщений



`Функция УстановитьWebhook(Знач Токен, Знач URL) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен Viber |
  | URL | --url | Строка | &#x2714; | URL для установки Webhook |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Viber



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен = "523b58ba82afffaa-7ef3b426...";
    URL   = "http://api.athenaeum.digital/hs/viber";

    Результат = OPI_Viber.УстановитьWebhook(Токен, URL);
```
