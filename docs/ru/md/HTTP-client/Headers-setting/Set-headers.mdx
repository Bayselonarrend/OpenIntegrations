# Установить заголовки 
 Устанавливает коллекцию заголовков запросов



`Функция УстановитьЗаголовки(Знач Значение, Знач ПолнаяЗамена = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Значение | - | Произвольный | &#x2714; | Стрктура или соответствие заголовков запрос |
  | ПолнаяЗамена | - | Булево | &#x2716; | Очищает все существующие заголовки перед установкой |

  
  Возвращаемое значение:   ОбработкаОбъект.OPI_HTTPКлиент -  Этот же объект обработки

<br/>

:::tip
`ПолнаяЗамена` также очищает заголовки, установленные ранее другими способами (например, заголовки авторизации)
:::

:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
