# Отправить данные частями 
 Отправляет тело несколькми запросами с заголовком Content-Range



`Функция ОтправитьДанныеЧастями(Знач РазмерЧасти = 5242880, Знач Метод = "PUT") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | РазмерЧасти | - | Число | &#x2716; | Размер одной части для отправки |
  | Метод | - | Строка | &#x2716; | HTTP метод запроса |

  
  Возвращаемое значение:   ОбработкаОбъект.OPI_HTTPКлиент -  Этот же объект обработки

<br/>


:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
