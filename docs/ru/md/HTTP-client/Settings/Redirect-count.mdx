# Количество переадресаций 
 Устанавливает максимальное число допустимых переадресаций



`Функция КоличествоПереадресаций(Знач Значение) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Значение | - | Число | &#x2714; | Количество переадресаций |

  
  Возвращаемое значение:   ОбработкаОбъект.OPI_HTTPКлиент -  Этот же объект обработки

:::tip
По умолчанию: `5`
:::

:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
