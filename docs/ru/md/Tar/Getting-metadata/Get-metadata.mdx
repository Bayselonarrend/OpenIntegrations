# Получить метаданные
 Получает метаданные и плоский список файлов архива



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьМетаданные(Знач Архив) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Архив | --src | Строка, ДвоичныеДанные | &#x2714; | Путь к архиву на диске или двоичные данные |

  | Параметр | Описание |
  |---|---|

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ПутьАрхива = Параметры["Tar_ArchivePath"];

    Результат = OPI_Tar.ПолучитьМетаданные(ПутьАрхива);
```
