# Разархивировать каталог
 Распаковывает tar-архив в указанный каталог



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция РазархивироватьКаталог(Знач Архив, Знач КаталогНазначения = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Архив | --src | Строка, ДвоичныеДанные | &#x2714; | Путь к архиву на диске или двоичные данные |
  | КаталогНазначения | --dest | Строка | &#x2716; | Путь каталога для распаковки. В соответствие, если не заполнено |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - Информация о выполнении
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его данные (только для 1С и OneScript) |


  </TabItem>

:::tip
Параметры с типом данных Двоичные данные могут принимать также пути к файлам на диске и URL
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ПутьАрхива        = Параметры["Tar_ArchivePath"];
    КаталогНазначение = Параметры["Tar_DestDir"];

    Результат = OPI_Tar.РазархивироватьКаталог(ПутьАрхива, КаталогНазначение);
```
