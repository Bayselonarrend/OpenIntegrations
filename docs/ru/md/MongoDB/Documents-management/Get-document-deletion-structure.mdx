# Получить структуру удаления документов
 Формирует структу данных для обновления



`Функция ПолучитьСтруктуруУдаленияДокументов(Знач Фильтр, Знач Количество = 0, Сопоставление = Неопределено) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Фильтр | --query | Структура Из КлючИЗначение | &#x2714; | Фильтр документов |
  | Количество | --limit | Число | &#x2716; | Лимит удаления: 1 > первый документ, 0 > все документы |
  | Сопоставление | --cltn | Структура Из КлючИЗначение | &#x2716; | Особые настройки правил сравнения полей при поиске |

  
  Возвращаемое значение:   Структура - Получить структуру удаления документов



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Фильтр    = Новый Структура("stringField,doubleField", "Текст", 999);
    Результат = OPI_MongoDB.ПолучитьСтруктуруУдаленияДокументов(Фильтр, 1); // Массив или одиночное
```
    



```json title="Результат"
{
 "q": {
  "stringField": "Текст",
  "doubleField": 999
 },
 "limit": 1
}
```
