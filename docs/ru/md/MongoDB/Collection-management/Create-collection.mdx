# Создать коллекцию
 Создает новую коллекцию с указанными параметрами



`Функция СоздатьКоллекцию(Знач Соединение, Знач Наименование, Знач База = Неопределено, Знач Параметры = Неопределено) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Соединение | --dbc | Строка, Произвольный | &#x2714; | Соединение или строка подключения |
  | Наименование | --name | Строка | &#x2714; | Имя новой коллекции |
  | База | --db | Строка | &#x2716; | Имя базы. Текущая база, если не указано |
  | Параметры | --params | Структура Из КлючИЗначение | &#x2716; | Дополнительные параметры создания |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - Результат выполнения операции
