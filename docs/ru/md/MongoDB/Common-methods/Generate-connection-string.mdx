# Сформировать строку подключения
 Формирует строку для подключения к серверу MongoDB



`Функция СформироватьСтрокуПодключения(Знач Адрес, Знач База = "", Знач Логин = "", Знач Пароль = "", Знач Параметры = Неопределено) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Адрес | --addr | Строка | &#x2714; | IP адрес с портом или доменное имя для подключения |
  | База | --db | Строка | &#x2716; | База для подключения, если необходимо |
  | Логин | --usr | Строка | &#x2716; | Имя пользователя авторизации |
  | Пароль | --pwd | Строка | &#x2716; | Пароль пользователя для авторизации |
  | Параметры | --params | Структура Из КлючИЗначение, Неопределено | &#x2716; | Дополнительные параметры подключения |

  
  Возвращаемое значение:   Строка - Строка подключения MongoDB

<br/>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Адрес  = "127.0.0.1:1234";
    Логин  = "bayselonarrend";
    Пароль = "12we...";
    База   = "main";
```
