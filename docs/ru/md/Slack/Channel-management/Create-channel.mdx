# Создать канал
 Создает новый канал



`Функция СоздатьКанал(Знач Токен, Знач Название, Знач Приватный = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | Название | --title | Строка | &#x2714; | Наименование канала |
  | Приватный | --private | Булево | &#x2716; | Создать канал приватным |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Slack

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен = "xoxb-6965308400114-696804637...";
    Имя   = "testconv" + Строка(Новый УникальныйИдентификатор);

    Результат = OPI_Slack.СоздатьКанал(Токен, Имя);
```
