# Выгнать пользователя из канала
 Удаляет указанного пользователя из канала



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ВыгнатьПользователяИзКанала(Знач Токен, Знач Канал, Знач Пользователь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | Канал | --channel | Строка | &#x2714; | ID канала |
  | Пользователь | --user | Строка | &#x2714; | ID пользователя |
