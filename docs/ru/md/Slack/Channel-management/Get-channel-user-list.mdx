# Получить список пользователей канала
 Получает список пользователей указанного канала



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьСписокПользователейКанала(Знач Токен, Знач Канал, Знач Курсор = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | Канал | --channel | Строка | &#x2714; | Идентификатор канала |
  | Курсор | --cursor | Строка | &#x2716; | Указатель из предыдущего запроса, если строк результата > 100 |

  Соответствие Из КлючИЗначение - сериализованный JSON ответа от Slack
  </div>

  | Параметр | Описание |
  |---|---|
  | proxy | ИнтернетПрокси или структура полей Протокол, Сервер, Порт, Пользователь, Пароль, ИспользоватьАутентификациюОС |

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен = "xoxb-6965308400114-696804637...";
    Канал = "C09PUMJU2GM";

    Результат = OPI_Slack.ПолучитьСписокПользователейКанала(Токен, Канал);
```
