# Получить список внешних файлов
 Получает список внешних файлов пользователя или канала

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | Канал | --channel | Строка | &#x2716; | Канал для отбора |
  | Курсор | --cursor | Строка | &#x2716; | Указатель из предыдущего запроса, если строк результата > 100 |
