# Получить список рабочих областей
 Получает список рабочих областей, в которых подключен бот



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьСписокРабочихОбластей(Знач Токен, Знач Курсор = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | Курсор | --cursor | Строка | &#x2716; | Указатель из предыдущего запроса, если строк результата > 100 |
