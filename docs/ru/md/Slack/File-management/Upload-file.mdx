# Загрузить файл
 Загружает файл на сервера Slack



`Функция ЗагрузитьФайл(Знач Токен, Знач Файл, Знач ИмяФайла, Знач Заголовок, Знач Канал = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | Файл | --file | Строка,ДвоичныеДанные | &#x2714; | Файл для загрузки |
  | ИмяФайла | --filename | Строка | &#x2714; | Имя файла с расширением |
  | Заголовок | --title | Строка | &#x2714; | Имя файла в Slack |
  | Канал | --channel | Строка | &#x2716; | ID канала |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Slack

<br/>

:::tip
Параметры с типом данных Двоичные данные могут принимать также пути к файлам на диске и URL
:::
<br/>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен = "xoxb-6965308400114-696804637...";
    Канал = "C06UFNUTKUL";
    Файл  = "https://hut.openintegrations.dev/test_data/document.docx"; // URL, Двоичные данные или Путь к файлу

    ИмяФайла  = "megadoc.docx";
    Заголовок = "Новый файл";

    Результат = OPI_Slack.ЗагрузитьФайл(Токен, Файл, ИмяФайла, Заголовок);

    Результат = OPI_Slack.ЗагрузитьФайл(Токен, Файл, ИмяФайла, Заголовок, Канал);
```
