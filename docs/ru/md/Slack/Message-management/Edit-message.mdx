# Изменить сообщение
 Изменяет состав существующего сообщения



`Функция ИзменитьСообщение(Знач Токен, Знач Канал, Знач Отметка, Знач Текст = "", Знач МассивБлоков = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | Канал | --channel | Строка | &#x2714; | Идентификатор канала |
  | Отметка | --stamp | Строка | &#x2714; | Временная отметка сообщения |
  | Текст | --text | Строка | &#x2716; | Новый текст сообщения |
  | МассивБлоков | --blocks | Массив Из Структура | &#x2716; | JSON массива описаний блоков |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Slack

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен   = "xoxb-6965308400114-696804637...";
    Канал   = "C06UFNUTKUL";
    Отметка = "1747771991.004039";

    Текст = "Тестовое сообщение 2";

    Результат = OPI_Slack.ИзменитьСообщение(Токен, Канал, Отметка, Текст);
```
