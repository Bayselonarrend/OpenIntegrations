# Удалить сообщение
 Удаляет сообщение канала по timestamp



`Функция УдалитьСообщение(Знач Токен, Знач Канал, Знач Отметка, Знач ЭтоОтложенное = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | Канал | --channel | Строка | &#x2714; | Идентификатор канала |
  | Отметка | --stamp | Строка | &#x2714; | Временная отметка или ID сообщения |
  | ЭтоОтложенное | --issheduled | Булево | &#x2716; | Признак удаления отложенного сообщения |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Slack

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен   = "xoxb-6965308400114-696804637...";
    Канал   = "C06UFNUTKUL";
    Отметка = "1752610075.718239";

    Результат = OPI_Slack.УдалитьСообщение(Токен, Канал, Отметка);
```
