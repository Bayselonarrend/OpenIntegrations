# Получить список ответов на сообщение
 Получает массив сообщений, которые являются ответом на указанное



`Функция ПолучитьСписокОтветовНаСообщение(Знач Токен, Знач Канал, Знач Отметка, Знач Курсор = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен бота |
  | Канал | --channel | Строка | &#x2714; | Идентификатор канала |
  | Отметка | --stamp | Строка | &#x2714; | Временная отметка или ID сообщения |
  | Курсор | --cursor | Строка | &#x2716; | Указатель из предыдущего запроса, если строк результата > 100 |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Slack



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен   = "xoxb-6965308400114-696804637...";
    Канал   = "C06UFNUTKUL";
    Отметка = "1765392210.321779";

    Результат = OPI_Slack.ПолучитьСписокОтветовНаСообщение(Токен, Канал, Отметка);
```
