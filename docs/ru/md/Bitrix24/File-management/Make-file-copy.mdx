# Скопировать файл
 Копирует файл из одного расположения в другое



`Функция СкопироватьФайл(Знач URL, Знач IDФайла, Знач IDКаталога, Знач Токен = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL внешнего вебхука или адрес Bitrix24 при использовании токена |
  | IDФайла | --fileid | Строка, Число | &#x2714; | ID файла для копирования |
  | IDКаталога | --folderid | Строка, Число | &#x2714; | ID каталога размещения копии |
  | Токен | --token | Строка | &#x2716; | Токен авторизации, если используется не вебхук |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Bitrxi24

:::tip
Метод в документации API: [disk.file.copyto](https://dev.1c-bitrix.ru/rest_help/disk/file/disk_file_copyto.php)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    IDКаталога = "46676";
    IDФайла    = "46672";

    URL = "https://b24-ar17wx.bitrix24.by/rest/1/av5...";

    Результат = OPI_Bitrix24.СкопироватьФайл(URL, IDФайла, IDКаталога);

    IDКаталога = "46678";

    URL   = "b24-ar17wx.bitrix24.by";
    Токен = "a5b00469006e9f06006b12e400000001000...";

    Результат = OPI_Bitrix24.СкопироватьФайл(URL, IDФайла, IDКаталога, Токен);
```
