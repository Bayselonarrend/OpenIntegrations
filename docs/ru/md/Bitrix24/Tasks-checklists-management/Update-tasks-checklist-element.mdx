# Изменить элемент чеклиста задачи
 Изменяет текст элемента чеклиста задачи



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ИзменитьЭлементЧеклистаЗадачи(Знач URL, Знач IDЗадачи, Знач IDЭлемента, Знач Текст, Знач Токен = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL внешнего вебхука или адрес Bitrix24 при использовании токена |
  | IDЗадачи | --task | Число, Строка | &#x2714; | ID задачи |
  | IDЭлемента | --element | Число, Строка | &#x2714; | ID элемента чеклиста |
  | Текст | --text | Строка | &#x2714; | Текст (заголовок) пункта чек листа |
  | Токен | --token | Строка | &#x2716; | Токен авторизации, если используется не вебхук |

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL        = "https://b24-ar17wx.bitrix24.by/rest/1/av5...";
    IDЗадачи   = "8718";
    IDЭлемента = "6160";

    Текст = "Новый текст пункта чек-листа";

    Результат = OPI_Bitrix24.ИзменитьЭлементЧеклистаЗадачи(URL, IDЗадачи, IDЭлемента, Текст);

    URL        = "b24-ar17wx.bitrix24.by";
    Токен      = "a5b00469006e9f06006b12e400000001000...";
    IDЗадачи   = "8720";
    IDЭлемента = "6164";

    Результат = OPI_Bitrix24.ИзменитьЭлементЧеклистаЗадачи(URL, IDЗадачи, IDЭлемента, Текст, Токен);
```
