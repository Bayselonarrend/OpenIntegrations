# Получить настройки учета времени
 Получает настройки учета времени пользователя



`Функция ПолучитьНастройкиУчетаВремени(Знач URL, Знач IDПользователя = "", Знач Токен = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL внешнего вебхука или адрес Bitrix24 при использовании токена |
  | IDПользователя | --userid | Строка, Число | &#x2716; | ID пользователя для учета. Текущий по умолчанию |
  | Токен | --token | Строка | &#x2716; | Токен авторизации, если используется не вебхук |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Bitrxi24

:::tip
Метод в документации API: [timeman.settings](https://dev.1c-bitrix.ru/rest_help/timeman/base/timeman_settings.php)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL = "https://b24-ar17wx.bitrix24.by/rest/1/h0m...";

    Результат = OPI_Bitrix24.ПолучитьНастройкиУчетаВремени(URL);

    URL            = "b24-ar17wx.bitrix24.by";
    Токен          = "bafe5c69006e9f06006b12e400000001000...";
    IDПользователя = 1;

    Результат = OPI_Bitrix24.ПолучитьНастройкиУчетаВремени(URL, IDПользователя, Токен);
```
