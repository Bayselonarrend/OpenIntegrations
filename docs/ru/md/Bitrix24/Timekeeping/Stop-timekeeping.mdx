# Завершить учет времени
 Завершает учет времени пользователя



`Функция ЗавершитьУчетВремени(Знач URL, Знач IDПользователя = "", Знач Время = "", Знач Отчет = "", Знач Токен = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL внешнего вебхука или адрес Bitrix24 при использовании токена |
  | IDПользователя | --userid | Строка, Число | &#x2716; | ID пользователя для учета. Текущий по умолчанию |
  | Время | --time | Дата | &#x2716; | Время заверш. учета. Дата должна совпадать с текущей. Текущее время по ум. |
  | Отчет | --report | Строка | &#x2716; | Причина изменения. Обязателен при указании Время и отключенном свободном графике |
  | Токен | --token | Строка | &#x2716; | Токен авторизации, если используется не вебхук |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Bitrxi24

:::tip
Метод в документации API: [timeman.close](https://dev.1c-bitrix.ru/rest_help/timeman/base/timeman_close.php)
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL = "https://b24-ar17wx.bitrix24.by/rest/1/h0m...";

    Результат = OPI_Bitrix24.ЗавершитьУчетВремени(URL);

    URL            = "b24-ar17wx.bitrix24.by";
    Токен          = "6ad03969006e9f06006b12e400000001000...";
    IDПользователя = 1;
    Время          = OPI_Инструменты.ПолучитьТекущуюДату();
    Отчет          = "Отгул";

    Результат = OPI_Bitrix24.ЗавершитьУчетВремени(URL, IDПользователя, Время, Отчет, Токен);
```
