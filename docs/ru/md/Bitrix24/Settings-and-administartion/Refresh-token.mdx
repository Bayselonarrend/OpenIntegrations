# Обновить токен
 Обновляет токен по Refresh токену



`Функция ОбновитьТокен(Знач ClientID, Знач ClientSecret, Знач Refresh) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ClientID | --clientid | Строка | &#x2714; | Client ID из настроек приложения |
  | ClientSecret | --clientsecret | Строка | &#x2714; | Client secret из настроек приложения |
  | Refresh | --refresh | Строка | &#x2714; | Refresh токен |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Bitrxi24

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ClientID     = "local.6667fc928a50a9.70414732";
    ClientSecret = "ZeKyeYIgy2NsHZqsIHY6GfG1V...";
    Refresh      = "68013b67006e9f06006b12e400000001000...";

    Результат = OPI_Bitrix24.ОбновитьТокен(ClientID, ClientSecret, Refresh);
```
