# Изменить комментарий
 Изменяет текст существующего комментария



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ИзменитьКомментарий(Знач Токен, Знач База, Знач Таблица, Знач Запись, Знач Комментарий, Знач Текст) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | База | --base | Строка | &#x2714; | Идентификатор базы данных |
  | Таблица | --table | Строка | &#x2714; | Идентификатор таблицы |
  | Запись | --record | Строка | &#x2714; | Идентификатор записи в таблице |
  | Комментарий | --comment | Строка | &#x2714; | Идентификатор комментария |
  | Текст | --text | Строка | &#x2714; | Новый текст комментария |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - сериализованный JSON ответа от Airtable
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его объект (только для 1С и OneScript) |


  </TabItem>
</Tabs>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен       = "patNn4BXW66Yx3pdj.5b93c53cab554a8387de02d...";
    База        = "appJiRrCJhRhnaVAo";
    Таблица     = "tblFGmvh2eldM8MuJ";
    Запись      = "recuJsvpt9DG2SuEk";
    Комментарий = "comPMHUtwEsa66el1";

    Текст     = "Тестовый комментарий (изм.)";
    Результат = OPI_Airtable.ИзменитьКомментарий(Токен, База, Таблица, Запись, Комментарий, Текст);
```
