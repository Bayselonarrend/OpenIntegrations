# Удалить записи
 Удаляет одну или массив записей по идентификаторам



`Функция УдалитьЗаписи(Знач Токен, Знач База, Знач Таблица, Знач Записи) Экспорт`

<Tabs>
  <TabItem value="params" label="Параметры" default>

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | База | --base | Строка | &#x2714; | Идентификатор базы данных |
  | Таблица | --table | Строка | &#x2714; | Идентификатор таблицы |
  | Записи | --records | Строка, Массив из Строка | &#x2714; | Идентификатор или массив идентификаторов записей |

  
  </TabItem>
  <TabItem value="extended" label="Расширенный вызов">

  | Параметр | Описание |
  |---|---|
  | proxy | ИнтернетПрокси или структура полей Протокол, Сервер, Порт, Пользователь, Пароль, ИспользоватьАутентификациюОС |
  | timeout | Таймаут выполнения запроса |
  | adv_response | Оформляет ответ в виде полной структуры с полями code, body и headers |


  </TabItem>
</Tabs>
  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Airtable



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен   = "patNn4BXW66Yx3pdj.5b93c53cab554a8387de02d...";
    База    = "appJiRrCJhRhnaVAo";
    Таблица = "tblFGmvh2eldM8MuJ";
    Запись  = "recuJsvpt9DG2SuEk";

    Результат = OPI_Airtable.УдалитьЗаписи(Токен, База, Таблица, Запись);
```
    

 <Tabs>
