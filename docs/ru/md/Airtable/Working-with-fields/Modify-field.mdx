# Изменить поле
 Изменяет имя и|или описание существующего поля таблицы



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ИзменитьПоле(Знач Токен, Знач База, Знач Таблица, Знач Поле, Знач Наименование = "", Знач Описание = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | База | --base | Строка | &#x2714; | Идентификатор базы База |
  | Таблица | --table | Строка | &#x2714; | Идентификатор таблицы |
  | Поле | --field | Строка | &#x2714; | Идентификатор поля |
  | Наименование | --title | Строка | &#x2716; | Новое наименование |
  | Описание | --description | Строка | &#x2716; | Новое описание |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - сериализованный JSON ответа от Airtable
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его объект (только для 1С и OneScript) |


  </TabItem>
</Tabs>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен   = "patNn4BXW66Yx3pdj.5b93c53cab554a8387de02d...";
    База    = "appJiRrCJhRhnaVAo";
    Таблица = "tblFGmvh2eldM8MuJ";
    Поле    = "fldm9baY6FkgCPaCi";

    Имя      = Строка(Новый УникальныйИдентификатор) + "(изм.)";
    Описание = "Новое описание";

    Результат = OPI_Airtable.ИзменитьПоле(Токен, База, Таблица, Поле, Имя, Описание);
```
