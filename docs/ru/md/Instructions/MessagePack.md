# MessagePack

Этот раздел посвящен библиотеке для работы с форматом MessagePack в 1С:Предприятие, OneScript и CLI. На данной странице описаны все действия, необходимые для полноценного начала работы

## Начало работы

MessagePack — это эффективный формат бинарной сериализации данных. Он позволяет обмениваться данными между различными языками программирования, как JSON, но при этом занимает меньше места и работает быстрее.

Библиотека предоставляет две основные функции для работы с форматом:

### Сериализация данных

Для преобразования данных в формат MessagePack используйте функцию `СериализоватьДанные`:

```bsl
Данные = Новый Структура;
Данные.Вставить("имя", "Иван");
Данные.Вставить("возраст", 30);

ДвоичныеДанные = OPI_MessagePack.СериализоватьДанные(Данные);
```

### Десериализация данных

Для восстановления данных из формата MessagePack используйте функцию `ДесериализоватьДанные`:

```bsl
ВосстановленныеДанные = OPI_MessagePack.ДесериализоватьДанные(ДвоичныеДанные);
```

## Поддерживаемые типы данных

Библиотека поддерживает сериализацию и десериализацию следующих типов данных:

- **Неопределено** — преобразуется в nil формата MessagePack
- **Булево** — логические значения Истина/Ложь
- **Строка** — текстовые данные в кодировке UTF-8
- **ДвоичныеДанные** — бинарные данные
- **Массив** — упорядоченные коллекции элементов
- **Структура/Соответствие** — коллекции пар ключ-значение (ключи преобразуются в строки)
- **Число** — целые и вещественные числа (автоматически определяется оптимальный формат хранения)

:::note
При сериализации ключей структур и соответствий, они автоматически преобразуются в строковый формат
:::
