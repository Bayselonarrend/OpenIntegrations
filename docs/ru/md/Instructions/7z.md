# 7z

import LibraryIntro from '@site/src/components/LibraryIntro';

<LibraryIntro module="OPI_7z" cli="7z" use="oint/formats/7z" lang="ru"/>

Этот раздел посвящен библиотеке для работы с 7z архивами в 1С:Предприятие, OneScript и CLI. На данной странице описаны все действия, необходимые для полноценного начала работы

## Начало работы

Библиотека предоставляет возможности для работы с архивами формата 7z: создание архивов с различными настройками сжатия, распаковка существующих архивов, работа с парольной защитой и шифрованием.

### Основные возможности

**Архивация каталогов**

Функция `АрхивироватьКаталог` позволяет запаковать файлы указанного каталога в архив 7z. Можно указать путь для сохранения архива или получить его как двоичные данные.

```bsl
Каталог = "C:\МойКаталог";
ПутьАрхива = "C:\Архивы\архив.7z";

Результат = OPI_7z.АрхивироватьКаталог(Каталог, ПутьАрхива);
```

**Распаковка архивов**

Функция `РазархивироватьКаталог` позволяет распаковать 7z архив в указанный каталог. Поддерживается работа с защищенными паролем архивами.

```bsl
ПутьАрхива = "C:\Архивы\архив.7z";
КаталогНазначения = "C:\Распаковка";
Пароль = "МойПароль123";

Результат = OPI_7z.РазархивироватьКаталог(ПутьАрхива, КаталогНазначения, Пароль);
```

### Настройки архивации

Функция `ПолучитьСтруктуруНастроекАрхивации` возвращает структуру с дополнительными настройками для создания архива:

- **password** — пароль для защиты архива
- **method** — метод сжатия: `Lzma2` (по умолчанию), `Lzma`, `Bzip2`, `Ppmd`, `Copy`
- **level** — уровень сжатия от 0 до 9 (по умолчанию 6)
- **solid** — использование solid-архивации (сжатие одним потоком): `Истина`/`Ложь` (по умолчанию)
- **encrypt_header** — шифрование заголовка архива: `Истина` (по умолчанию при наличии пароля)/`Ложь`
- **threads** — количество потоков для LZMA2 (по умолчанию 1)
- **chunk_size** — размер блока данных при использовании нескольких потоков
- **dictionary_size** — размер словаря для LZMA2
- **ppmd_order** — порядок модели для метода Ppmd
- **ppmd_memory** — объем памяти в байтах для метода Ppmd
- **filters** — массив фильтров предобработки: `X86`, `PPC`, `IA64`, `ARM`, `ARM64`, `ARM_THUMB`, `SPARC`, `RISCV`, `DELTA`
- **delta_distance** — расстояние для фильтра DELTA (по умолчанию 1)

**Пример использования настроек:**

```bsl
Настройки = OPI_7z.ПолучитьСтруктуруНастроекАрхивации(Истина);
Настройки["password"] = "СекретныйПароль";
Настройки["method"] = "Lzma2";
Настройки["level"] = 9;
Настройки["encrypt_header"] = Истина;
Настройки["threads"] = 4;

Результат = OPI_7z.АрхивироватьКаталог(Каталог, ПутьАрхива, Настройки);
```

:::tip
Для максимального сжатия используйте метод `Lzma2` с уровнем сжатия `9`. Для более быстрого создания архива снизьте уровень сжатия или используйте метод `Copy`
:::

:::important
При использовании шифрования заголовка (`encrypt_header = Истина`) имена файлов в архиве также будут зашифрованы и не будут видны без ввода пароля
:::
