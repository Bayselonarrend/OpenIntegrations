# TCP

import LibraryIntro from '@site/src/components/LibraryIntro';

<LibraryIntro module="OPI_TCP" cli="tcp" use="oint/api/tcp" lang="ru"/>

Этот раздел посвящен библиотеке для работы с TCP соединениями в 1С:Предприятие, OneScript и CLI. На данной странице описаны все действия, необходимые для полноценного начала работы

## Начало работы

Библиотека предоставляет возможность работы с TCP протоколом в двух режимах: как **клиент** (подключение к удаленному серверу) и как **сервер** (прием входящих соединений). Библиотека поддерживает работу с двоичными данными и текстом, использование TLS шифрования и прокси-серверов.

### Работа в режиме клиента

Для работы в режиме клиента необходимо:

1. Подготовить адрес подключения в формате `хост:порт` (например, `localhost:8080`)
2. При необходимости настроить TLS при помощи функции `ПолучитьНастройкиTls`
3. При необходимости настроить прокси при помощи функции `ПолучитьНастройкиПрокси`
4. Открыть соединение при помощи функции `ОткрытьСоединение`
5. Использовать функции отправки и получения данных
6. Закрыть соединение при помощи функции `ЗакрытьСоединение`

```bsl
Адрес = "example.com:80";

// Открытие соединения
Соединение = OPI_TCP.ОткрытьСоединение(Адрес);

// Отправка данных
Данные = "GET / HTTP/1.1" + Символы.ПС + "Host: example.com" + Символы.ПС + Символы.ПС;
OPI_TCP.ОтправитьСтроку(Соединение, Данные);

// Получение ответа
Ответ = OPI_TCP.ПрочитатьСтроку(Соединение);

// Закрытие соединения
OPI_TCP.ЗакрытьСоединение(Соединение);
```

### Работа с TLS

Для работы с защищенными соединениями используйте настройки TLS:

```bsl
// Настройка TLS с отключением проверки сертификатов
НастройкиTLS = OPI_TCP.ПолучитьНастройкиTls(Истина);

// Настройка TLS с указанием пути к сертификату
НастройкиTLS = OPI_TCP.ПолучитьНастройкиTls(Ложь, "/path/to/certificate.pem");

Адрес = "example.com:443";
Соединение = OPI_TCP.ОткрытьСоединение(Адрес, НастройкиTLS);
```

### Работа через прокси

Библиотека поддерживает работу через прокси-серверы:

```bsl
// Настройка прокси
НастройкиПрокси = OPI_TCP.ПолучитьНастройкиПрокси("proxy.example.com", 1080, "socks5", "login", "password");

Адрес = "example.com:80";
Соединение = OPI_TCP.ОткрытьСоединение(Адрес, , НастройкиПрокси);
```

Поддерживаемые типы прокси: `socks5`, `socks4`, `http`

### Одиночный запрос

Для выполнения простого запроса без явного управления соединением используйте функцию `ОбработатьЗапрос`:

```bsl
Адрес = "example.com:80";
Данные = "GET / HTTP/1.1" + Символы.ПС + "Host: example.com" + Символы.ПС + Символы.ПС;

Ответ = OPI_TCP.ОбработатьЗапрос(Адрес, Данные);
```

### Работа в режиме сервера

Для создания TCP сервера необходимо:

1. При необходимости настроить логирование при помощи функции `ПолучитьНастройкиЛогирования`
2. Запустить сервер при помощи функции `ЗапуститьСервер`
3. Получать входящие данные при помощи функций `ПолучитьДанныеОчередногоСоединения` или `ПолучитьДанныеСоединения`
4. Отправлять ответы при помощи функции `ОтправитьДанные`
5. Остановить сервер при помощи функции `ОстановитьСервер`

```bsl
Порт = 8080;
РазмерПула = 100;

// Настройка логирования (опционально)
НастройкиЛога = OPI_TCP.ПолучитьНастройкиЛогирования(Истина, 300, "/path/to/log.txt");

// Запуск сервера
Сервер = OPI_TCP.ЗапуститьСервер(Порт, РазмерПула, НастройкиЛога);

// Получение данных из очереди
Сообщение = OPI_TCP.ПолучитьДанныеОчередногоСоединения(Сервер, 5000);

Если Сообщение["result"] Тогда
    IDСоединения = Сообщение["connection_id"];
    ВходящиеДанные = Сообщение["message"];
    
    // Отправка ответа
    Ответ = "Данные получены";
    OPI_TCP.ОтправитьДанные(Сервер, IDСоединения, Ответ);
КонецЕсли;

// Остановка сервера
OPI_TCP.ОстановитьСервер(Сервер);
```

### Работа с данными

Библиотека поддерживает работу как с двоичными данными, так и со строками:

```bsl
// Чтение и отправка строк
Строка = OPI_TCP.ПрочитатьСтроку(Соединение, "UTF-8");
OPI_TCP.ОтправитьСтроку(Соединение, "Привет, мир!", "UTF-8");

// Чтение и отправка двоичных данных
ДвоичныеДанные = OPI_TCP.ПрочитатьДвоичныеДанные(Соединение, 1024);
OPI_TCP.ОтправитьДвоичныеДанные(Соединение, ДвоичныеДанные);
```

### Использование маркеров конца сообщения

При чтении данных можно указать маркер, по достижении которого чтение завершится:

```bsl
// Чтение до символа перевода строки
Данные = OPI_TCP.ПрочитатьСтроку(Соединение, "UTF-8", Символы.ПС);

// Чтение до пользовательского маркера
Маркер = ПолучитьДвоичныеДанныеИзСтроки("END");
Данные = OPI_TCP.ПрочитатьДвоичныеДанные(Соединение, , Маркер);
```

:::important
При работе с бесконечным потоком входящих данных обязательно указывайте параметр `МаксимальныйРазмер` или используйте маркер конца сообщения, так как бесконечное получение данных может привести к зависанию
:::
