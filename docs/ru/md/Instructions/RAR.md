# RAR

import LibraryIntro from '@site/src/components/LibraryIntro';

<LibraryIntro module="OPI_RAR" cli="rar" use="oint/formats/rar" lang="ru"/>

Этот раздел посвящен библиотеке для работы с RAR архивами в 1С:Предприятие, OneScript и CLI. На данной странице описаны все действия, необходимые для полноценного начала работы

## Начало работы

Библиотека предоставляет возможности для работы с архивами формата RAR: распаковка существующих архивов, работа с парольной защитой, получение списка файлов и метаданных архива.

:::important
Данная библиотека предназначена только для **распаковки** RAR архивов. Создание RAR архивов не поддерживается
:::

### Основные возможности

**Распаковка архивов**

Функция `РазархивироватьКаталог` позволяет распаковать RAR архив в указанный каталог. Поддерживается работа с защищенными паролем архивами.

```bsl
ПутьАрхива = "C:\Архивы\архив.rar";
КаталогНазначения = "C:\Распаковка";
Пароль = "МойПароль123";

Результат = OPI_RAR.РазархивироватьКаталог(ПутьАрхива, КаталогНазначения, Пароль);
```

**Выборочная распаковка файлов**

Функция `РазархивироватьФайлы` позволяет распаковать только определенные файлы из архива по списку их путей внутри архива.

```bsl
ПутьАрхива = "C:\Архивы\архив.rar";
КаталогНазначения = "C:\Распаковка";

МассивПутей = Новый Массив;
МассивПутей.Добавить("Документы/отчет.docx");
МассивПутей.Добавить("Изображения/фото.jpg");

Результат = OPI_RAR.РазархивироватьФайлы(ПутьАрхива, МассивПутей, КаталогНазначения);
```

### Получение информации об архиве

**Список файлов**

Функция `ПолучитьСписокФайлов` возвращает иерархический список файлов и каталогов, содержащихся в архиве.

```bsl
ПутьАрхива = "C:\Архивы\архив.rar";
Пароль = "МойПароль123";

СписокФайлов = OPI_RAR.ПолучитьСписокФайлов(ПутьАрхива, Пароль);
```

**Метаданные архива**

Функция `ПолучитьМетаданные` возвращает метаданные архива и плоский список всех файлов с их характеристиками.

```bsl
ПутьАрхива = "C:\Архивы\архив.rar";

Метаданные = OPI_RAR.ПолучитьМетаданные(ПутьАрхива);
```

### Настройки архивации

Функция `ПолучитьСтруктуруНастроекАрхивации` возвращает структуру с дополнительными настройками для работы с архивом:

- **password** — пароль для работы с защищенным архивом

**Пример использования настроек:**

```bsl
Настройки = OPI_RAR.ПолучитьСтруктуруНастроекАрхивации(Истина);
Настройки["password"] = "СекретныйПароль";

Результат = OPI_RAR.РазархивироватьКаталог(ПутьАрхива, КаталогНазначения, Настройки["password"]);
```

### Работа с двоичными данными

Библиотека поддерживает работу не только с файлами на диске, но и с двоичными данными архива в памяти:

```bsl
// Чтение архива в память
ДвоичныеДанные = Новый ДвоичныеДанные("C:\Архивы\архив.rar");

// Распаковка из памяти
Результат = OPI_RAR.РазархивироватьКаталог(ДвоичныеДанные, "C:\Распаковка");

// Получение списка файлов из памяти
СписокФайлов = OPI_RAR.ПолучитьСписокФайлов(ДвоичныеДанные);
```

:::tip
При работе с большими архивами рекомендуется использовать пути к файлам на диске вместо двоичных данных для оптимизации использования памяти
:::

:::note
При распаковке без указания каталога назначения результат будет возвращен как соответствие с именами файлов и их двоичными данными
:::
