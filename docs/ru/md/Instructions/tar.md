# tar

import LibraryIntro from '@site/src/components/LibraryIntro';

<LibraryIntro module="OPI_Tar" cli="tar" use="oint/formats/tar" lang="ru"/>

Этот раздел посвящен библиотеке для работы с tar архивами в 1С:Предприятие, OneScript и CLI. На данной странице описаны все действия, необходимые для полноценного начала работы

## Начало работы

Библиотека предоставляет возможности для работы с архивами формата tar: создание архивов, распаковка, извлечение отдельных файлов, модификация существующих архивов и работа с gzip сжатием.

### Основные возможности

**Архивация каталогов**

Функция `АрхивироватьКаталог` позволяет запаковать файлы указанного каталога в архив tar. Можно указать путь для сохранения архива или получить его как двоичные данные.

```bsl
Каталог = "C:\МойКаталог";
ПутьАрхива = "C:\Архивы\архив.tar";

Результат = OPI_Tar.АрхивироватьКаталог(Каталог, ПутьАрхива);
```

**Распаковка архивов**

Функция `РазархивироватьКаталог` позволяет распаковать tar архив в указанный каталог.

```bsl
ПутьАрхива = "C:\Архивы\архив.tar";
КаталогНазначения = "C:\Распаковка";

Результат = OPI_Tar.РазархивироватьКаталог(ПутьАрхива, КаталогНазначения);
```

**Распаковка отдельных файлов**

Функция `РазархивироватьФайлы` позволяет извлечь только выбранные файлы из архива по списку их путей внутри архива.

```bsl
ПутьАрхива = "C:\Архивы\архив.tar";
Пути = Новый Массив;
Пути.Добавить("docs/readme.txt");
Пути.Добавить("config/settings.json");

КаталогНазначения = "C:\Распаковка";

Результат = OPI_Tar.РазархивироватьФайлы(ПутьАрхива, Пути, КаталогНазначения);
```

**Модификация архивов**

Функция `ИзменитьАрхив` позволяет добавлять новые файлы, заменять или удалять существующие файлы в tar архиве без полной перепаковки.

```bsl
ПутьАрхива = "C:\Архивы\архив.tar";

// Файлы для добавления или замены
ДобавляемыеФайлы = Новый Соответствие;
ДобавляемыеФайлы.Вставить("docs/new.txt", "C:\Файлы\новый.txt");

// Файлы для удаления
УдаляемыеПути = Новый Массив;
УдаляемыеПути.Добавить("old/obsolete.txt");

Результат = OPI_Tar.ИзменитьАрхив(ПутьАрхива, ДобавляемыеФайлы, УдаляемыеПути);
```

### Настройки архивации

Функция `ПолучитьСтруктуруНастроекАрхивации` возвращает структуру с дополнительными настройками для создания архива:

- **gzip** — применять gzip сжатие к архиву: `Истина`/`Ложь` (по умолчанию `Ложь`)
- **gzip_level** — уровень gzip сжатия от 0 до 9 (по умолчанию 6, где 0 — без сжатия, 9 — максимальное сжатие)

**Пример использования настроек:**

```bsl
Настройки = OPI_Tar.ПолучитьСтруктуруНастроекАрхивации(Истина);
Настройки["gzip"] = Истина;
Настройки["gzip_level"] = 9;

Каталог = "C:\МойКаталог";
ПутьАрхива = "C:\Архивы\архив.tar.gz";

Результат = OPI_Tar.АрхивироватьКаталог(Каталог, ПутьАрхива, Настройки);
```

:::tip
Использование gzip сжатия позволяет значительно уменьшить размер архива. Для максимального сжатия используйте `gzip_level = 9`, для более быстрого создания архива — меньшие значения (например, 1-3)
:::

:::important
При изменении существующего архива на диске параметры gzip должны задаваться в структуре Настройки. Архив на диске изменяется непосредственно на месте
:::

### Получение информации об архиве

**Получение списка файлов**

Функция `ПолучитьСписокФайлов` возвращает иерархический список всех файлов и каталогов в архиве.

```bsl
ПутьАрхива = "C:\Архивы\архив.tar";

СписокФайлов = OPI_Tar.ПолучитьСписокФайлов(ПутьАрхива);
```

**Получение метаданных**

Функция `ПолучитьМетаданные` возвращает метаданные архива и плоский список всех файлов с подробной информацией.

```bsl
ПутьАрхива = "C:\Архивы\архив.tar";

Метаданные = OPI_Tar.ПолучитьМетаданные(ПутьАрхива);
```
