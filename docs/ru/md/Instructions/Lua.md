# Lua

import LibraryIntro from '@site/src/components/LibraryIntro';

<LibraryIntro module="OPI_Lua" cli="lua" use="oint/api/lua" lang="ru"/>

Этот раздел посвящен библиотеке для работы с Lua в 1С:Предприятие и OneScript. На данной странице описаны все действия, необходимые для полноценного начала работы

## Начало работы

Библиотека предоставляет возможность выполнения Lua скриптов непосредственно из 1С:Предприятие и OneScript при помощи внешних компонент.

### Поддерживаемые версии

Библиотека поддерживает две версии Lua:

- **Lua54** — стабильная версия Lua 5.4
- **LuaJIT** — JIT-компилятор Lua с повышенной производительностью

### Создание виртуальной машины

Для выполнения Lua кода необходимо создать виртуальную машину (VM):

```bsl
// Создание VM Lua 5.4
VM = OPI_Lua.СоздатьVM("Lua54");

// Создание VM LuaJIT
VM = OPI_Lua.СоздатьVM("LuaJIT");

// Создание VM с настройками логирования
НастройкиЛогирования = Новый Структура;
НастройкиЛогирования.Вставить("level", "debug");
НастройкиЛогирования.Вставить("path", "C:\logs\lua.log");

VM = OPI_Lua.СоздатьVM("Lua54", НастройкиЛогирования);
```

### Выполнение кода

Библиотека поддерживает два способа выполнения Lua кода:

#### Выполнение из строки

```bsl
КодLua = "return 2 + 2";
Результат = OPI_Lua.ВыполнитьКодИзСтроки(VM, КодLua);
// Результат = 4
```

#### Выполнение из файла

```bsl
ПутьКФайлу = "C:\scripts\script.lua";
Результат = OPI_Lua.ВыполнитьКодИзФайла(VM, ПутьКФайлу);
```

:::important
При возникновении ошибки выполнения Lua кода будет вызвано исключение. Рекомендуется использовать конструкцию `Попытка...Исключение` для обработки возможных ошибок
:::

### Упрощенное использование

Вместо создания объекта VM можно передавать версию Lua напрямую в функции выполнения:

```bsl
Результат = OPI_Lua.ВыполнитьКодИзСтроки("Lua54", "return 'Hello from Lua'");
Результат = OPI_Lua.ВыполнитьКодИзФайла("LuaJIT", "C:\scripts\script.lua");
```

В этом случае VM будет создана автоматически.
