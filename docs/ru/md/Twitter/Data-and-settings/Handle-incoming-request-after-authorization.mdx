# Обработка входящего запроса после авторизации 
 Метод для вставки в http-сервис, адрес которого указывается в redirect_uri



`Функция ОбработкаВходящегоЗапросаПослеАвторизации(Запрос) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Запрос | - | HTTPСервисЗапрос | &#x2714; | Запрос, приходящий на http |

  
  Возвращаемое значение:   HTTPОтвет, Произвольный, ДвоичныеДанные - Результат чтения JSON ответа сервера

<br/>

:::tip
Вызывает метод получения токена, так как для получения токена из кода, приходящего на redirect_uri после авторизации через браузер есть всего 30 секунд
:::

:::caution
**NOCLI:** данный метод недоступен для использования в CLI версии
:::
