# 
Метод для вставки в http-сервис, адрес которого указывается в redirect_uri Вызывает метод получения токена, так как для получения токена из кода, приходящего на redirect_uri после авторизации через браузер есть всего 30 секунд



`Функция ОбработкаВходящегоЗапросаПослеАвторизации(Запрос) Экспорт // Service`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Запрос | --сервис | HTTPСервисЗапрос | &#x2714; | Запрос, приходящий на http |

  
  Возвращаемое значение:   HTTPОтвет, Произвольный, ДвоичныеДанные - Результат чтения JSON ответа сервера
