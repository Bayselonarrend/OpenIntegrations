# Получить токен
 Получает токен по коду, полученному при авторизации по ссылке из ПолучитьСсылкуАвторизации



`Функция ПолучитьТокен(Знач Код, Знач Параметры = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Код | --code | Строка | &#x2714; | Код, полученный из авторизации См.ПолучитьСсылкуАвторизации |
  | Параметры | --auth | Структура из Строка | &#x2716; | Данные авторизации. См.ПолучитьСтандартныеПараметры |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Twitter



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Параметры = ПолучитьПараметрыАвторизацииТвиттер();
    Код       = "123456";

    Результат = OPI_Twitter.ПолучитьТокен(Код, Параметры);
```
