# Создать произвольный твит
 Создает твит с произвольным содержимым



`Функция СоздатьПроизвольныйТвит(Знач Текст = "", Знач МассивМедиа = "", Знач МассивВариантовОпроса = "", Знач ДлительностьОпроса = "", Знач Параметры = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Текст | --text | Строка | &#x2716; | Текст твита |
  | МассивМедиа | --media | Массив из Строка,ДвоичныеДанные | &#x2716; | Массив двоичных данных или путей к файлам |
  | МассивВариантовОпроса | --options | Массив из Строка | &#x2716; | Массив вариантов опроса, если необходимо |
  | ДлительностьОпроса | --dur | Строка,Число | &#x2716; | Длительность опроса, если необходимо (опрос без длительности не создается) |
  | Параметры | --auth | Структура из Строка | &#x2716; | Данные авторизации. См.ПолучитьСтандартныеПараметры |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Twitter

<br/>

:::tip
Параметры с типом данных Двоичные данные могут принимать также пути к файлам на диске и URL
:::
<br/>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Параметры = ПолучитьПараметрыАвторизацииТвиттер();
    Текст     = "Тестовый твитт" + Строка(Новый УникальныйИдентификатор);

    Картинка1 = "https://hut.openintegrations.dev/test_data/picture.jpg";  // URL, Двоичные данные или Путь к файлу
    Картинка2 = "https://hut.openintegrations.dev/test_data/picture2.jpg"; // URL, Двоичные данные или Путь к файлу

    МассивКартинок = Новый Массив;
    МассивКартинок.Добавить(Картинка1);
    МассивКартинок.Добавить(Картинка2);

    МассивМедиа = OPI_Twitter.ЗагрузитьМассивВложений(МассивКартинок, "tweet_image", Параметры);
    Результат   = OPI_Twitter.СоздатьПроизвольныйТвит(Текст, МассивМедиа, , , Параметры);
```
