# Загрузить массив вложений
 Загружает файлы на сервер и возвращает их ID



`Функция ЗагрузитьМассивВложений(Знач МассивФайлов, Знач ТипВложений, Знач Параметры = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | МассивФайлов | --files | Массив из Строка, ДвоичныеДанные | &#x2714; | Массив файлов для загрузки |
  | ТипВложений | --type | Строка | &#x2714; | Тип вложений: tweet_video, tweet_image, tweet_gif |
  | Параметры | --auth | Структура из Строка | &#x2716; | Данные авторизации. См.ПолучитьСтандартныеПараметры |

  
  Возвращаемое значение:   Массив Из Строка -  Массив ID медиа

<br/>

:::tip
Параметры с типом данных Двоичные данные могут принимать также пути к файлам на диске и URL
:::
<br/>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Параметры = ПолучитьПараметрыАвторизацииТвиттер();

    Картинка1 = "https://hut.openintegrations.dev/test_data/picture.jpg";  // URL, Двоичные данные или Путь к файлу
    Картинка2 = "https://hut.openintegrations.dev/test_data/picture2.jpg"; // URL, Двоичные данные или Путь к файлу

    МассивКартинок = Новый Массив;
    МассивКартинок.Добавить(Картинка1);
    МассивКартинок.Добавить(Картинка2);

    Результат = OPI_Twitter.ЗагрузитьМассивВложений(МассивКартинок, "tweet_image", Параметры);
```
