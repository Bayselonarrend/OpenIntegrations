# Получить информацию о таблице
 Получает информацию о таблице



`Функция ПолучитьИнформациюОТаблице(Знач Таблица, Знач Соединение = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Таблица | --table | Строка | &#x2714; | Имя таблицы |
  | Соединение | --db | Строка, Произвольный | &#x2716; | Существующее соединение или путь к базе |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - Результат выполнения запроса

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    База    = "C:\Users\bayse\AppData\Local\Temp\v8_D0B3_24.sqlite";
    Таблица = "test";

    Результат = OPI_SQLite.ПолучитьИнформациюОТаблице(Таблица, База);
```
