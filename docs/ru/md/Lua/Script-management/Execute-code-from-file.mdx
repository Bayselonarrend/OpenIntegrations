# Выполнить код из файла
 Выполняет Lua скрипт на диске



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ВыполнитьКодИзФайла(Знач Lua, Знач Путь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Lua | --lua | Произвольный | &#x2714; | Компонента Lua или версия Lua для запуска |
  | Путь | --path | Строка | &#x2714; | Путь к файлу скрипта |

:::

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ФайлСкрипта = ПолучитьИмяВременногоФайла("lua");
    ПолучитьДвоичныеДанныеИзСтроки("return 7").Записать(ФайлСкрипта);

    Результат = OPI_Lua.ВыполнитьКодИзФайла("Lua54", ФайлСкрипта);

    OPI_Инструменты.УдалитьФайлВПопытке(ФайлСкрипта, "Не удалось удалить временный файл после теста!");
```
