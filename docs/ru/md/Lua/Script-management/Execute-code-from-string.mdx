# Выполнить код из строки
 Выполняет Lua скрипт из переданной строки



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ВыполнитьКодИзСтроки(Знач Lua, Знач Код) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Lua | --lua | Произвольный | &#x2714; | Компонента Lua или версия Lua для запуска |
  | Код | --code | Строка | &#x2714; | Код строкой |


<div className="return-value-note">

  </div>
</div>

  | Параметр | Описание |
  |---|---|
  | addin_mode | Ручной выбор режима подключения внешней компоненты (для 1С): Изолированно, НеИзолированно |
