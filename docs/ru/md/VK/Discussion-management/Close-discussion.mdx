# Закрыть обсуждение
 Закрывает или удаляет обсуждение



`Функция ЗакрытьОбсуждение(Знач IDОбсуждения, Знач УдалитьПолностью = Ложь, Знач Параметры = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | IDОбсуждения | --topic | Строка,Число | &#x2714; | ID обсуждения |
  | УдалитьПолностью | --remove | Булево | &#x2716; | Удалить полностью (Истина) или закрыть |
  | Параметры | --auth | Структура из Строка | &#x2716; | JSON авторизации или путь к .json |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от VK

<br/>




```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Параметры    = ПолучитьПараметрыВК();
    ИДОбсуждения = "54106885";
    Результат    = OPI_VK.ЗакрытьОбсуждение(ИДОбсуждения, Ложь, Параметры);
```
