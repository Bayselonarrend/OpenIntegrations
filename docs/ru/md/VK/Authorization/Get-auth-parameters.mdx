# Получить параметры авторизации
 Получает структуру параметров авторизации



`Функция ПолучитьПараметрыАвторизации(Знач IDГруппы, Знач IDПриложения, Знач ТокенАвторизации) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | IDГруппы | --group | Строка, Число | &#x2714; | ID целевой группы |
  | IDПриложения | --app | Строка, Число | &#x2714; | ID приложения |
  | ТокенАвторизации | --token | Строка | &#x2714; | Токен авторизации |

  
  Возвращаемое значение:   Структура - Получить параметры авторизации
