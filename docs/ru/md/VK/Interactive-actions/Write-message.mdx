  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Текст | --text | Строка | &#x2714; | Текст сообщения |
  | IDПользователя | --user | Строка | &#x2714; | ID пользователя адресата |
  | Communitytoken | --ct | Строка | &#x2714; | Токен бота чата сообщества, который можно получить в настройках |
  | Клавиатура | --keyboard | Строка | &#x2716; | JSON клавиатуры. См. [`СформироватьКлавиатуру`](/docs/VK/Others/Form-keyboard) |
  | Параметры | --auth | Структура из Строка | &#x2716; | Параметры авторизации. См. [`ПолучитьПараметрыАвторизации`](/docs/VK/Authorization/Get-auth-parameters) |
