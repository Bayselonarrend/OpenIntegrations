# Сделать репост
 Делает репост записи



`Функция СделатьРепост(Знач IDПоста, Знач IDСтены = "", Знач ЦелеваяСтена = "", Знач Рекламный = Ложь, Знач Параметры = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | IDПоста | --post | Строка, Число | &#x2714; | ID поста |
  | IDСтены | --from | Строка, Число | &#x2716; | ID стены расположения поста |
  | ЦелеваяСтена | --to | Строка, Число | &#x2716; | ID целевой стены или группы |
  | Рекламный | --ad | Булево | &#x2716; | Признак рекламного поста |
  | Параметры | --auth | Структура из Строка | &#x2716; | Параметры авторизации. См. [`ПолучитьПараметрыАвторизации`](/docs/VK/Authorization/Get-auth-parameters) |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от VK



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Параметры = ПолучитьПараметрыВК();
    IDПоста   = 2571;
    IDСтены   = -218704372;

    Результат = OPI_VK.СделатьРепост(IDПоста, IDСтены, , , Параметры);
```
