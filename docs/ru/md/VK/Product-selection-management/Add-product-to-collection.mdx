# Добавить товар в подборку
 Добавляет товар в подборку



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ДобавитьТоварВПодборку(Знач МассивТоваров, Знач Подборка, Знач Параметры = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | МассивТоваров | --items | Массив из Строка, Число | &#x2714; | Массив товаров или товар |
  | Подборка | --sel | Строка | &#x2714; | ID подборки |
  | Параметры | --auth | Структура из Строка | &#x2716; | Параметры авторизации. См. [`ПолучитьПараметрыАвторизации`](/docs/VK/Authorization/Get-auth-parameters) |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - сериализованный JSON ответа от VK
  </div>
</div>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его объект (только для 1С и OneScript) |


  </TabItem>

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Параметры = ПолучитьПараметрыВК();

    Товар    = "12115889";
    Подборка = "144";

    Результат = OPI_VK.ДобавитьТоварВПодборку(Товар, Подборка, Параметры);
```
