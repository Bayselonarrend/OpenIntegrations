# Изменить подборку товаров
 Изменяет свойства подборки товаров



`Функция ИзменитьПодборкуТоваров(Знач Название, Знач Подборка, Знач Картинка = "", Знач Основная = Ложь, Знач Скрытая = Ложь, Знач Параметры = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Название | --title | Строка | &#x2714; | Новое название подборки |
  | Подборка | --sel | Строка | &#x2714; | ID подборки |
  | Картинка | --picture | Строка,ДвоичныеДанные | &#x2716; | Новая картинка подборки |
  | Основная | --main | Булево | &#x2716; | Основная |
  | Скрытая | --hidden | Булево | &#x2716; | Скрытая |
  | Параметры | --auth | Структура из Строка | &#x2716; | JSON авторизации или путь к .json |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от VK

<br/>

:::tip
Параметры с типом данных Двоичные данные могут принимать также пути к файлам на диске и URL
:::
<br/>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Параметры = ПолучитьПараметрыВК();

    Название = "Измененная подборка";
    Подборка = "137";

    Результат = OPI_VK.ИзменитьПодборкуТоваров(Название, Подборка, , , , Параметры);
```
