# Удалить подборку
 Удаляет подборку по ID



`Функция УдалитьПодборку(Знач Подборка, Знач Параметры = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Подборка | --sel | Строка | &#x2714; | ID подборки |
  | Параметры | --auth | Структура из Строка | &#x2716; | Параметры авторизации. См. [`ПолучитьПараметрыАвторизации`](/docs/VK/Authorization/Get-auth-parameters) |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от VK



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Параметры = ПолучитьПараметрыВК();

    Подборка = "149";

    Результат = OPI_VK.УдалитьПодборку(Подборка, Параметры);
```
