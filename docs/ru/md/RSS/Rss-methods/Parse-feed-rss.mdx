# Разобрать фид (RSS)
 Разбирает XML RSS фида и возвращает структуру с данными канала и элементами



`Функция РазобратьФидRSS(Знач ТекстXML) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ТекстXML | --xml | Строка | &#x2714; | XML содержимое RSS фида |

  
  Возвращаемое значение:   Структура Из КлючИЗначение - Структура канала



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ФидXML = "";

    Результат = OPI_RSS.РазобратьФидRSS(ФидXML);
```
