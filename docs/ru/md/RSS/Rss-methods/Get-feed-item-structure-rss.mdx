# Получить структуру элемента фида (RSS)
 Получает структуру параметров элемента RSS фида



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьСтруктуруЭлементаФидаRSS(Знач Пустая = Ложь, Знач КакСоответствие = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Пустая | --empty | Булево | &#x2716; | Истина > структура с пустыми значениями, Ложь > в значениях будут описания полей |
  | КакСоответствие | --map | Булево | &#x2716; | Истина > возвращает поля как соответствие |

  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

*Для текущего метода нет дополнительных параметров расширенного вызова.*


  </TabItem>
</Tabs>
  
  Возвращаемое значение:   Структура Из КлючИЗначение - Структура полей элемента фида



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Результат = OPI_RSS.ПолучитьСтруктуруЭлементаФидаRSS();
```
    

 <Tabs>
  
    <TabItem value="bash" label="Bash" default>
        ```bash
            oint rss ПолучитьСтруктуруЭлементаФидаRSS \
              --empty false \
              --map true
        ```
    </TabItem>
  
    <TabItem value="bat" label="CMD/Bat" default>
        ```batch
            oint rss ПолучитьСтруктуруЭлементаФидаRSS ^
              --empty false ^
              --map true
        ```
    </TabItem>
</Tabs>
