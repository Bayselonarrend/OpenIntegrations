# Разобрать фид (Atom)
 Разбирает XML Atom фида и возвращает структуру с данными фида и элементами



`Функция РазобратьФидAtom(Знач ТекстXML) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ТекстXML | --xml | Строка | &#x2714; | XML содержимое Atom фида |

  
  Возвращаемое значение:   Структура Из КлючИЗначение - Структура фида



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ФидXML = "<?xml version="1.0" encoding="UTF-8"?>
<feed xmlns="http://www.w3.org/2005/Atom">
    <title>Тестовый Atom фид</title>
    <id>https://example.com/feed</id>
    <link href="https://example.com" rel="alternate"/>
    <updated>2026-03-07T12:32:51</updated>
    <entry>
        <title>Первая запись</title>
        <id>https://example.com/entry1</id>
        <link href="https://example.com/entry1" rel="alternate"/>
        <updated>2026-03-07T12:32:51</updated>
        <summary>Краткое описание первой записи</summary>
        <content type="html">Полное содержимое первой записи</content>
        <author>
            <name>Автор Первый</name>
        </author>
        <published>2026-03-07T12:32:51</published>
    </entry>
    <entry>
        <title>Вторая запись</title>
        <id>https://example.com/entry2</id>
        <link href="https://example.com/entry2" rel="alternate"/>
        <updated>2026-03-07T12:32:51</updated>
        <summary>Краткое описание второй записи</summary>
        <content type="html">Полное содержимое второй записи</content>
        <author>
            <name>Автор Второй</name>
        </author>
        <published>2026-03-07T12:32:51</published>
    </entry>
</feed>";

    Результат = OPI_RSS.РазобратьФидAtom(ФидXML);
```
