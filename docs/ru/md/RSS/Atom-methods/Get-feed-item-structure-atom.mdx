# Получить структуру элемента фида (Atom)
 Получает структуру параметров элемента Atom фида



`Функция ПолучитьСтруктуруЭлементаФидаAtom(Знач Пустая = Ложь, Знач КакСоответствие = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Пустая | --empty | Булево | &#x2716; | Истина > структура с пустыми значениями, Ложь > в значениях будут описания полей |
  | КакСоответствие | --map | Булево | &#x2716; | Истина > возвращает поля как соответствие |

  
  Возвращаемое значение:   Структура Из КлючИЗначение - Структура полей элемента фида



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Результат = OPI_RSS.ПолучитьСтруктуруЭлементаФидаAtom();
```
