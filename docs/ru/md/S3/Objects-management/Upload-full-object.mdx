# Загрузить объект целиком
 Загружает объект на сервер не используя загрузку по частям



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ЗагрузитьОбъектЦеликом(Знач Наименование, Знач Бакет, Знач Содержимое, Знач ОсновныеДанные, Знач Заголовки = Неопределено, Знач Каталог = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Наименование | --name | Строка | &#x2714; | Наименование объекта в бакете |
  | Бакет | --bucket | Строка | &#x2714; | Имя бакета для загрузки объекта |
  | Содержимое | --data | Строка, ДвоичныеДанные | &#x2714; | Данные или путь к файлу для загрузки |
  | ОсновныеДанные | --basic | Структура Из КлючИЗначение | &#x2714; | Основные данные запроса. См. [`ПолучитьСтруктуруДанных`](/docs/S3/Common-methods/Get-basic-data-structure) |
  | Заголовки | --headers | Соответствие Из КлючИЗначение | &#x2716; | Дополнительные заголовки запроса, если необходимо |
  | Каталог | --dir | Булево | &#x2716; | Истина > Path style URL, Ложь > Virtual hosted style URL |

  | Параметр | Описание |
  |---|---|
  | proxy | ИнтернетПрокси или структура полей Протокол, Сервер, Порт, Пользователь, Пароль, ИспользоватьАутентификациюОС |
  | timeout | Таймаут выполнения запроса |
  | adv_response | Оформляет ответ в виде полной структуры с полями code, body и headers |
  | retries | Число попыток отправки при коде `5**` или внутренних ошибках клиента |
  | dontwait | Создает фоновое задание и возвращает его объект (только для 1С и OneScript) |


  </TabItem>
</Tabs>

:::tip
Метод в документации AWS: [PutObject](https://docs.aws.amazon.com/AmazonS3/latest/API/API_PutObject.html)

Это служебный метод. Для простого сценария загрузки файлов предназначен единый метод `ЗагрузитьОбъект`<br/>Использование данного метода для больших файлов может приводить к сбоям

Параметры с типом данных Двоичные данные могут принимать также пути к файлам на диске и URL
:::
<br/>


```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL       = "s3.openintegrations.dev";
    AccessKey = "bayselo...";
    SecretKey = "12we34...";
    Region    = "BTC";

    ОсновныеДанные = OPI_S3.ПолучитьСтруктуруДанных(URL, AccessKey, SecretKey, Region);

    Наименование = "pictureSmall.jpg";
    Содержимое   = "https://hut.openintegrations.dev/test_data/picture.jpg"; // URL, Путь или Двоичные данные

    Каталог = Истина; // Формирование URL в path-style
    Бакет = "w567hjy1";
    Результат = OPI_S3.ЗагрузитьОбъектЦеликом(Наименование, Бакет, Содержимое, ОсновныеДанные, , Каталог);
```
