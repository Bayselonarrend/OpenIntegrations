# Получить структуру данных
 Возвращает основные данные запроса в структурированном виде



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьСтруктуруДанных(Знач URL, Знач AccessKey, Знач SecretKey, Знач Region, Знач Service = "s3") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL: домен для обычных методов или полный URL с параметрами для прямой отправки запросов |
  | AccessKey | --access | Строка | &#x2714; | Access key для авторизации |
  | SecretKey | --secret | Строка | &#x2714; | Secret key для авторизации |
  | Region | --region | Строка | &#x2714; | Регион сервиса |
  | Service | --service | Строка | &#x2716; | Вид сервиса, если отличен от s3 |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Структура Из КлючИЗначение -  Структура основных данных запроса
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его данные (только для 1С и OneScript) |


  </TabItem>

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL       = "s3.openintegrations.dev";
    AccessKey = "bayselo...";
    SecretKey = "12we34...";
    Region    = "BTC";

    Результат = OPI_S3.ПолучитьСтруктуруДанных(URL, AccessKey, SecretKey, Region);
```
