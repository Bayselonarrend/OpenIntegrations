# Получить токен
 Получает токен авторизации по логину и паролю



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьТокен(Знач Логин, Знач Пароль) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Логин | --login | Строка | &#x2714; | Логин пользователя |
  | Пароль | --password | Строка | &#x2714; | Пароль пользователя |
