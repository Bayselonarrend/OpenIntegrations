# Получить настройки (логин/пароль)
 Получает настройки соединения с авторизацией через логин и пароль



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПолучитьНастройкиЛогинПароль(Знач Хост, Знач Порт, Знач Логин, Знач Пароль = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Хост | --host | Строка | &#x2714; | Хост SSH |
  | Порт | --port | Число | &#x2714; | Порт SSH |
  | Логин | --user | Строка | &#x2714; | Имя пользователя SSH |
  | Пароль | --pass | Строка | &#x2716; | Пароль пользователя SSH |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Структура Из КлючИЗначение -  Конфигурация соединения
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его объект (только для 1С и OneScript) |


  </TabItem>
</Tabs>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Хост   = "172.33.0.13";
    Порт   = "2222";
    Логин  = "bayselonarrend";
    Пароль = "12we...";

    Результат = OPI_SSH.ПолучитьНастройкиЛогинПароль(Хост, Порт, Логин, Пароль);
```
