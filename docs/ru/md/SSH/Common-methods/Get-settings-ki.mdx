# Получить настройки (keyboard-interactive)
 Получает настройки соединения с keyboard-interactive авторизацией



`Функция ПолучитьНастройкиKI(Знач Хост, Знач Порт, Знач Логин, Знач ПорядокВвода) Экспорт`

<Tabs>
  <TabItem value="params" label="Параметры" default>

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Хост | --host | Строка | &#x2714; | Хост SSH |
  | Порт | --port | Число | &#x2714; | Порт SSH |
  | Логин | --user | Строка | &#x2714; | Имя пользователя SSH |
  | ПорядокВвода | --prompts | Массив Из Строка | &#x2714; | Массив ответов на вопросы сервера |

  
  </TabItem>
  <TabItem value="extended" label="Расширенный вызов">

*Для текущего метода нет дополнительных параметров расширенного вызова.*


  </TabItem>
</Tabs>
  
  Возвращаемое значение:   Структура Из КлючИЗначение -  Конфигурация соединения



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Хост   = "172.33.0.13";
    Порт   = "2222";
    Логин  = "bayselonarrend";
    Пароль = "12we...";

    МассивОтветов = Новый Массив;
    МассивОтветов.Добавить("yes");
    МассивОтветов.Добавить(Пароль);

    Результат = OPI_SSH.ПолучитьНастройкиKI(Хост, Порт, Логин, МассивОтветов);
```
    



```json title="Результат"
{
 "auth_type": "keyboard_interactive",
 "host": "172.33.0.13",
 "port": 2222,
 "username": "bayselonarrend",
 "keyboard_responses": [
  "yes",
  "***"
 ]
}
```
