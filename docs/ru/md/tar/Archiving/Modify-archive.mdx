# Изменить архив
 Добавляет, заменяет и удаляет файлы в существующем tar-архиве



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ИзменитьАрхив(Знач Архив, Знач ДобавляемыеФайлы = Неопределено, Знач УдаляемыеПути = Неопределено, Знач Настройки = Неопределено) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Архив | --src | Строка, ДвоичныеДанные | &#x2714; | Путь к архиву на диске или двоичные данные |
  | ДобавляемыеФайлы | --additions | Соответствие Из КлючИЗначение | &#x2716; | Ключ: путь в архиве; значение: путь к файлу или двоичные данные |
  | УдаляемыеПути | --deletions | Массив Из Строка | &#x2716; | Пути файлов в архиве для удаления |
  | Настройки | --settings | Соответствие Из КлючИЗначение | &#x2716; | Настройки изменения. См. [`ПолучитьСтруктуруНастроекАрхивации`](/docs/tar/Archiving/Get-archiving-settings-structure) |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  ДвоичныеДанные, Соответствие Из КлючИЗначение - Результат операции для архива на диске или новые двоичные данные архива

  </div>
</div>
  
  </TabItem>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его данные (только для 1С и OneScript) |


  </TabItem>
