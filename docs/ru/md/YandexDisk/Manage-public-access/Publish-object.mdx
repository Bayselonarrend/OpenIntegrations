# Опубликовать объект
 Публикует объект диска в публичный доступ



`Функция ОпубликоватьОбъект(Знач Токен, Знач Путь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Путь | --path | Строка | &#x2714; | Путь к публикуемому объекту |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Yandex

<br/>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен = "y0__xCOranuBRj1uTsgs8z_iRV...";
    Путь  = "/d3b18a1c-a25e-4de8-b996-8e904330bf32.png";

    Результат = OPI_YandexDisk.ОпубликоватьОбъект(Токен, Путь);
```
