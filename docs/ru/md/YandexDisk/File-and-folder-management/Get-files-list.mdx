# Получить список файлов
Получает список файлов с или без отбора по типу Список доступных типов: audio, backup, book, compressed, data, development, diskimage, document, encoded, executable, flash, font, mage, settings, spreadsheet, text, unknown, video, web



`Функция ПолучитьСписокФайлов(Знач Токен, Знач Количество = 0, Знач СмещениеОтНачала = 0, Знач ОтборПоТипу = "", Знач СортироватьПоДате = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Количество | --amount | Число,Строка | &#x2716; | Количество возвращаемых объектов |
  | СмещениеОтНачала | --offset | Число | &#x2716; | Смещение для получение объектов не из начала списка |
  | ОтборПоТипу | --type | Строка | &#x2716; | Отбор по типу файла |
  | СортироватьПоДате | --datesort | Булево | &#x2716; | Истина > сортировать по дате, Ложь > по алфавиту |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Yandex



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен            = "y0__xCOranuBRj1uTsgs8z_iRV...";
    Количество       = 2;
    СмещениеОтНачала = 1;
    ОтборПоТипу      = "image";

    Результат = OPI_YandexDisk.ПолучитьСписокФайлов(Токен, Количество, СмещениеОтНачала, ОтборПоТипу);
```
    

 <Tabs>
  
    <TabItem value="bash" label="Bash" default>
        ```bash
            oint yadisk ПолучитьСписокФайлов \
              --token "***" \
              --amount 2 \
              --offset 1 \
              --type "image"
        ```
    </TabItem>
  
    <TabItem value="bat" label="CMD/Bat" default>

        ```batch
            oint yadisk ПолучитьСписокФайлов ^
              --token "***" ^
              --amount 2 ^
              --offset 1 ^
              --type "image"
        ```
