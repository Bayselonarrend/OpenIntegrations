# Получить ссылку для скачивания
 Получает ссылку для скачивания файла



`Функция ПолучитьСсылкуДляСкачивания(Знач Токен, Знач Путь) Экспорт`

<Tabs>
  <TabItem value="params" label="Параметры" default>

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Путь | --path | Строка | &#x2714; | Путь к файлу для скачивания |

  
  </TabItem>
  <TabItem value="extended" label="Расширенный вызов">

  | Параметр | Описание |
  |---|---|
  | proxy | ИнтернетПрокси или структура полей Протокол, Сервер, Порт, Пользователь, Пароль, ИспользоватьАутентификациюОС |
  | timeout | Таймаут выполнения запроса |
  | adv_response | Оформляет ответ в виде полной структуры с полями code, body и headers |


  </TabItem>
</Tabs>
  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Yandex



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен = "y0__xCOranuBRj1uTsgs8z_iRV...";
    Путь  = "/c20d5eca-8524-41ce-aa92-521e71dd6b17.png";

    Результат = OPI_YandexDisk.ПолучитьСсылкуДляСкачивания(Токен, Путь);
```
    

 <Tabs>
