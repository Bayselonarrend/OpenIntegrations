# Получить объект
 Получает информацию об объекте диска по заданному пути



`Функция ПолучитьОбъект(Знач Токен, Знач Путь) Экспорт`

<Tabs>
  <TabItem value="params" label="Параметры" default>

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Путь | --path | Строка | &#x2714; | Путь к папке или файлу |

  
  </TabItem>
  <TabItem value="extended" label="Расширенный вызов">

  | Параметр | Описание |
  |---|---|
  | proxy | ИнтернетПрокси или структура полей Протокол, Сервер, Порт, Пользователь, Пароль, ИспользоватьАутентификациюОС |
  | timeout | Таймаут выполнения запроса |
  | adv_response | Оформляет ответ в виде полной структуры с полями code, body и headers |


  </TabItem>
</Tabs>
  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Yandex



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен = "y0__xCOranuBRj1uTsgs8z_iRV...";
    Путь  = "/ec98cf96-9738-40a1-b9bc-a24a2f049cb0.png";

    Результат = OPI_YandexDisk.ПолучитьОбъект(Токен, Путь);
```
    

 <Tabs>
  
    <TabItem value="bash" label="Bash" default>
        ```bash
            oint yadisk ПолучитьОбъект \
              --token "***" \
              --path "/b780008e-003e-4f52-82cc-be893b2c682a.png"
        ```
    </TabItem>
  
    <TabItem value="bat" label="CMD/Bat" default>

