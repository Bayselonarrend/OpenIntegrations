# Получить объект
 Получает информацию об объекте диска по заданному пути



`Функция ПолучитьОбъект(Знач Токен, Знач Путь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Путь | --path | Строка | &#x2714; | Путь к папке или файлу |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Yandex



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен = "y0__xCOranuBRj1uTsgs8z_iRV...";
    Путь  = "/ec98cf96-9738-40a1-b9bc-a24a2f049cb0.png";

    Результат = OPI_YandexDisk.ПолучитьОбъект(Токен, Путь);
```
