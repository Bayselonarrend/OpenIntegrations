# Загрузить файл
 Загружает файл на диск по заданному пути



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ЗагрузитьФайл(Знач Токен, Знач Путь, Знач Файл, Знач Перезаписывать = Ложь) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Путь | --path | Строка | &#x2714; | Путь для сохранение файла на Диске |
  | Файл | --file | Строка, ДвоичныеДанные | &#x2714; | Файл для загрузки |
  | Перезаписывать | --rewrite | Булево | &#x2716; | Перезаписывать, если файл с таким именем уже существует |

```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Путь1 = "/" + Строка(Новый УникальныйИдентификатор) + ".png";
    Путь2 = "/" + Строка(Новый УникальныйИдентификатор) + ".png";

    Токен    = "y0__xCOranuBRj1uTsgs8z_iRV...";
    Картинка = "https://hut.openintegrations.dev/test_data/picture.jpg"; // URL

    ИВФ = ПолучитьИмяВременногоФайла("png"); // Путь
    КопироватьФайл(Картинка, ИВФ);

    Результат = OPI_YandexDisk.ЗагрузитьФайл(Токен, Путь1, Картинка, Истина);

    Результат = OPI_YandexDisk.ЗагрузитьФайл(Токен, Путь2, ИВФ, Истина);
```
