# Удалить объект
 Удаляет объект по заданному пути



`Функция УдалитьОбъект(Знач Токен, Знач Путь, Знач ВКорзину = Истина) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Токен | --token | Строка | &#x2714; | Токен |
  | Путь | --path | Строка | &#x2714; | Путь к удаляемой папке или файлу |
  | ВКорзину | --can | Булево | &#x2716; | В корзину |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Yandex

<br/>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    Токен = "y0__xCOranuBRj1uTsgs8z_iRV...";
    Путь  = "/bfca8f29-5b7a-4dc4-bd3a-3cdec0c4c961.png";

    Результат = OPI_YandexDisk.УдалитьОбъект(Токен, Путь, Ложь);
```
