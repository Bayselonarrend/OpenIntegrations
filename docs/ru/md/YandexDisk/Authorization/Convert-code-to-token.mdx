# Преобразовать код в токен
 Преобразовывает код в токен после ввода кода при выполнении ПолучитьКодПодтверждения



<Tabs>
  <TabItem value="params" label="Параметры" default>

`Функция ПреобразоватьКодВТокен(Знач ClientId, Знач ClientSecret, Знач КодУстройства) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ClientId | --id | Строка | &#x2714; | Client id |
  | ClientSecret | --secret | Строка | &#x2714; | Client secret |
  | КодУстройства | --device | Строка | &#x2714; | device_code из ПолучитьКодПодтверждения |


<div className="return-value-note">
  <div className="return-value-note__title">Возвращаемое значение</div>
  <div className="return-value-note__value">
  Соответствие Из КлючИЗначение - сериализованный JSON ответа от Yandex
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Расширенный вызов{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="Справка о расширенном вызове" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Параметр | Описание |
  |---|---|
  | dontwait | Создает фоновое задание и возвращает его объект (только для 1С и OneScript) |


  </TabItem>
</Tabs>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ClientID      = "cc2ce9d60deb4975be58866d32dbe7aa";
    ClientSecret  = "7cc0b903d33e41c1...";
    КодУстройства = "12345678";

    Результат = OPI_YandexDisk.ПреобразоватьКодВТокен(ClientID, ClientSecret, КодУстройства);
```
