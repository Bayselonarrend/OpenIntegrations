# Обновить токен
 Обновляет токен по Refresh token



`Функция ОбновитьТокен(Знач ClientId, Знач ClientSecret, Знач RefreshToken) Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | ClientId | --id | Строка | &#x2714; | Client id |
  | ClientSecret | --secret | Строка | &#x2714; | Client secret |
  | RefreshToken | --refresh | Строка | &#x2714; | Refresh token |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - сериализованный JSON ответа от Yandex



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    ClientID     = "cc2ce9d60deb4975be58866d32dbe7aa";
    ClientSecret = "7cc0b903d33e41c1...";
    RefreshToken = "2:AAA:AAAAAF3KVo4:1:7lDA887FZi3e_EYF:4LGI1j1d3RMVs-4532KZK7ZQbG0F...";

    Результат = OPI_YandexDisk.ОбновитьТокен(ClientID, ClientSecret, RefreshToken);
```
