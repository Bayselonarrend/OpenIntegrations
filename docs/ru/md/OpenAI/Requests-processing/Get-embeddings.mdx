# Получить представления
 Получает представления (embeddings) для заданных вводных



`Функция ПолучитьПредставления(Знач URL, Знач Токен, Знач Модель, Знач Текст, Знач ДопПараметры = "", Знач ДопЗаголовки = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | URL | --url | Строка | &#x2714; | URL сервера OpenAI |
  | Токен | --token | Строка | &#x2714; | Токен авторизации OpenAI |
  | Модель | --model | Строка | &#x2714; | Имя модели |
  | Текст | --input | Массив Из Строка | &#x2714; | Строка или массив строк запросов |
  | ДопПараметры | --options | Структура Из КлючИЗначение | &#x2716; | Доп. параметры запроса, если необходимо |
  | ДопЗаголовки | --headers | Соответствие Из КлючИЗначение | &#x2716; | Доп. заголовки запроса, если необходимо |

  
  Возвращаемое значение:   Соответствие Из КлючИЗначение - Результат обработки

<br/>

:::tip
Метод в документации API: [Create embeddings](https://platform.openai.com/docs/api-reference/embeddings/create)
:::
<br/>



```bsl title="Пример использования для 1С:Предприятие/OneScript"
    URL   = "https://hut.openintegrations.dev/localai/";
    Токен = "12We...";

    Текст  = "What is 1C:Enterprise?";
    Модель = "text-embedding-ada-002";

    Результат = OPI_OpenAI.ПолучитьПредставления(URL, Токен, Модель, Текст);
```
