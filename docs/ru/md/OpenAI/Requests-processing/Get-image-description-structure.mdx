# Получить структуру описания картинок
 Получить структуру описания картинок для генерации



`Функция ПолучитьСтруктуруОписанияКартинок(Знач Промпт, Знач Количество, Знач Фон = "", Знач Размер = "", Знач ДопПараметры = "") Экспорт`

  | Параметр | CLI опция | Тип | Обяз. | Назначение |
  |-|-|-|-|-|
  | Промпт | --prompt | Строка | &#x2714; | Текстовое описание картинки для генерации |
  | Количество | --amount | Число | &#x2714; | Количество изображений для генерации |
  | Фон | --bg | Строка | &#x2716; | Вариант генерации фона: transparent, opaque, auto |
  | Размер | --size | Строка | &#x2716; | Вариант размера сгенерированных изображений |
  | ДопПараметры | --options | Структура Из КлючИЗначение | &#x2716; | Доп. параметры запроса, если необходимо |

  
  Возвращаемое значение:   Структура -  Структура полей

:::tip
Набор полей и их интерпретация может отличаться в зависимости от используемой модели
:::
