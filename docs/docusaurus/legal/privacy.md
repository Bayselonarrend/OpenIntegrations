# ПОЛИТИКА КОНФИДЕНЦИАЛЬНОСТИ

## 1. ОБЩИЕ ПОЛОЖЕНИЯ

1.1. Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей сайта https://openintegrations.dev (далее — «Сайт») Индивидуальным предпринимателем Титовым Антоном Сергеевичем (далее — «Оператор»).

1.2. Используя Сайт, Вы соглашаетесь с условиями настоящей Политики.

## 2. СОСТАВ ПЕРСОНАЛЬНЫХ ДАННЫХ

2.1. Оператор может собирать следующую информацию:
- Фамилия, имя, отчество;
- Адрес электронной почты;
- Данные об оплате (через сервис Boosty);
- Техническая информация (IP-адрес, данные cookies).

## 3. ЦЕЛИ СБОРА ПЕРСОНАЛЬНЫХ ДАННЫХ

3.1. Персональные данные собираются для следующих целей:
- предоставление доступа к образовательным курсам;
- обработка платежей;
- техническая поддержка;
- информирование о новых курсах и акциях.

## 4. ОБРАБОТКА ПЕРСОНАЛЬНЫХ ДАННЫХ

4.1. Обработка персональных данных осуществляется с согласия субъекта персональных данных.

4.2. Оператор принимает необходимые меры для защиты персональных данных от неправомерного доступа.

## 5. ПЕРЕДАЧА ПЕРСОНАЛЬНЫХ ДАННЫХ

5.1. Оператор может передавать персональные данные следующим третьим лицам:
- **Boosty (OOO «БУСТИ»)** — для обработки платежей;
- **Хостинг-провайдеры** — для технического обеспечения работы Сайта.

5.2. Передача персональных данных иным третьим лицам осуществляется только с согласия субъекта персональных данных.

## 6. ХРАНЕНИЕ ПЕРСОНАЛЬНЫХ ДАННЫХ

6.1. Персональные данные хранятся в течение срока, необходимого для достижения целей обработки.

## 7. ПРАВА СУБЪЕКТА ПЕРСОНАЛЬНЫХ ДАННЫХ

7.1. Вы имеете право:
- на доступ к своим персональным данным;
- на уточнение (обновление, изменение) персональных данных;
- на уничтожение персональных данных;
- на отзыв согласия на обработку персональных данных.

## 8. COOKIES И СИСТЕМЫ АНАЛИТИКИ

8.1. Сайт использует минимально необходимые cookies для:
- функционирования базовых функций сайта
- запоминания предпочтения темы (светлая/темная)
- сбора анонимной статистики посещаемости (если подключена аналитика)

8.2. Мы не используем cookies для:
- отслеживания поведения пользователей
- таргетированной рекламы
- сбора персональных данных без согласия

8.3. Вы можете отключить cookies в настройках браузера, 
однако это может повлиять на работу некоторых функций сайта.

## 9. БЕЗОПАСНОСТЬ ПЕРСОНАЛЬНЫХ ДАННЫХ

9.1. Оператор принимает необходимые организационные и технические меры для защиты персональных данных.
