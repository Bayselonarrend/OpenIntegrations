# Execute bytecode 
 Executes previously compiled code



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function ExecuteBytecode(Val Lua, Val Bytecode) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Lua | - | Arbitrary | &#x2714; | Lua AddIn or Lua version to run |
  | Bytecode | - | BinaryData | &#x2714; | Bytecode to execute |

