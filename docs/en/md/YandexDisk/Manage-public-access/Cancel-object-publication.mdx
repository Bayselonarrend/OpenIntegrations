# Unpublish object
 Unpublishes a previously published object



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function CancelObjectPublication(Val Token, Val Path) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Path | --path | String | &#x2714; | Path to the previously published object |

  <div className="return-value-note__title">Returns</div>

 Map Of KeyAndValue - serialized JSON response from Yandex
