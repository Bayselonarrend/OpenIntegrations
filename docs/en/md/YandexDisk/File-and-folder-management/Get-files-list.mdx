# Get list of files
Gets a list of files with or without filtering by type List of available types: audio, backup, book, compressed, data, development, diskimage, document, encoded, executable, flash, font, mage, settings, spreadsheet, text, unknown, video, web



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetFilesList(Val Token, Val Count = 0, Val OffsetFromStart = 0, Val FilterByType = "", Val SortByDate = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Count | --amount | Number, String | &#x2716; | Number of returned objects |
  | OffsetFromStart | --offset | Number | &#x2716; | Offset for getting objects not from the beginning of the list |
  | FilterByType | --type | String | &#x2716; | Filter by file type |
  | SortByDate | --datesort | Boolean | &#x2716; | True > sort by date, False > alphabetically |
