# Upload file by URL
 Downloads a file to disk from the specified URL



`Function UploadFileByURL(Val Token, Val Path, Val Address) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Path | --path | String | &#x2714; | Path to place the downloaded file |
  | Address | --url | String | &#x2714; | File URL |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Yandex



```bsl title="1C:Enterprise/OneScript code example"
    Token   = "y0__xCOranuBRj1uTsgs8z_iRV...";
    Address = "https://hut.openintegrations.dev/test_data/picture.jpg";
    Path    = "/" + String(New UUID) + ".png";

    Options = New Structure;
    Options.Insert("token", Token);
    Options.Insert("path" , Path);
    Options.Insert("url"  , Address);

    Result = OPI_TestDataRetrieval.ExecuteTestCLI("yadisk", "UploadFileByURL", Options);
```
