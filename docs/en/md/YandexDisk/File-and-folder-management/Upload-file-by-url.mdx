# Upload file by URL
 Downloads a file to disk from the specified URL



<Tabs>
  <TabItem value="params" label="Parameters" default>

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Path | --path | String | &#x2714; | Path to place the downloaded file |
  | Address | --url | String | &#x2714; | File URL |
