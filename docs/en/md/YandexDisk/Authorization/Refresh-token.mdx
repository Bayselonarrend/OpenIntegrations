# Refresh token
 Updates token by Refresh token



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function RefreshToken(Val ClientId, Val ClientSecret, Val RefreshToken) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | ClientId | --id | String | &#x2714; | Client id |
  | ClientSecret | --secret | String | &#x2714; | Client secret |
  | RefreshToken | --refresh | String | &#x2714; | Refresh token |
