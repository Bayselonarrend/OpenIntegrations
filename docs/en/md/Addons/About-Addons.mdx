# About Add-ons

**Add-ons** are various tools based on the Open Integration Package but developed as separate projects. They extend the functionality or expand the application scope of the Open Integration Package in specific domains and typically depend on one of its core distributions

This section lists all currently available add-ons. Installation and operation may vary, so refer to each add-on's dedicated subsection for details
