# Update scheduled task
 Changes the schedule of the selected scheduled task



`Function UpdateScheduledTask(Val Project, Val TaskID, Val Schedule = "", Val HandlersKey = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Project | --proj | String | &#x2714; | Project filepath |
  | TaskID | --task | String | &#x2714; | Task ID |
  | Schedule | --cron | String | &#x2716; | Schedule in extended cron format |

  
  Returns:  Structure Of KeyAndValue - Task update result

<br/>



 

 <Tabs>
  
    <TabItem value="bash" label="Bash" default>
        ```bash
            melezh  UpdateScheduledTask \
              --proj "/proj.melezh" \
              --task 1 \
              --cron "0 8 8 * * * *"
        ```
    </TabItem>
  
    <TabItem value="bat" label="CMD/Bat" default>
        ```batch

              --proj "/proj.melezh" ^
              --task 1 ^
              --cron "0 8 8 * * * *"
        ```
    </TabItem>
