# Delete scheduled task
 Deletes scheduled task from project



`Function DeleteScheduledTask(Val Project, Val TaskID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Project | --proj | String | &#x2714; | Project filepath |
  | TaskID | --task | Number | &#x2714; | Task ID |

  
  Returns:  Structure Of KeyAndValue - Deleting result

<br/>



 

 <Tabs>
  
    <TabItem value="bash" label="Bash" default>
        ```bash

              --proj "/proj.melezh" \
              --task 1
        ```
