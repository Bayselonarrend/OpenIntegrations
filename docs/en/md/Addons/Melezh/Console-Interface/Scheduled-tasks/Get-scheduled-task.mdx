# Get scheduled task
 Gets task information by ID



`Function GetScheduledTask(Val Project, Val TaskID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Project | --proj | String | &#x2714; | Project filepath |
  | TaskID | --task | String | &#x2714; | Task ID |

  
  Returns:  Structure Of KeyAndValue - Handlers information

<br/>



 

 <Tabs>
  
    <TabItem value="bash" label="Bash" default>
        ```bash
            melezh  GetScheduledTask \
              --proj "/proj.melezh" \
              --task 1
        ```
    </TabItem>
  
    <TabItem value="bat" label="CMD/Bat" default>
        ```batch
            melezh  GetScheduledTask ^
              --proj "/proj.melezh" ^
              --task 1
        ```
    </TabItem>
</Tabs>
