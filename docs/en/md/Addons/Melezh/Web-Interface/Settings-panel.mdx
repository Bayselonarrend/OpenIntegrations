The final panel in the web interface is the **Settings Panel**.

## Overview

![Settings Panel](../../../../static/img/Docs/Melezh/en/20.png)

The Settings Panel displays all general project configurations that can be modified (except for the web console password). This includes logging settings that allow you to:

- Change the log storage directory
- Enable/disable recording of specific request elements
- Set maximum data size saved per event
