# Set string body 
 Sets the body of the request from the string



`Function SetStringBody(Val Data, Val WriteBOM = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Data | - | String | &#x2714; | Request body data |
  | WriteBOM | - | Boolean | &#x2716; | True > BOM will be added |

  
  Returns:  DataProcessorObject.OPI_HTTPClient - This processor object

<br/>


:::caution
**NOCLI:** this method is not available in CLI version
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    URL = "https://bin.openintegrations.dev";
    URL = URL + "/post";

    Text     = "Hello world!";
    Encoding = "Windows-1251";

    Result = OPI_HTTPRequests.NewRequest()
        .Initialize(URL)
        .UseEncoding(Encoding)
        .SetStringBody(Text) // <---
        .ProcessRequest("POST")
        .ReturnResponseAsJSONObject();
```
