# Execute request 
 Executes the request if it was created earlier



`Function ExecuteRequest(Forced = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Forced | - | Boolean | &#x2716; | Attempted execution without additional checks |

  
  Returns:  DataProcessorObject.OPI_HTTPClient - This processor object

<br/>


:::caution
**NOCLI:** this method is not available in CLI version
:::
<br/>



```bsl title="1C:Enterprise/OneScript code example"
    URL = FunctionParameters["HTTP_URL"];
    URL = URL + "/get";

    Result = OPI_HTTPRequests.NewRequest()
        .Initialize()
        .SetURL(URL)
        .ProcessRequest("GET", False)
        .ExecuteRequest() // <---
        .ReturnResponseAsJSONObject();
```
