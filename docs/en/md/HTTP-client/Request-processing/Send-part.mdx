# Send part 
 Sends a request with the specified part of the body and the Content-Range header



`Function SendPart(Val StartPosition, Val ByteCount, Val Method = "PUT") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | StartPosition | - | Number | &#x2714; | Start position in request body |
  | ByteCount | - | Number | &#x2714; | Number of bytes from the specified start position for sending |
  | Method | - | String | &#x2716; | Request HTTP method |

  
  Returns:  DataProcessorObject.OPI_HTTPClient - This processor object

<br/>


:::caution
**NOCLI:** this method is not available in CLI version
:::
