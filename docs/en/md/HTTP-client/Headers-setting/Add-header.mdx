# Add header 
 Adds a header to the request header set



`Function AddHeader(Val Name, Val Value) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Name | - | String | &#x2714; | Header key |
  | Value | - | String | &#x2714; | Header value |

  
  Returns:  DataProcessorObject.OPI_HTTPClient - This processor object

<br/>


:::caution
**NOCLI:** this method is not available in CLI version
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    URL = "https://bin.openintegrations.dev";
    URL = URL + "/get";

    Result = OPI_HTTPRequests.NewRequest()
        .Initialize()
        .SetURL(URL)
        .AddHeader("X-Header1", "Value1") // <---
        .AddHeader("X-Header2", "Value2") // <---
        .ProcessRequest("GET")
        .ReturnResponseAsJSONObject();
```
