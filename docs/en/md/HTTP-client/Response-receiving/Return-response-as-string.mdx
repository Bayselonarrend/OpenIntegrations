# Return response as string 
 Returns the body of the response as a string



`Function ReturnResponseAsString(Val Forced = False, Val ExceptionOnError = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Forced | - | Boolean | &#x2716; | False > The processor object will be returned instead of the response if there were errors in it |
  | ExceptionOnError | - | Boolean | &#x2716; | Causes an exception with a log if there were errors during processing |

  
  Returns:  Arbitrary - The response or the same processing object

<br/>


:::caution
**NOCLI:** this method is not available in CLI version
:::
