# Clear cells
 Clears the value in cells



`Function ClearCells(Val Token, Val Spreadsheet, Val CellsArray, Val Sheet = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Spreadsheet | --spreadsheet | String | &#x2714; | SpreadsheetID |
  | CellsArray | --cells | Array of String | &#x2714; | Array of cells like A1 to be cleared |
  | Sheet | --sheetname | String | &#x2716; | Sheet name (first sheet by default) |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google



```bsl title="1C:Enterprise/OneScript code example"
    Token       = "ya29.a0ATkoCc7NIaMoQyko9kNNsGA_NcNp2ra5WH144UV1BR2r9JdyTQocbBSQiR9QcLAI2btfyyE1OMQUkgMHs9xVvwkMlDaQ8dqWiQIftCNobgkG15t59vzvmxN2o...";
    Spreadsheet = "1r-GaurPYMb0giRz_CruzO25QhbwKR1dJ_ptgreEUMkU";
    Sheet       = "Sheet2";

    CellsArray = New Array;
    CellsArray.Add("B2");
    CellsArray.Add("A3");
    CellsArray.Add("B4");

    Result = OPI_GoogleSheets.ClearCells(Token, Spreadsheet, CellsArray, Sheet);
```
