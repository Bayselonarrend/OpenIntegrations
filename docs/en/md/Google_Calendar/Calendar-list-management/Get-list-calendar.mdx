# Get list calendar
 Gets a calendar from the user's list by ID



`Function GetListCalendar(Val Token, Val Calendar) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Calendar | --calendar | String | &#x2714; | Calendar ID |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Token    = "ya29.a0AeDClZCK27Pm2ZdUtEbN2HBs1Z_Cn1cGHTJoz3WARfMz0a65CdiiYfkOxC4ffw2tmZhzHx8ErHfYMQxHad19rUG-zyvipaCEkZw_DgEOd...";
    Calendar = "dab5d3445c73b7ed25725760d7f531a74a1be2c11a708994fa3546b6606dffdf@group.calendar.google.com";

    Result = OPI_GoogleCalendar.GetListCalendar(Token, Calendar);
```
