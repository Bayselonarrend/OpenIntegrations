# Get list calendar
 Gets a calendar from the user's list by ID



`Function GetListCalendar(Val Token, Val Calendar) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Calendar | --calendar | String | &#x2714; | Calendar ID |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Token    = "ya29.a0AZYkNZjPdv9SxyICmqh9ci1psUDsod4nm51k501h79FNnNVA8M4kk-12mOzBfaqyI2IjZpX5Jhp_Ex6onBvGGzr5JzMV9ipqaYWhn7NE8...";
    Calendar = "1787427d11cac1af55bf37e5112f26cd1468a657fdd99be3b213d0754454e427@group.calendar.google.com";

    Result = OPI_GoogleCalendar.GetListCalendar(Token, Calendar);
```
