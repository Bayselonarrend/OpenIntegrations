# Get list calendar
 Gets a calendar from the user's list by ID



`Function GetListCalendar(Val Token, Val Calendar) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Calendar | --calendar | String | &#x2714; | Calendar ID |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Token    = "ya29.a0AeDClZAfxs19DgmwOO8cVD3QKtotHfCNic5iCoPbnD8IzxjvXSUVVyCXTuWw9t0RG7seizlNqpNWBH7sHrQAyS0SeFgQ6chxYmOYUMDNc...";
    Calendar = "06db13e5cd499d76e185d0b904920e3726060189a2a1d09614ccd8c23550829c@group.calendar.google.com";

    Result = OPI_GoogleCalendar.GetListCalendar(Token, Calendar);
```
