# Get list calendar
 Gets a calendar from the user's list by ID



`Function GetListCalendar(Val Token, Val Calendar) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Calendar | --calendar | String | &#x2714; | Calendar ID |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Token    = "ya29.a0AZYkNZjSeF77YyjGa9DbxFuZPdPjS9UMNDftvPRx3_jSDSQFViqbsfHMbuUUWr-kFLPIANJ1NfuC59lZrwgGClVK1ae8HNnsa5lMjY32Y...";
    Calendar = "0750432b0f7fff1e2cd33160de79654c00233caf1b3646b23b12356452addb1b@group.calendar.google.com";

    Result = OPI_GoogleCalendar.GetListCalendar(Token, Calendar);
```
