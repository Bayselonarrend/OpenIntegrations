# Get list calendar
 Gets a calendar from the user's list by ID



`Function GetListCalendar(Val Token, Val Calendar) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Calendar | --calendar | String | &#x2714; | Calendar ID |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Token    = "ya29.a0ARW5m74eRsl1DAAdxlovoi-Df8oZ2JjSlTvfq0WCV2MtxYK3cHV0O7rHGNIuSABPcqh6Ok0GnUZGw1k2_nl4ExSWkjExTaDr0QyABTVfv...";
    Calendar = "8b9841ea8885305f61ef8bbbcf9b729a601eae9401d71bb824d754c9b5fa05ef@group.calendar.google.com";

    Result = OPI_GoogleCalendar.GetListCalendar(Token, Calendar);
```
