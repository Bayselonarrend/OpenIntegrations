# Get list calendar
 Gets a calendar from the user's list by ID



`Function GetListCalendar(Val Token, Val Calendar) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Calendar | --calendar | String | &#x2714; | Calendar ID |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Token    = "ya29.a0AeDClZDbtPTdsMASWLXTAmyOPaFlWRnH_95-IQ3_vaFI6ThHieeCjeEC4cheU6_vsdkpr_o-77O-5uS5GSL-tYacOX9nIGeOh5NyANLDR...";
    Calendar = "2f51e6bbafcd42ce443bf984bd436de3073101ac1e5bcffabbc93a581967e539@group.calendar.google.com";

    Result = OPI_GoogleCalendar.GetListCalendar(Token, Calendar);
```
