# Get list calendar
 Gets a calendar from the user's list by ID



`Function GetListCalendar(Val Token, Val Calendar) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Calendar | --calendar | String | &#x2714; | Calendar ID |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Token    = "ya29.a0AS3H6NwrH32o3TZZmc3e_ByEcGAt2XTmAPcv99GD_9QwcQfnRxKuI-SiaY_8supYx1k4moGVpqwHhRCIfTJiGob3aJdVZrAGlzpNd-xICMBDGVrfB-eNxg-2l...";
    Calendar = "26de68e8bc4f521c9666060f8176578fd33aca850e0250d4e095b555368e529d@group.calendar.google.com";

    Result = OPI_GoogleCalendar.GetListCalendar(Token, Calendar);
```
