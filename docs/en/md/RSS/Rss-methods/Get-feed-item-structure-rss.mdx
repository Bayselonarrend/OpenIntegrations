# Get feed item structure (RSS)
 Gets the structure of parameters for an RSS feed item



`Function GetFeedItemStructureRSS(Val Clear = False, Val AsMap = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Clear | --empty | Boolean | &#x2716; | True > structure with empty valuse, False > field descriptions at values |
  | AsMap | --map | Boolean | &#x2716; | True > returns fields as map |

  
  Returns:  Structure Of KeyAndValue - Feed item fields structure



```bsl title="1C:Enterprise/OneScript code example"
    Result = OPI_RSS.GetFeedItemStructureRSS();
```
