# Parse feed (RSS)
 Parses the XML of an RSS feed and returns a structure with channel data and items



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function ParseFeedRSS(Val XMLText) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | XMLText | --xml | String | &#x2714; | XML content of RSS feed |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 Structure Of KeyAndValue - Channel structure
  </div>
</div>
  
  </TabItem>

```bsl title="1C:Enterprise/OneScript code example"
    FeedXML = "";

    Result = OPI_RSS.ParseFeedRSS(FeedXML);
```
