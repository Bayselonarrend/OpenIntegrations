# Create feed RSS
 Creates RSS feed text



`Function CreateFeedRSS(Val ChannelTitle, Val ChannelDescription, Val ChannelLink, Val Items, Val UpdateDate = Undefined) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | ChannelTitle | --name | String | &#x2714; | Channel title |
  | ChannelDescription | --descr | String | &#x2714; | Channel description |
  | ChannelLink | --link | String | &#x2714; | Channel link |
  | Items | --items | Array of Structure | &#x2714; | Feed items. See GetFeedItemStructure |
  | UpdateDate | --date | Date, Undefined | &#x2716; | Update date. Current if not filled |

  
  Returns:  String - Channel XML feed
