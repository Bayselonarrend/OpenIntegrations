# Remove chat members
 Removes users from the chat



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function RemoveChatMembers(Val Token, Val ChatID, Val Users) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Bot token |
  | ChatID | --chatid | String, Number | &#x2714; | Chat ID |
  | Users | --members | String, Number, Array of String, Number | &#x2714; | The member or members of the chat room to remove |
