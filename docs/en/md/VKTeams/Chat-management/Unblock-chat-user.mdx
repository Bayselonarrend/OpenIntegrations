# Unblock chat user
 Unblocks a previously blocked user in the chat room



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function UnblockChatUser(Val Token, Val ChatID, Val UserID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Bot token |
  | ChatID | --chatid | String, Number | &#x2714; | Chat ID |
  | UserID | --userid | String, Number | &#x2714; | User ID |

  <div className="return-value-note__title">Returns</div>

 Map Of KeyAndValue - serialized JSON response from VK Teams

:::tip
To call this method, the bot must be an administrator in the chat room.

Method at API documentation: [GET ​​/chats/unblockUser](https://teams.vk.com/botapi/#/chats/get_chats_unblockUser)
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    Token  = "001.3501506236.091...";
    ChatID = "689203963@chat.agent";

```
