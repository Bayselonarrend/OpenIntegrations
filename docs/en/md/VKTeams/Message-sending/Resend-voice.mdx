# Resend voice
 Sends a previously uploaded voice message by ID



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function ResendVoice(Val Token, Val ChatID, Val FileID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Bot token |
  | ChatID | --chatid | String, Number | &#x2714; | Chat ID for sending |
  | FileID | --fileid | String, Number | &#x2714; | File ID of voice message |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 Map Of KeyAndValue - serialized JSON response from VK Teams
  </div>
</div>
