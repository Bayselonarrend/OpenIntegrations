# Get images
 Generates images based on the specified description



`Function GetImages(Val URL, Val Token, Val Model, Val Description, Val AdditionalHeaders = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | OpenAI server URL |
  | Token | --token | String | &#x2714; | OpenAI authorization token |
  | Model | --model | String | &#x2714; | Models name |
  | Description | --descr | Structure Of KeyAndValue | &#x2714; | Generation parameters. See GetImageDescriptionStructure |
  | AdditionalHeaders | --headers | Map Of KeyAndValue | &#x2716; | Additional request headers, if necessary |

  
  Returns:  Map Of KeyAndValue - Processing result

<br/>

:::tip
Method at API documentation: [Create image](https://platform.openai.com/docs/api-reference/images/create)
:::
<br/>



```bsl title="1C:Enterprise/OneScript code example"
    URL   = "https://api.vsegpt.ru/";
    Token = "sk-or-vv-c5884ebe9f13fb143194bb07ecb...";

    Model       = "img2img-google/flash-edit";
    Description = OPI_OpenAI.GetImageDescriptionStructure("Yellow alpaca", 2);
    Result      = OPI_OpenAI.GetImages(URL, Token, Model, Description);
```
