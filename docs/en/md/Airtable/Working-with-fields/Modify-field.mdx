# Modify field
 Changes the name and/or description of an existing table field



`Function ModifyField(Val Token, Val Base, Val Table, Val Field, Val Name = "", Val Description = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Base | --base | String | &#x2714; | Base identifier Base |
  | Table | --table | String | &#x2714; | Table identifier |
  | Field | --field | String | &#x2714; | Field identifier |
  | Name | --title | String | &#x2716; | New name |
  | Description | --description | String | &#x2716; | New description |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Airtable



```bsl title="1C:Enterprise/OneScript code example"
    Token = "patNn4BXW66Yx3pdj.5b93c53cab554a8387de02d...";
    Base  = "appUIQ8XPNF3PZhGa";
    Table = "tblvfl4HSemaVTUHp";
    Field = "fldZf5nzeAF4PRtqX";

    Name        = String(New UUID) + "(change.)";
    Description = "New description";

    Result = OPI_Airtable.ModifyField(Token, Base, Table, Field, Name, Description);
```
