# Get list of bases
 Gets the list of available bases



`Function GetListOfBases(Val Token, Val Indent = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Indent | --offset | String | &#x2716; | Next page identifier of the base list from the previous request |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Airtable



```bsl title="1C:Enterprise/OneScript code example"
    Token = "patNn4BXW66Yx3pdj.5b93c53cab554a8387de02d...";

    Options = New Structure;
    Options.Insert("token", Token);

    Result = OPI_TestDataRetrieval.ExecuteTestCLI("airtable", "GetListOfBases", Options);
```
