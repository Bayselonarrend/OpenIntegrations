# Edit main forum topic name
 Edits the name of the main forum topic



`Function EditMainForumTopicName(Val Token, Val ChatID, Val Title) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | ChatID | --forum | String, Number | &#x2714; | Topic chat ID |
  | Title | --title | String | &#x2714; | New main topic name |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Telegram

:::tip
Method at API documentation: [editGeneralForumTopic](https://core.telegram.org/bots/api#editgeneralforumtopic)
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    Title = "New main topic name " + String(New UUID);
    Token = "6129457865:AAFyzNYOAFbu...";
    Chat  = "-1001971186208";

    Result = OPI_Telegram.EditMainForumTopicName(Token, Chat, Title);
```
