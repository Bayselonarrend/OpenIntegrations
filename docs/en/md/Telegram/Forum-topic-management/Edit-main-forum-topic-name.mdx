# Edit main forum topic name
 Edits the name of the main forum topic



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function EditMainForumTopicName(Val Token, Val ChatID, Val Title) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | ChatID | --forum | String, Number | &#x2714; | Topic chat ID |
  | Title | --title | String | &#x2714; | New main topic name |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 Map Of KeyAndValue - serialized JSON response from Telegram
