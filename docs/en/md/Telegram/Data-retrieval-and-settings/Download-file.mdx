# Download file
 Download file from Telegram servers



`Function DownloadFile(Val Token, Val FileID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | FileID | --fileid | String | &#x2714; | File ID for downloading |

  
  Returns:  BinaryData - file's binary data

<br/>

:::tip
Method at API documentation: [getFile](https://core.telegram.org/bots/api#getfile)
:::
<br/>



```bsl title="1C:Enterprise/OneScript code example"
    FileID = "CQACAgIAAx0EcNsaZQACSu9odqzqksR3SiMScst4AZ_Pfwcw3gACFnEAAqvHuUu_w8inAAEgIT82BA";
    Token  = "6129457865:AAFyzNYOAFbu...";

    Result = OPI_Telegram.DownloadFile(Token, FileID);
```
