# Unpin message
 Unpins a message in the chat header



`Function UnpinMessage(Val Token, Val ChatID, Val MessageID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Bot token |
  | ChatID | --chat | String, Number | &#x2714; | Target chat ID |
  | MessageID | --message | String, Number | &#x2714; | Target message ID |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Telegram

:::tip
Method at API documentation: [unpinChatMessage](https://core.telegram.org/bots/api#unpinchatmessage)
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    Token     = "6129457865:AAFyzNYOAFbu...";
    ChannelID = "@testsichee";
    MessageID = "24268";

    Result = OPI_Telegram.UnpinMessage(Token, ChannelID, MessageID);
```
