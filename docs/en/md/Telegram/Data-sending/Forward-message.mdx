# Forward message
 Forwards a message between chats or within a chat



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function ForwardMessage(Val Token, Val OriginalID, Val FromID, Val ToID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Bot token |
  | OriginalID | --message | String, Number | &#x2714; | Original message ID |
  | FromID | --from | String, Number | &#x2714; | Chat ID of the original message |
  | ToID | --to | String, Number | &#x2714; | Target chat ID or ChatID*TopicID |

  
  </TabItem>
  <TabItem value="extended" label={<span>Advanced call{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="About advanced call" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Parameter | Description |
  |---|---|
  | proxy | InternetProxy or a structure with fields Protocol, Host, Port, User, Password, UseOSAuthentication |
  | timeout | Request execution timeout |
  | adv_response | Formats the response as a complete structure with fields code, body, and headers |


  </TabItem>
</Tabs>
  
  Returns:  Map Of KeyAndValue - serialized JSON response from Telegram

:::tip
Method at API documentation: [forwardMessage](https://core.telegram.org/bots/api#forwardmessage)
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    Token     = "6129457865:AAFyzNYOAFbu...";
    ChatID    = "461699897";
    ChannelID = "@testsichee";
    MessageID = "22536";

    Result = OPI_Telegram.ForwardMessage(Token, MessageID, ChannelID, ChatID);
```
