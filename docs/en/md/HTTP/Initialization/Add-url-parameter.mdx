# Add URL parameter 
 Adds a query parameter to the URL in the request



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function AddURLParameter(Val Name, Val Value, Val IgnoreEmpty = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Name | - | String | &#x2714; | Request parameter key |
  | Value | - | Arbitrary | &#x2714; | Request parameter value |
  | IgnoreEmpty | - | Boolean | &#x2716; | Do not add parameter if empty value is passed |
