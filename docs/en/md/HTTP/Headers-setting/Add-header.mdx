# Add header 
 Adds a header to the request header set



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function AddHeader(Val Name, Val Value, Val IgnoreEmpty = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Name | - | String | &#x2714; | Header key |
  | Value | - | String | &#x2714; | Header value |
  | IgnoreEmpty | - | Boolean | &#x2716; | Do not add header if empty value is passed |

  <div className="return-value-note__title">Returns</div>

```bsl title="1C:Enterprise/OneScript code example"
    URL = "https://bin.openintegrations.dev";
    URL = URL + "/get";

    Result = OPI_HTTPRequests.NewRequest()
        .Initialize()
        .SetURL(URL)
        .AddHeader("X-Header1", "Value1") // <---
        .AddHeader("X-Header2", "Value2") // <---
        .ProcessRequest("GET")
        .ReturnResponseAsJSONObject();
```
