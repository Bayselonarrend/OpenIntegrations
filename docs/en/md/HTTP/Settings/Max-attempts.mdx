# Max attempts 
 Sets the maximum number of retry attempts for sending a request in case of 5** codes or client internal errors



`Function MaxAttempts(Val Value) Export`

<Tabs>
  <TabItem value="params" label="Parameters" default>

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Value | - | Number | &#x2714; | Retry count |

  
  </TabItem>
  <TabItem value="extended" label="Advanced call">

*This method has no additional advanced call parameters.*


  </TabItem>
</Tabs>
  
  Returns:  DataProcessorObject.OPI_HTTPClient - This processor object

:::tip
By default: `0`
:::

:::caution
**NOCLI:** this method is not available in CLI version
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    Result = OPI_HTTPRequests.NewRequest()
        .Initialize()
        .MaxAttempts(10)
        .ReturnSettings();
```
