# Send data in parts 
 Sends the body in multiple requests with the Content-Range header



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function SendDataInParts(Val ChunkSize = 5242880, Val Method = "PUT") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | ChunkSize | - | Number | &#x2716; | Size of one part for sending |
  | Method | - | String | &#x2716; | Request HTTP method |

```bsl title="1C:Enterprise/OneScript code example"
    URL = "https://bin.openintegrations.dev";
    URL = URL + "/put";

    ChunkSize = 524288;
    Image     = "https://hut.openintegrations.dev/test_data/picture.jpg"; // URL, Path or Binary Data

    Result = OPI_HTTPRequests.NewRequest()
        .Initialize(URL)
        .SetBinaryBody(Image)
        .SendDataInParts(ChunkSize) // <---
        .ReturnResponseAsJSONObject();
```
