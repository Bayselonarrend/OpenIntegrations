# Return connection 
 Returns the object of the current HTTP connection



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function ReturnConnection(Forced = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Forced | - | Boolean | &#x2716; | False > The processor object will be returned instead of the connection if there were errors in it |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 DataProcessorObject.OPI_HTTPClient, HTTPConnection, Undefined - Connection or the same processor object
  </div>
</div>

    URL = "https://bin.openintegrations.dev";
