# Get settings (keyboard-interactive))
 Retrieves connection settings for keyboard-interactive authentication



`Function GetSettingsKI(Val Host, Val Port, Val Login, Val InputOrder) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Host | --host | String | &#x2714; | SSH host |
  | Port | --port | Number | &#x2714; | SSH port |
  | Login | --user | String | &#x2714; | SSH username |
  | InputOrder | --prompts | Array Of String | &#x2714; | Array of responses to server questions |

  
  Returns:  Structure Of KeyAndValue - Connection configuration
