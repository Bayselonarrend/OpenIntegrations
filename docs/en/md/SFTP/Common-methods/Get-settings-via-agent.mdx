# Get settings (via agent)
 Gets connection settings with SSH Agent authentication



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetSettingsViaAgent(Val Host, Val Port, Val Login) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Host | --host | String | &#x2714; | SSH host |
  | Port | --port | Number | &#x2714; | SSH port |
  | Login | --user | String | &#x2714; | SSH username |

  
  </TabItem>
  <TabItem value="extended" label={<span>Advanced call{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="About advanced call" onClick={(e) => e.stopPropagation()}>?</a></span>}>

*This method has no additional advanced call parameters.*


  </TabItem>
</Tabs>
  
  Returns:  Structure Of KeyAndValue - Connection configuration



```bsl title="1C:Enterprise/OneScript code example"
    Host   = "172.33.0.13";
    Port   = "2222";
    Login  = "bayselonarrend";
    Result = OPI_SFTP.GetSettingsViaAgent(Host, Port, Login);
```
