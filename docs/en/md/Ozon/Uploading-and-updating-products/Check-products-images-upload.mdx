# Check products images upload
 Check images upload by products IDs



`Function CheckProductsImagesUpload(Val ClientID, Val APIKey, Val ProductsID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | ClientID | --clientid | String | &#x2714; | Client identifier |
  | APIKey | --apikey | String | &#x2714; | API key |
  | ProductsID | --products | String, Number, Array of String, Number | &#x2714; | Product IDs to be inspected |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Ozon Seller API

<br/>

:::tip
Method at API documentation: [post /v2/product/pictures/info](https://docs.ozon.ru/api/seller/#operation/ProductAPI_ProductInfoPictures)
:::
<br/>



```bsl title="1C:Enterprise/OneScript code example"
    ClientID  = "2479669";
    APIKey    = "09f65e9f-262d-4aca...";
    ProductID = "2710250299";

    Result = OPI_Ozon.CheckProductsImagesUpload(ClientID, APIKey, ProductID);
```
