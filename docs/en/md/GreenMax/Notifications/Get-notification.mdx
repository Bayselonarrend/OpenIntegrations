# Get notification
 Gets the latest notification from the queue



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetNotification(Val AccessParameters, Val Timeout = 5, Val Delete = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | AccessParameters | --access | Structure Of KeyAndValue | &#x2714; | Access parameters. See FormAccessParameters |
  | Timeout | --timeout | Number | &#x2716; | Timeout for waiting for a notification if the queue is empty |
  | Delete | --del | Boolean | &#x2716; | Delete notification from the queue after retrieval |

    Result           = OPI_GreenMax.GetNotification(AccessParameters);
