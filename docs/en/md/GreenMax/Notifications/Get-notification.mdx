# Get notification
 Gets the latest notification from the queue



`Function GetNotification(Val AccessParameters, Val Timeout = 5, Val Delete = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | AccessParameters | --access | Structure Of KeyAndValue | &#x2714; | Access parameters. See FormAccessParameters |
  | Timeout | --timeout | Number | &#x2716; | Timeout for waiting for a notification if the queue is empty |
  | Delete | --del | Boolean | &#x2716; | Delete notification from the queue after retrieval |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Green API

:::tip
To get the next notification, the current notification must be deleted

Method at API documentation: [ReceiveNotification](https://green-api.com/v3/docs/api/receiving/technology-http-api/ReceiveNotification/)
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    ApiUrl           = "https://3100.api.green-api.com/v3";
    MediaUrl         = "https://3100.api.green-api.com/v3";
    IdInstance       = "31003...";
    ApiTokenInstance = "17915d8e0b4b4975a183478da...";

    AccessParameters = OPI_GreenMax.FormAccessParameters(ApiUrl, MediaUrl, IdInstance, ApiTokenInstance);
    Result           = OPI_GreenMax.GetNotification(AccessParameters);
```
