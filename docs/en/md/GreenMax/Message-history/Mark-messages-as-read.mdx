# Mark messages as read
 Marks messages in the chat as read



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function MarkMessagesAsRead(Val AccessParameters, Val ChatID, Val MessageID = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | AccessParameters | --access | Structure Of KeyAndValue | &#x2714; | Access parameters. See FormAccessParameters |
  | ChatID | --chat | String | &#x2714; | Chat ID |
  | MessageID | --message | String | &#x2716; | Message ID. All messages, unless specified |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 Map Of KeyAndValue - serialized JSON response from Green API
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Advanced call{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="About advanced call" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Parameter | Description |
  |---|---|
  | proxy | InternetProxy or a structure with fields Protocol, Host, Port, User, Password, UseOSAuthentication |
  | timeout | Request execution timeout |
  | adv_response | Formats the response as a complete structure with fields code, body, and headers |
  | retries | Number of send attempts on `5**` status codes or internal client errors |
  | dontwait | Creates a background job and returns its object (for 1C and OneScript only) |


  </TabItem>
</Tabs>

:::tip
When specifying the message ID, all messages after the specified one will be marked as read

Method at API documentation: [ReadChat](https://green-api.com/v3/docs/api/marks/ReadChat/)
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    ApiUrl           = "https://3100.api.green-api.com/v3";
    MediaUrl         = "https://3100.api.green-api.com/v3";
    IdInstance       = "31003...";
    ApiTokenInstance = "17915d8e0b4b4975a183478da...";

    ChatID = 12345678;

    AccessParameters = OPI_GreenMax.FormAccessParameters(ApiUrl, MediaUrl, IdInstance, ApiTokenInstance);
    Result           = OPI_GreenMax.MarkMessagesAsRead(AccessParameters, ChatID);
```
