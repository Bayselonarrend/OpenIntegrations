# Get contact list
 Gets account contact list



`Function GetContactList(Val AccessParameters, Val Count = Undefined) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | AccessParameters | --access | Structure Of KeyAndValue | &#x2714; | Access parameters. See FormAccessParameters |
  | Count | --count | Number | &#x2716; | Contact count to retrieve. All if not specified |

  
  Returns:  Array Of Map - serialized JSON response from Green API

<br/>

:::tip
Method at API documentation: [GetContacts](https://green-api.com/v3/docs/api/service/GetContacts/)
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    ApiUrl           = "https://3100.api.green-api.com/v3";
    MediaUrl         = "https://3100.api.green-api.com/v3";
    IdInstance       = "31003...";
    ApiTokenInstance = "17915d8e0b4b4975a183478da...";

    Count = 1;

    AccessParameters = OPI_GreenMax.FormAccessParameters(ApiUrl, MediaUrl, IdInstance, ApiTokenInstance);
    Result           = OPI_GreenMax.GetContactList(AccessParameters, Count);
```
