# Get contact information
 Gets information about the selected contact



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetContactInformation(Val AccessParameters, Val ChatID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | AccessParameters | --access | Structure Of KeyAndValue | &#x2714; | Access parameters. See FormAccessParameters |
  | ChatID | --chat | String | &#x2714; | Chat identifier |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>

```bsl title="1C:Enterprise/OneScript code example"
    ApiUrl           = "https://3100.api.green-api.com/v3";
    MediaUrl         = "https://3100.api.green-api.com/v3";
    IdInstance       = "31003...";
    ApiTokenInstance = "17915d8e0b4b4975a183478da...";

    ContactID = 87654321;
    AccessParameters = OPI_GreenMax.FormAccessParameters(ApiUrl, MediaUrl, IdInstance, ApiTokenInstance);
    Result           = OPI_GreenMax.GetContactInformation(AccessParameters, ContactID);
```
