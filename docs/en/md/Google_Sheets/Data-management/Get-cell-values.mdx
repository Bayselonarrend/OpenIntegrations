# Get cell values
 Gets cell values of the table



`Function GetCellValues(Val Token, Val Spreadsheet, Val CellsArray = "", Val Sheet = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Spreadsheet | --spreadsheet | String | &#x2714; | SpreadsheetID |
  | CellsArray | --cells | Array of String | &#x2716; | Array of A1 type cells to get (whole sheet if not filled) |
  | Sheet | --sheetname | String | &#x2716; | Sheet name (first sheet by default) |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google

<br/>

    Spreadsheet = "1mPXsdn2dQXSGxubILyd2DWJlqqaF6t3m4AlmgWsHOP4";
