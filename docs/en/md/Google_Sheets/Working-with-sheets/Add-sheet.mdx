# Add sheet
 Adds a new sheet to the spreadsheet



`Function AddSheet(Val Token, Val Spreadsheet, Val Name) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Spreadsheet | --spreadsheet | String | &#x2714; | Spreadsheet identifier |
  | Name | --title | String | &#x2714; | NewSheetName |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google

<br/>

```bsl title="1C:Enterprise/OneScript code example"
    Result = OPI_GoogleSheets.AddSheet(Token, Spreadsheet, Name);
```
