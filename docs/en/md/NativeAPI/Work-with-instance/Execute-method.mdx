# Invoke method
 Calls the instance method of the AddIn by name or number



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function ExecuteMethod(Val Host, Val Instance, Val Method, Val Parameters = Undefined, Val Properties = Undefined) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Host | --host | String, Arbitrary | &#x2714; | Host object or path to the AddIn. See CreateHost |
  | Instance | --class | String | &#x2714; | Instance UUID or class name for temporary instance |
  | Method | --method | String | &#x2714; | Method name |
  | Parameters | --params | Array Of Arbitrary | &#x2716; | Parameters of the called function |
  | Properties | --props | Map Of KeyAndValue | &#x2716; | Values for setting instance properties before calling |


<div className="return-value-note">
