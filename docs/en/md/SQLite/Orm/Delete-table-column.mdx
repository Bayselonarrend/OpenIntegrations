# Delete table column
 Deletes a column from the table



`Function DeleteTableColumn(Val Table, Val Name, Val Connection = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Table | --table | String | &#x2714; | Table name |
  | Name | --name | String | &#x2714; | Column name |
  | Connection | --db | String, Arbitrary | &#x2716; | Existing connection or database path |

  
  Returns:  Map Of KeyAndValue - Result of query execution

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Base  = "C:\Users\Administrator\AppData\Local\Temp\v8_6002_332.sqlite";
    Table = "test";
    Name  = "new_col";

    Result = OPI_SQLite.DeleteTableColumn(Table, Name, Base);
```
