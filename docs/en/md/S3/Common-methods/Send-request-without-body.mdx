# Send request without body
 Sends a simple http request without a body



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function SendRequestWithoutBody(Val Method, Val BasicData, Val ExpectedBinary = False, Val Headers = Undefined) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Method | --method | String | &#x2714; | HTTP method |
  | BasicData | --basic | Structure Of KeyAndValue | &#x2714; | Basic request data (with full URL). See GetBasicDataStructure |
  | ExpectedBinary | --binary | Boolean | &#x2716; | Disables an attempt to convert the response to JSON |
  | Headers | --headers | Map Of KeyAndValue | &#x2716; | Additional request headers, if necessary |

```bsl title="1C:Enterprise/OneScript code example"
    URL       = "s3.openintegrations.dev" + "/opi-newbucket2";
    AccessKey = "bayselo...";
    SecretKey = "12we34...";
    Region    = "BTC";

    BasicData = OPI_S3.GetBasicDataStructure(URL, AccessKey, SecretKey, Region);
    Method    = "GET";

    Result = OPI_S3.SendRequestWithoutBody(Method, BasicData);
```
