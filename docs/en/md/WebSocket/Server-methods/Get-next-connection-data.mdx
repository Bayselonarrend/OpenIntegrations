# Get next connection data 
 Gets data from the buffer of the next active connection in the queue where there are incoming data



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetNextConnectionData(Val ServerObject, Val Timeout = 1000) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | ServerObject | - | Arbitrary | &#x2714; | Object of running server component |
  | Timeout | - | Number | &#x2716; | Waiting period for new data if the queue is empty (in ms) |
  | MaxSize | - | Number | &#x2714; | Maximum size of data to receive |
