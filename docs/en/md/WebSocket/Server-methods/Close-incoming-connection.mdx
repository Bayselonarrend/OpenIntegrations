# Close incoming connection 
 Closes incoming server connection by ID



`Function CloseIncomingConnection(Val ServerObject, Val ConnectionID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | ServerObject | - | Arbitrary | &#x2714; | Object of running server component |
  | ConnectionID | - | String | &#x2714; | Connection identifier |

  
  Returns:  Map Of KeyAndValue - Execution result


:::caution
**NOCLI:** this method is not available in CLI version
:::
