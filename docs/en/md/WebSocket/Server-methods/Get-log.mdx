# Get log 
 Retrieves server log data (with logging to memory enabled)



`Function GetLog(Val ServerObject, Val AsString = False, Val EventCount = 100) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | ServerObject | - | Arbitrary | &#x2714; | Object of running server component |
  | AsString | - | Boolean | &#x2716; | True > returns log as a single string, False > as an array |
  | EventCount | - | Number | &#x2716; | Number of recent events to retrieve. 0 > no limits |

  
  Returns:  String, Map Of KeyAndValue - Log as a string or a map with the full execution result


:::caution
**NOCLI:** this method is not available in CLI version
:::
