# Send image
 Sends an image to a chat or channel



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function SendImage(Val Token, Val URL, Val UserID, Val SendingToChannel, Val Description = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | URL | --picture | String | &#x2714; | Image URL |
  | UserID | --user | String, Number | &#x2714; | User ID. For channel > administrator, for bot > recipient |
  | SendingToChannel | --ischannel | boolean | &#x2714; | Sending to channel or bot chat |
  | Description | --description | String | &#x2716; | Image annotation |
