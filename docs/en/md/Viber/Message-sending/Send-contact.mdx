# Send contact
 Sends a contact with a phone number to a chat or channel



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function SendContact(Val Token, Val ContactName, Val PhoneNumber, Val UserID, Val SendingToChannel) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | ContactName | --name | String | &#x2714; | Contact name |
  | PhoneNumber | --phone | String | &#x2714; | Phone number |
  | UserID | --user | String, Number | &#x2714; | User ID. For channel > administrator, for bot > recipient |
  | SendingToChannel | --ischannel | Boolean | &#x2714; | Sending to channel or bot chat |
