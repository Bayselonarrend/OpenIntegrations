# Get delivery cash transfers
 Gets information about orders for which cash on delivery was transferred to the online store



`Function GetDeliveryCashTransfers(Val Token, Val ReceivingDate, Val TestAPI = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Auth token |
  | ReceivingDate | --date | Date | &#x2714; | Date for which data should be retrieved |
  | TestAPI | --testapi | Boolean | &#x2716; | Flag to use test API for requests |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from CDEK

<br/>

:::tip
Method at API documentation: [Cash on delivery transfer information](https://api-docs.cdek.ru/71060213.html)
:::
<br/>



```bsl title="1C:Enterprise/OneScript code example"
    Token         = FunctionParameters["CDEK_Token"];
    ReceivingDate = OPI_Tools.GetCurrentDate();

    Result = OPI_CDEK.GetDeliveryCashTransfers(Token, ReceivingDate, True);
```
