# Get order by number
 Receives the order by CDEK number or number from customer IB



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetOrderByNumber(Val Token, Val OrderNumber, Val Internal = False, Val TestAPI = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Auth token |
  | OrderNumber | --number | String | &#x2714; | Order number |
  | Internal | --internal | Boolean | &#x2716; | Type of order number. True > number in the customer base, False > CDEK number |
  | TestAPI | --testapi | Boolean | &#x2716; | Flag to use test API for requests |
