# Create prealert
 Creates a prealert for the order list



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function CreatePrealert(Val Token, Val UUIDArray, Val TransferDate, Val Point, Val TestAPI = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Auth token |
  | UUIDArray | --uuids | String, Array of String | &#x2714; | One or an array of order UUIDs |
  | TransferDate | --date | Date | &#x2714; | Planned date of transfer of orders to CDEK |
  | Point | --point | String | &#x2714; | Code of shipment point to which it is planned to transfer orders |
  | TestAPI | --testapi | Boolean | &#x2716; | Flag to use test API for requests |
