# Delete courier invitation




`Function DeleteCourierInvitation(Val Token, Val UUID, Val TestAPI = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Auth token |
  | UUID | --uuid | String | &#x2714; | UUID of the invitation for deletion |
  | TestAPI | --testapi | Boolean | &#x2716; | Flag to use test API for requests |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from CDEK

<br/>

:::tip
Method at API documentation: [Deleting a courier invitation](https://api-docs.cdek.ru/33828880.html)
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    Token = FunctionParameters["CDEK_Token"];
    UUID  = FunctionParameters["CDEK_IntakeUUID"];

    Result = OPI_CDEK.DeleteCourierInvitation(Token, UUID, True);
```
