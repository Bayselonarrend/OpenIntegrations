# Get delivery appointment
 Receives information about the delivery date, time and address agreed with the customer



`Function GetDeliveryAppointment(Val Token, Val UUID, Val TestAPI = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Auth token |
  | UUID | --uuid | String | &#x2714; | Appointment UUID |
  | TestAPI | --testapi | Boolean | &#x2716; | Flag to use test API for requests |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from CDEK

:::tip
The appointment must be previously created. See RegisterDeliveryAppointment

Method at API documentation: [Information about delivery appointment](https://api-docs.cdek.ru/36989601.html)
:::
