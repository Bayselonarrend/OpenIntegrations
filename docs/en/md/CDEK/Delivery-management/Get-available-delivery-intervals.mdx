# Get available delivery intervals
 Gets available delivery intervals for order



`Function GetAvailableDeliveryIntervals(Val Token, Val UUID, Val TestAPI = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Auth token |
  | UUID | --uuid | String | &#x2714; | Order UUID |
  | TestAPI | --testapi | Boolean | &#x2716; | Flag to use test API for requests |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from CDEK

<br/>

:::tip
Method at API documentation: [Getting delivery intervals (RU)](https://api-docs.cdek.ru/154160502.html)
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    Token = FunctionParameters["CDEK_Token"];
    UUID  = FunctionParameters["CDEK_OrderUUID"];

    Result = OPI_CDEK.GetAvailableDeliveryIntervals(Token, UUID, True);
```
