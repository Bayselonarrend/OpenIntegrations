# Get actions list
 Gets the list of actions of the selected counter



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetActionsList(Val Token, Val CounterID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Auth token |
  | CounterID | --counter | String, Number | &#x2714; | Counter ID |

  
  </TabItem>
  <TabItem value="extended" label={<span>Advanced call{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="About advanced call" onClick={(e) => e.stopPropagation()}>?</a></span>}>

*This method has no additional advanced call parameters.*


  </TabItem>
</Tabs>
  
  Returns:  Map Of KeyAndValue - serialized JSON response from Yandex

:::tip
Method at API documentation: [List of actions](https://yandex.ru/dev/metrika/en/management/openapi/operation/operations)
:::

```bsl title="1C:Enterprise/OneScript code example"
    Token     = "y0_AgAAAABdylaOAAy9KgAAAAEXh6i...";
    CounterID = "105009635";

    Result = OPI_YandexMetrika.GetActionsList(Token, CounterID);
```
