# Get counter
 Gets information about the counter by ID



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetCounter(Val Token, Val CounterID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Auth token |
  | CounterID | --counter | String, Number | &#x2714; | Counter ID |
