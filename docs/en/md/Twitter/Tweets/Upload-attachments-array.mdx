# Upload attachments array
 Uploads files to the server and returns their IDs



`Function UploadAttachmentsArray(Val ArrayOfFiles, Val AttachmentsType, Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | ArrayOfFiles | --files | Array of String, BinaryData | &#x2714; | Array of files to be uploaded |
  | AttachmentsType | --type | String | &#x2714; | Attachment type: tweet_video, tweet_image, tweet_gif |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization data. See GetStandardParameters |

  
  Returns:  Array Of String - Media ID array

<br/>

:::tip
Parameters with Binary data type can also accept file paths on disk and URLs
:::
<br/>



```bsl title="1C:Enterprise/OneScript code example"
    Parameters = GetTwitterAuthData();

    Image1 = "https://hut.openintegrations.dev/test_data/picture.jpg"; // URL, Binary Data or File path
    Image2 = "https://hut.openintegrations.dev/test_data/picture2.jpg"; // URL, Binary Data or File path

    ImageArray = New Array;
    ImageArray.Add(Image1);
    ImageArray.Add(Image2);

    Result = OPI_Twitter.UploadAttachmentsArray(ImageArray, "tweet_image", Parameters);
```
