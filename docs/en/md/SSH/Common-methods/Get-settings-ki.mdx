# Get settings (keyboard-interactive))
 Retrieves connection settings for keyboard-interactive authentication



`Function GetSettingsKI(Val Host, Val Port, Val Login, Val InputOrder) Export`

<Tabs>
  <TabItem value="params" label="Parameters" default>

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Host | --host | String | &#x2714; | SSH host |
  | Port | --port | Number | &#x2714; | SSH port |
  | Login | --user | String | &#x2714; | SSH username |
  | InputOrder | --prompts | Array Of String | &#x2714; | Array of responses to server questions |

  
  </TabItem>
  <TabItem value="extended" label="Advanced call">

*This method has no additional advanced call parameters.*


  </TabItem>
</Tabs>
  
  Returns:  Structure Of KeyAndValue - Connection configuration



```bsl title="1C:Enterprise/OneScript code example"
    Host     = "172.33.0.13";
    Port     = "2222";
    Login    = "bayselonarrend";
    Password = "12we...";

    AnswersArray = New Array;
    AnswersArray.Add("yes");
    AnswersArray.Add(Password);

    Result = OPI_SSH.GetSettingsKI(Host, Port, Login, AnswersArray);
```
