# Return child blocks
 Returns list of child blocks of parent block



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function ReturnChildBlocks(Val Token, Val BlockID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | BlockID | --block | String | &#x2714; | Parent block ID |
