# Mark message as unreaded
 Mark current and all messages after as unreaded



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function MarkMessageAsUnreaded(Val URL, Val ChatID, Val MessageID, Val Token = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | URL of webhook or a Bitrix24 domain, when token used |
  | ChatID | --chat | String, Number | &#x2714; | Chat ID (as chatXXX) or User ID (as XXX) |
  | MessageID | --message | String, Number | &#x2714; | ID of last unreaded message |
  | Token | --token | String | &#x2716; | Access token, when app auth method used |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>

```bsl title="1C:Enterprise/OneScript code example"
    URL    = "https://b24-ar17wx.bitrix24.by/rest/1/av5...";
    ChatID = "3390";

    OPI_TypeConversion.GetLine(ChatID);

    MessageID  = "67534";

    Result = OPI_Bitrix24.MarkMessageAsUnreaded(URL, ChatString, MessageID);

    URL       = "b24-ar17wx.bitrix24.by";

```
