# Delete task time accounting
 Deletes record of time accounting



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function DeleteTaskTimeAccounting(Val URL, Val TaskID, Val RecordID, Val Token = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | URL of webhook or a Bitrix24 domain, when token used |
  | TaskID | --task | String, Number | &#x2714; | ID of task for time accounting |
  | RecordID | --record | String, Number | &#x2714; | Time record ID |
  | Token | --token | String | &#x2716; | Access token, when app auth method used |
