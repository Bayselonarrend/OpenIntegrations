# Add task to favorites list
 Add task to favorites list



`Function AddTaskToFavorites(Val URL, Val TaskID, Val Token = "") Export`

<Tabs>
  <TabItem value="params" label="Parameters" default>

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | URL of webhook or a Bitrix24 domain, when token used |
  | TaskID | --task | Number, String | &#x2714; | Task ID |
  | Token | --token | String | &#x2716; | Access token, when app auth method used |

  
  </TabItem>
  <TabItem value="extended" label="Advanced call">

*This method has no additional advanced call parameters.*


  </TabItem>
</Tabs>
  
  Returns:  Map Of KeyAndValue - serialized JSON of answer from Bitrix24 API

:::tip
Method at API documentation: [tasks.task.favorite.add](https://dev.1c-bitrix.ru/rest_help/tasks/task/tasks/tasks_task_favorite_add.php)
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    URL    = "https://b24-ar17wx.bitrix24.by/rest/1/av5...";
    TaskID = "8718";

    Result = OPI_Bitrix24.AddTaskToFavorites(URL, TaskID);

    URL    = "b24-ar17wx.bitrix24.by";
    Token  = "a5b00469006e9f06006b12e400000001000...";
    TaskID = "8720";

    Result = OPI_Bitrix24.AddTaskToFavorites(URL, TaskID, Token);
```
