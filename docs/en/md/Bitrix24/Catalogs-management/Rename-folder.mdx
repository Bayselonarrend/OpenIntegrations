# Rename folder
 Change folder name



`Function RenameFolder(Val URL, Val FolderID, Val Name, Val Token = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | URL of webhook or a Bitrix24 domain, when token used |
  | FolderID | --folderid | String, Number | &#x2714; | Folder identifier |
  | Name | --title | String | &#x2714; | New folders name |
  | Token | --token | String | &#x2716; | Access token, when app auth method used |

  
  Returns:  Map Of KeyAndValue - serialized JSON of answer from Bitrix24 API

<br/>

:::tip
Method at API documentation: [disk.folder.rename](https://dev.1c-bitrix.ru/rest_help/disk/folder/disk_folder_rename.php)
:::
<br/>



```bsl title="1C:Enterprise/OneScript code example"
    Name      = "New catalog";
    Filename2 = "New folder 2";

    FolderID2 = "42172";
    URL       = "https://b24-ar17wx.bitrix24.by/rest/1/h0m...";

    Result = OPI_Bitrix24.RenameFolder(URL, FolderID2, Filename2);

    FolderID = "42174";
    URL      = "b24-ar17wx.bitrix24.by";
    Token    = "75d8c668006e9f06006b12e400000001000...";

    Result = OPI_Bitrix24.RenameFolder(URL, FolderID, Name, Token);
```
