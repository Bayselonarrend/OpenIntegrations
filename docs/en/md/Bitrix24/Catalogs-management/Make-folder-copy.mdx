# Copy folder
 Copy one folder to another



`Function MakeFolderCopy(Val URL, Val FolderID, Val DestinationID, Val Token = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | URL of webhook or a Bitrix24 domain, when token used |
  | FolderID | --folderid | String, Number | &#x2714; | Folder identifier |
  | DestinationID | --tagetid | String, Number | &#x2714; | ID of target folder |
  | Token | --token | String | &#x2716; | Access token, when app auth method used |

  
  Returns:  Map Of KeyAndValue - serialized JSON of answer from Bitrix24 API

<br/>

:::tip
Method at API documentation: [disk.folder.copyto](https://dev.1c-bitrix.ru/rest_help/disk/folder/disk_folder_copyto.php)
:::
<br/>



```bsl title="1C:Enterprise/OneScript code example"
    DestinationID = "35622";
    FolderID      = "35624";

    URL = "https://b24-ar17wx.bitrix24.by/rest/1/h0m...";

    Result = OPI_Bitrix24.MakeFolderCopy(URL, FolderID, DestinationID);

    DestinationID = "35624";
    FolderID      = "35622";

    URL   = "b24-ar17wx.bitrix24.by";
    Token = "11ae5d68006e9f06006b12e400000001000...";

    Result = OPI_Bitrix24.MakeFolderCopy(URL, FolderID, DestinationID, Token);

    OPI_TestDataRetrieval.Check_BitrixObject(Result);
```
