# Refresh token
 Update token by refresh token



`Function RefreshToken(Val ClientID, Val ClientSecret, Val Refresh) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | ClientID | --clientid | String | &#x2714; | Client ID from app settings |
  | ClientSecret | --clientsecret | String | &#x2714; | Client secret from app settings |
  | Refresh | --refresh | String | &#x2714; | Refresh token |

  
  Returns:  Map Of KeyAndValue - serialized JSON of answer from Bitrix24 API



```bsl title="1C:Enterprise/OneScript code example"
    ClientID     = "local.6667fc928a50a9.70414732";
    ClientSecret = "ZeKyeYIgy2NsHZqsIHY6GfG1V...";
    Refresh      = "aa7d8469006e9f06006b12e400000001000...";

    Result = OPI_Bitrix24.RefreshToken(ClientID, ClientSecret, Refresh);
```
