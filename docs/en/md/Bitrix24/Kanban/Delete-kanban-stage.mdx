# ID of the stage to be deleted
 Removes a kanban (My Plan) stage, provided there are no tasks in it



<Tabs>

`Function DeleteKanbanStage(Val URL, Val StageID, Val AsAdmin = False, Val Token = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | URL of webhook or a Bitrix24 domain, when token used |
  | StageID | --stage | String, Number | &#x2714; | ID of stage to dele |
  | AsAdmin | --admin | Boolean | &#x2716; | Allows you to add stages without checking permissions (for administrators) |
  | Token | --token | String | &#x2716; | Access token, when app auth method used |
