# Get posts
 Gen post or array of post with ID or rights selection



`Function GetPosts(Val URL, Val PostID = "", Val Filter = "UA", Val Token = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | URL of webhook or a Bitrix24 domain, when token used |
  | PostID | --postid | String, Number | &#x2716; | Id of important post |
  | Filter | --sel | String | &#x2716; | Post selection by rights: UA all, SGn work group, Un user, DRn depart, Gn group |
  | Token | --token | String | &#x2716; | Access token, when app auth method used |

  
  Returns:  Map Of KeyAndValue - serialized JSON of answer from Bitrix24 API

<br/>

:::tip
Method at API documentation: [log.blogpost.get](https://dev.1c-bitrix.ru/rest_help/log/log_blogpost_get.php)
:::
<br/>



```bsl title="1C:Enterprise/OneScript code example"
    PostID = "2376";
    URL    = "https://b24-ar17wx.bitrix24.by/rest/1/h0m...";

    Result = OPI_Bitrix24.GetPosts(URL, PostID);

    URL   = "b24-ar17wx.bitrix24.by";
    Token = "4dee2c68006e9f06006b12e400000001000...";

    Result = OPI_Bitrix24.GetPosts(URL, PostID, , Token);
```
