# Add new recipients to a post
 Adds new groups or users to the recipients



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function AddPostRecipients(Val URL, Val PostID, Val Visibility, Val Token = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | URL of webhook or a Bitrix24 domain, when token used |
  | PostID | --postid | String, Number | &#x2714; | Post ID |
  | Visibility | --vision | String | &#x2714; | Array or 1 recipient: UA all, SGn w. group, Un user, DRn department, Gn group |
  | Token | --token | String | &#x2716; | Access token, when app auth method used |

```bsl title="1C:Enterprise/OneScript code example"
    Visibility = "UA";
    PostID     = "3326";
    URL        = "https://b24-ar17wx.bitrix24.by/rest/1/av5...";

    Result = OPI_Bitrix24.AddPostRecipients(URL, PostID, Visibility);

    URL   = "b24-ar17wx.bitrix24.by";
    Token = "a5b00469006e9f06006b12e400000001000...";

    Result = OPI_Bitrix24.AddPostRecipients(URL, PostID, Visibility, Token);
```
