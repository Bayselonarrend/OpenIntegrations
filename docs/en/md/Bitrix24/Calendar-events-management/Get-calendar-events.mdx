# Get calendar events
 Gets events of calendars with or without filters



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetCalendarEvents(Val URL, Val OwnerID, Val Type, Val Filter = "", Val Token = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | URL of webhook or a Bitrix24 domain, when token used |
  | OwnerID | --owner | String, Number | &#x2714; | Calendar owner ID |
  | Type | --type | String | &#x2714; | Calendar type: user, group, company_calendar |
  | Filter | --filter | Structure Of KeyAndValue | &#x2716; | Events filter. See GetCalendarEventsFilterStructure |
  | Token | --token | String | &#x2716; | Access token, when app auth method used |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 Map Of KeyAndValue - serialized JSON of answer from Bitrix24 API
  </div>
</div>
  
  </TabItem>

  | Parameter | Description |
  |---|---|

  | timeout | Request execution timeout |

```bsl title="1C:Enterprise/OneScript code example"
    URL     = "https://b24-ar17wx.bitrix24.by/rest/1/av5...";
    OwnerID = 1;
    Type    = "user";

    Result = OPI_Bitrix24.GetCalendarEvents(URL, OwnerID, Type);

    URL         = "b24-ar17wx.bitrix24.by";
    Token       = "a5b00469006e9f06006b12e400000001000...";
    EventID     = "3046";
    CalendarID1 = "1648";
    CalendarID2 = "1650";

    Tomorrow = OPI_Tools.GetCurrentDate() + 86400;
    NextDay  = Tomorrow + 86400;

    ArrayOfCalendars = New Array;
    ArrayOfCalendars.Add(CalendarID1);
    ArrayOfCalendars.Add(CalendarID2);

    Filter = New Structure;
    Filter.Insert("from"   , Tomorrow);
    Filter.Insert("to"     , NextDay);
    Filter.Insert("section", ArrayOfCalendars);

    Result = OPI_Bitrix24.GetCalendarEvents(URL, OwnerID, Type, Filter, Token);
```
