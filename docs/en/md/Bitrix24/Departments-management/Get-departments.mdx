# Get departments
 Gets a list of departments by ID or filter



`Function GetDepartments(Val URL, Val DepartmentID = "", Val Name = "", Val ParentID = "", Val HeadID = "", Val Token = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | URL of webhook or a Bitrix24 domain, when token used |
  | DepartmentID | --depid | String, Number | &#x2716; | Selection by department ID |
  | Name | --title | String | &#x2716; | Filter by name |
  | ParentID | --parentid | String, Number | &#x2716; | Filter by parent ID |
  | HeadID | --headid | String, Number | &#x2716; | Filter by manager ID |
  | Token | --token | String | &#x2716; | Access token, when app auth method used |

  
  Returns:  Map Of KeyAndValue - serialized JSON of answer from Bitrix24 API

:::tip
Method at API documentation: [department.get](https://dev.1c-bitrix.ru/rest_help/departments/department_get.php)
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    URL          = "https://b24-ar17wx.bitrix24.by/rest/1/h0m...";
    DepartmentID = "3214";

    Result = OPI_Bitrix24.GetDepartments(URL, DepartmentID);

    URL    = "b24-ar17wx.bitrix24.by";
    Token  = "bafe5c69006e9f06006b12e400000001000...";
    HeadID = 1;

    Result = OPI_Bitrix24.GetDepartments(URL, , , , HeadID, Token);
```
