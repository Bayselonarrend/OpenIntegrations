# Add comment to task
 Create new comment to the task



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function AddTaskComment(Val URL, Val TaskID, Val Text, Val Token = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | URL of webhook or a Bitrix24 domain, when token used |
  | TaskID | --task | Number, String | &#x2714; | Task ID |
  | Text | --text | String | &#x2714; | Comment text |
  | Token | --token | String | &#x2716; | Access token, when app auth method used |

```bsl title="1C:Enterprise/OneScript code example"
    URL    = FunctionParameters["Bitrix24_URL"];
    TaskID = FunctionParameters["Bitrix24_CommentsTaskID"];

    Text = "Task impossible, let's split up";

    Result = OPI_Bitrix24.AddTaskComment(URL, TaskID, Text);

    URL   = FunctionParameters["Bitrix24_Domain"];
    Token = FunctionParameters["Bitrix24_Token"];

    Text = "Task uninpossible, don't split up";

    Result = OPI_Bitrix24.AddTaskComment(URL, TaskID, Text, Token);
```
