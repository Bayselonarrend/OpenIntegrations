# Create channel
 Creates a new channel



`Function CreateChannel(Val Token, Val Name, Val Private = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Bot token |
  | Name | --title | String | &#x2714; | Channel name |
  | Private | --private | Boolean | &#x2716; | Create channel as private |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Slack

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Token = "xoxb-6965308400114-696804637...";
    Name  = "testconv" + String(New UUID);

    Result = OPI_Slack.CreateChannel(Token, Name);
```
