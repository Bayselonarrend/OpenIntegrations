# Make file public
 Creates a public URL for the file. Requires user token



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function MakeFilePublic(Val Token, Val FileID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | User token |
  | FileID | --fileid | String | &#x2714; | File identifier |
