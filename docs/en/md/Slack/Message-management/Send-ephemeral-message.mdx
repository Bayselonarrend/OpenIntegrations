# Send ephemeral message
Sends a message that arrives in the channel but is visible only to a specific user



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function SendEphemeralMessage(Val Token, Val Channel, Val Text = "", Val User = "", Val Blocks = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Bot token |
  | Channel | --channel | String | &#x2714; | Channel ID |
  | Text | --text | String | &#x2716; | Message text |
  | User | --user | String | &#x2716; | User ID |
  | Blocks | --blocks | Array of Structure | &#x2716; | Array of block descriptions |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 Map Of KeyAndValue - serialized JSON response from Slack
  </div>
</div>
  
  </TabItem>

  | Parameter | Description |
  |---|---|
  | proxy | InternetProxy or a structure with fields Protocol, Host, Port, User, Password, UseOSAuthentication |
  | timeout | Request execution timeout |

  | dontwait | Creates a background job and returns its object (for 1C and OneScript only) |

```bsl title="1C:Enterprise/OneScript code example"
    Token   = "xoxb-6965308400114-696804637...";
    Channel = "C06UFNUTKUL";
    User    = "U06UABH3APP";
    Image   = "https://hut.openintegrations.dev/test_data/picture.jpg";
    Text    = "Ephemeral message";

    Block  = OPI_Slack.GenerateImageBlock(Image, "Image");
    Result = OPI_Slack.SendEphemeralMessage(Token, Channel, Text, User, Block);
```
