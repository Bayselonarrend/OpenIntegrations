# Add external file
 Adds a new external file



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function AddExternalFile(Val Token, Val URL, Val Title) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Bot token |
  | URL | --url | String | &#x2714; | URL to external file |
  | Title | --title | String | &#x2714; | File title for Slack |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 Map Of KeyAndValue - serialized JSON response from Slack
  </div>
</div>
  
  </TabItem>

  | dontwait | Creates a background job and returns its object (for 1C and OneScript only) |

    Token = "xoxb-6965308400114-696804637...";
