# Update role
 Modifies the data of an existing role



`Function UpdateRole(Val Connection, Val Name, Val Base = Undefined, Val PrivilegesArray = Undefined, Val RoleArray = Undefined, Val Parameters = Undefined) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Connection | --dbc | String, Arbitrary | &#x2714; | Connection or connection string |
  | Name | --name | String | &#x2714; | Role name |
  | Base | --db | String | &#x2716; | Database name. Current database if not specified |
  | PrivilegesArray | --prvl | Array of Structure | &#x2716; | Array of privileges. See GetRolePrivilegeStructure |
  | RoleArray | --roles | Array Of Arbitrary | &#x2716; | Array of roles based on strings or structures specifying the database |
  | Parameters | --params | Structure Of KeyAndValue | &#x2716; | Additional deletion parameters |

  
  Returns:  Map Of KeyAndValue - Operation result

<br/>

:::tip
The passed roles and privileges completely replace the current role data

Role can be defined in one of the following formats:<br/>Structure: `{ role: "name", db: "database" }`<br/>String: `name рoлand`
:::
