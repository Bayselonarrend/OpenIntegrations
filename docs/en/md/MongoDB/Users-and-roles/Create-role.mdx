# Create role
 Creates a new role



`Function CreateRole(Val Connection, Val Name, Val Base = Undefined, Val PrivilegesArray = Undefined, Val RoleArray = Undefined, Val Parameters = Undefined) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Connection | --dbc | String, Arbitrary | &#x2714; | Connection or connection string |
  | Name | --name | String | &#x2714; | Role name |
  | Base | --db | String | &#x2716; | Database name. Current database if not specified |
  | PrivilegesArray | --prvl | Array of Structure | &#x2716; | Array of role privileges |
  | RoleArray | --roles | Array Of Arbitrary | &#x2716; | Array of roles based on strings or structures specifying the database |
  | Parameters | --params | Structure Of KeyAndValue | &#x2716; | Additional deletion parameters |

  
  Returns:  Map Of KeyAndValue - Operation result

<br/>

:::tip
Role can be defined in one of the following formats:<br/>Structure: `{ role: "&lt;name&gt;", db: "&lt;database&gt;" }`<br/>String: `name рoлand`

Privilege can be defined in one of the following formats:<br/>Structure for collection: `{ db: &lt;database&gt;, collection: &lt;collection name&gt; }`

Structure for toлаwithтера: `{ cluster : true }`
:::
