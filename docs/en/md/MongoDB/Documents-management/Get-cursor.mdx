# Get cursor
 Gets a cursor for batch retrieval of collection documents



`Function GetCursor(Val Connection, Val Collection, Val Base = Undefined, Val Filter = Undefined, Val Sort = Undefined, Val Parameters = Undefined) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Connection | --dbc | String, Arbitrary | &#x2714; | Connection or connection string |
  | Collection | --coll | String | &#x2714; | Collection name |
  | Base | --db | String | &#x2716; | Database name. Current database if not specified |
  | Filter | --filter | Structure Of KeyAndValue | &#x2716; | Document filter |
  | Sort | --sort | Structure Of KeyAndValue | &#x2716; | Selection sorting |
  | Parameters | --params | Structure Of KeyAndValue | &#x2716; | Additional retrieval parameters |

  
  Returns:  Map Of KeyAndValue - Operation result
