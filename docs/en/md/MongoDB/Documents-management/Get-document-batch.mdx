# Get document batch
 Gets the next batch of cursor documents



`Function GetDocumentBatch(Val Connection, Val Collection, Val CursorID, Val Base = Undefined, Val BatchSize = Undefined, Val Waiting = Undefined) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Connection | --dbc | String, Arbitrary | &#x2714; | Connection or connection string |
  | Collection | --coll | String | &#x2714; | Collection name |
  | CursorID | --cursor | Number | &#x2714; | Cursor ID |
  | Base | --db | String | &#x2716; | Database name. Current database if not specified |
  | BatchSize | --limit | Number | &#x2716; | Amount of records in the batch |
  | Waiting | --time | Number | &#x2716; | Maximum lock time during awaitData (in ms.) |

  
  Returns:  Map Of KeyAndValue - Operation result
