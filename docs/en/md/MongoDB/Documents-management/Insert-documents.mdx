# Insert documents
 Inserts new documents into the collection



`Function InsertDocuments(Val Connection, Val Collection, Val Values, Val Base = Undefined, Val Parameters = Undefined) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Connection | --dbc | String, Arbitrary | &#x2714; | Connection or connection string |
  | Collection | --coll | String | &#x2714; | Collection name |
  | Values | --docs | Array Of Arbitrary | &#x2714; | Array of docs objects |
  | Base | --db | String | &#x2716; | Database name. Current database if not specified |
  | Parameters | --params | Structure Of KeyAndValue | &#x2716; | Additional insert options |

  
  Returns:  Map Of KeyAndValue - Operation result

<br/>
