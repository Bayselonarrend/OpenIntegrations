# Add calendar to list
 Adds an existing calendar to the user's list



`Function AddCalendarToList(Val Token, Val Calendar) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Calendar | --calendar | String | &#x2714; | Calendar ID |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google



```bsl title="1C:Enterprise/OneScript code example"
    Token    = "ya29.a0Aa7pCA_GKuTstHik6bj1MJWZFLq6Rw5mQEZ6UpvNp9rgAbC-k5fpzhJPU_c4cEvVXwDo6GAXMYcBuPpzfZoPNP2pKbXWwL-bwnzotcuHaeGL9ZoSxPM95kRPm...";
    Calendar = "820a0b504e49dc31b520578e608186d90ce12d730bb5b46b48f372df1a63cd0b@group.calendar.google.com";

    Result = OPI_GoogleCalendar.AddCalendarToList(Token, Calendar);
```
