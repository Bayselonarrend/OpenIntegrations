# Delete calendar
 Deletes a calendar by ID



`Function DeleteCalendar(Val Token, Val Calendar) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Calendar | --calendar | String | &#x2714; | Calendar ID |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google

```bsl title="1C:Enterprise/OneScript code example"
    Calendar = "12c69698653bd6057b445328a998a2ed2db9dcfa0b2b9633324726f3c2399af9@group.calendar.google.com";

    Result = OPI_GoogleCalendar.DeleteCalendar(Token, Calendar);
```
