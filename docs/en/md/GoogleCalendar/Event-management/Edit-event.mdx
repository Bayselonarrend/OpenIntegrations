# Edit event
 Edits an existing event



`Function EditEvent(Val Token, Val Calendar, Val EventDescription, Val Event) Export`

<Tabs>
  <TabItem value="params" label="Parameters" default>

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Calendar | --calendar | String | &#x2714; | Calendar ID |
  | EventDescription | --props | String | &#x2714; | New event description |
  | Event | --event | String | &#x2714; | Event ID |

  
  </TabItem>
  <TabItem value="extended" label="Advanced call">

*This method has no additional advanced call parameters.*


  </TabItem>
</Tabs>
  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google

```bsl title="1C:Enterprise/OneScript code example"
    Calendar    = "f70efec034ef36b040f12b5d8938e54fc399d73cd37b8e765bdd86ebc4a67e4a@group.calendar.google.com";
    Event       = "l6kkhcjgt4n0a3qdonan09tv30";

    EventDescription = New Map;
    EventDescription.Insert("Description", Description);

    Result = OPI_GoogleCalendar.EditEvent(Token, Calendar, EventDescription, Event);
```
