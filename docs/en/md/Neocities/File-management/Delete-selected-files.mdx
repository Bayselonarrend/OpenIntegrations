# Delete selected files
 Deletes a file or multiple files from Neocities



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function DeleteSelectedFiles(Val Token, Val Paths) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Auth token |
  | Paths | --paths | String, Array of String | &#x2714; | Path or multiple paths of the files to be deleted |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 Map Of KeyAndValue - serialized JSON response from Neocities
  </div>
</div>
  
  </TabItem>

  | proxy | `InternetProxy` or a structure with fields `Protocol`, `Host`, `Port`, `User`, `Password`, `UseOSAuthentication` |

  | adv_response | Formats the response as a complete HTTP structure with fields `code`, `body`, and `headers` |

```bsl title="1C:Enterprise/OneScript code example"
    Token = "f1e616e20ab5893b...";

    Paths = New Array;
    Paths.Add("/test/pic1.png");
    Paths.Add("/test/gif.gif");
    Paths.Add("/pic2.png");

    Result = OPI_Neocities.DeleteSelectedFiles(Token, Paths);
```
