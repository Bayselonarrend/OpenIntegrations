# Get token
 Receives authorization token by login and password



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetToken(Val Login, Val Password) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Login | --login | String | &#x2714; | Users login |
  | Password | --password | String | &#x2714; | Users password |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 Map Of KeyAndValue - serialized JSON response from Neocities
  </div>
</div>
