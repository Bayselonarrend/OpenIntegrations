# Update file
 Updates file binary data



`Function UpdateFile(Val Token, Val Identifier, Val File, Val NewName = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Identifier | --object | String | &#x2714; | Identifier of the object to update |
  | File | --file | BinaryData,String | &#x2714; | File source for update |
  | NewName | --title | String | &#x2716; | New file name (if necessary) |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google

<br/>

:::tip
Parameters with Binary data type can also accept file paths on disk and URLs
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    Token      = "ya29.a0AQQ_BDQFeGE6sXaw7ceggeFv33b1k1QXPownfQX5VxQHa49L-lDOFphPG0xij3vtOLSerlEKGn_Qf-ohstVkCUcuNOpUaAC5ofYxuyXHNg4xZPjNH4y-U4lKX...";
    NewName    = "UpdatedFile.jpg";
    Identifier = "1RdcKPec-6rnIWHjq98vtgEDfjEEEYE2o";
    File       = "https://hut.openintegrations.dev/test_data/picture2.jpg"; // URL, Binary Data or File path

    Result = OPI_GoogleDrive.UpdateFile(Token, Identifier, File, NewName);
```
