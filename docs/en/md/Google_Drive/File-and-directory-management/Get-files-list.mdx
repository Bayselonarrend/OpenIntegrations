# Get list of files
 Gets the list of files



`Function GetFilesList(Val Token, Val NameContains = "", Val Directory = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | NameContains | --querry | String | &#x2716; | Filter by name |
  | Directory | --catalog | String | &#x2716; | Filter by parent directory ID |

  
  Returns:  Map Of KeyAndValue - Array of file mappings

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Token        = "ya29.a0AZYkNZiCaf3vW8I05cl_i6pLKCRWSjff1YhC7Yi0SgLO50zysjwrX_9QPi1WejmqgGdFjcbaOJKsPO3tvT7gB712mg1gxgxiL1pCUVthx...";
    Directory    = "root";
    NameContains = "data";

    Result = OPI_GoogleDrive.GetFilesList(Token, NameContains, Directory);
```
