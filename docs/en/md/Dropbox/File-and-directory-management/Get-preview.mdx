# Get preview
 Gets PDF or HTML preview of the object (for documents only)



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetPreview(Val Token, Val Path) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Path | --path | String | &#x2714; | Path to the object |
