# Get preview
 Gets PDF or HTML preview of the object (for documents only)



`Function GetPreview(Val Token, Val Path) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Path | --path | String | &#x2714; | Path to the object |

  
  Returns:  BinaryData - document preview

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Token = "sl.CAA7fvu74rYlykq37bYxZ1FdcMkoc31cOuQ4OJGxbuaMQ2ba2MmJBZ2k-Q-B0_3-wwPCWd9UGi...";
    Path  = "/New/mydoc.docx";

    Result = OPI_Dropbox.GetPreview(Token, Path);
```
