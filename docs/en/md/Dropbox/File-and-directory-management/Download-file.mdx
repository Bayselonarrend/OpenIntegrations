# Download file
 Downloads a file by the specified path or ID



`Function DownloadFile(Val Token, Val Path) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Path | --path | String | &#x2714; | Path or ID of the file |

  
  Returns:  BinaryData - binary data of the file

<br/>




```bsl title="Code example"
    Token = "sl.B-uquz3utwEHepKzyqLBfnFvmY1EWTYDus5LDjT5ux5srp9PJRtr7CvMv20nVl2rRGC3K4J_X5...";
    Path  = "/New/pic.png";

    Result = OPI_Dropbox.DownloadFile(Token, Path);
```
