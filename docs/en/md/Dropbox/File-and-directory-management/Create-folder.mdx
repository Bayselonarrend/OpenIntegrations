# Create folder
 Creates an empty directory at the selected path



`Function CreateFolder(Val Token, Val Path) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Path | --path | String | &#x2714; | Target path for creating the directory |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Dropbox

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Token = "sl.CBw46cIJLPk0Xg3MVRbs5VgDgSZozffBD6ltE5NDB8X7jDaZlIrHZmPEx60hZbiIUDVhfEjuNc...";
    Path  = "/New catalog";

    Result = OPI_Dropbox.CreateFolder(Token, Path);
```
