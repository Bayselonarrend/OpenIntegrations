# Delete object
 Deletes an object from the cloud drive



`Function DeleteObject(Val Token, Val Path, Val Irrecoverable = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Path | --path | String | &#x2714; | Path to the object to delete |
  | Irrecoverable | --permanently | String | &#x2716; | Delete object without the possibility of recovery |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Dropbox

<br/>



```bsl title="1C:Enterprise/OneScript code example"
    Path  = "/New/pic.png";
    Token = FunctionParameters["Dropbox_Token"];

    Result = OPI_Dropbox.DeleteObject(Token, Path);
```
