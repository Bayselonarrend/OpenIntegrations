# Upload file
 Uploads a file to the cloud drive



`Function UploadFile(Val Token, Val File, Val Path, Val Overwrite = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | File | --file | String, BinaryData | &#x2714; | Data file for upload |
  | Path | --path | String | &#x2714; | Save path on Dropbox |
  | Overwrite | --overwrite | Boolean | &#x2716; | Overwrite file in case of path conflicts |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Dropbox

<br/>

:::tip
Parameters with Binary data type can also accept file paths on disk and URLs
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    Path  = "/New/pic.png";
    Token = FunctionParameters["Dropbox_Token"];
    Image = FunctionParameters["Picture"];

    ImagePath = GetTempFileName("png");
    CopyFile(Image, ImagePath);

    Result = OPI_Dropbox.UploadFile(Token, ImagePath, Path, True);
```
