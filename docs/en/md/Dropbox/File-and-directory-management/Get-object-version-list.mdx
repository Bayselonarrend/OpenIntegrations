# Get list of object versions
 Gets the list of versions (revisions) of the object



`Function GetObjectVersionList(Val Token, Val Path, Val Count = 10) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Path | --path | String | &#x2714; | Path to the object |
  | Count | --amount | String, Number | &#x2716; | Number of the latest versions of the object to display |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Dropbox
