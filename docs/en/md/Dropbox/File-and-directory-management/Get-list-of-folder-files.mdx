# Get list of folder files
 Gets the list of the first files in the directory or continues getting the next ones when the cursor is specified



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetListOfFolderFiles(Val Token, Val Path = "", Val Detailed = False, Val Cursor = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Path | --path | String | &#x2716; | Path to the directory. Optional if the cursor is specified |
  | Detailed | --detail | Boolean | &#x2716; | Add information fields for mediafiles |
  | Cursor | --cursor | String | &#x2716; | Cursor from the previous request to get the next set of files |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 HTTPResponse - Get list of folder files
