# Copy object
 Copies a file or directory to the selected path



`Function CopyObject(Val Token, Val From, Val Target) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | From | --from | String | &#x2714; | Path to the original object |
  | Target | --to | String | &#x2714; | Target path for the new object |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Dropbox

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Original = "/New/pic.png";
    Copy     = "/New/pic_copy.png";
    Token    = "sl.B_P83AtWK3iWmiaVVtO7xtmH-oAB1JFAuYM0O4cp24jpCem-7yD8_3nHrlvjtoWiI7T4kRh7Yx...";

    Result = OPI_Dropbox.CopyObject(Token, Original, Copy);
```
