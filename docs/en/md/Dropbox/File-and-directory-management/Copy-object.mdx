# Copy object
 Copies a file or directory to the selected path



`Function CopyObject(Val Token, Val From, Val Target) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | From | --from | String | &#x2714; | Path to the original object |
  | Target | --to | String | &#x2714; | Target path for the new object |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Dropbox
