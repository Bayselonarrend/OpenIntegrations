# Add users to file
 Defines access to the file for third-party users



`Function AddUsersToFile(Val Token, Val FileID, Val EmailAddresses, Val ViewOnly = True) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | FileID | --fileid | String | &#x2714; | ID of the file to be accessed |
  | EmailAddresses | --emails | String, Array of String | &#x2714; | List of email addresses of users being added |
  | ViewOnly | --readonly | Boolean | &#x2716; | Prohibits file editing for the external user |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Dropbox

<br/>

```bsl title="1C:Enterprise/OneScript code example"
    Token = "sl.CBL71_21WiKphqlWV34l0tlFn8eGn_YrnPrMexNmHltofj3DkBRsnpQEFY5O5H6i-iRl3bPDBi...";

    File  = "kJU6-a-pT48AAAAAAAAABw";

    Result = OPI_Dropbox.AddUsersToFile(Token, File, Email, False);
```
