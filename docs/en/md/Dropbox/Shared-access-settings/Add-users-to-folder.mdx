# Add users to folder
 Grants external users access to the directory



`Function AddUsersToFolder(Val Token, Val FolderID, Val EmailAddresses, Val ViewOnly = True) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | FolderID | --folder | String | &#x2714; | ID of the public catalog (shared folder ID) |
  | EmailAddresses | --emails | String, Array of String | &#x2714; | List of email addresses of users being added |
  | ViewOnly | --readonly | Boolean | &#x2716; | Prohibits file editing for the external user |

  
  Returns:  Undefined - empty response
