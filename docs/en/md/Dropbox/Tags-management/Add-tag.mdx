# Add tag
 Add new tag to file or folder



`Function AddTag(Val Token, Val Path, Val Tag) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Path | --path | String | &#x2714; | Path to the object for which the tag needs to be created |
  | Tag | --tag | String | &#x2714; | Tag text |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Dropbox

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Tag   = "Important";
    Token = "sl.CEDGlKWN_ZDFEtmK9tny8Rxbmib5H8Pe02nRPwYlcKt-wgv_vaOyaasY534_erb80ReU6XyQqt...";
    Path  = "/New/mydoc.docx";

    Result = OPI_Dropbox.AddTag(Token, Path, Tag);
```
