# Add tag
 Add new tag to file or folder



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function AddTag(Val Token, Val Path, Val Tag) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Path | --path | String | &#x2714; | Path to the object for which the tag needs to be created |
  | Tag | --tag | String | &#x2714; | Tag text |
