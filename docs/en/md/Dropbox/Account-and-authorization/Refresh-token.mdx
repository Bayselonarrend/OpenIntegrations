# Refresh token
 Gets a new token based on the refresh token



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function RefreshToken(Val AppKey, Val AppSecret, Val RefreshToken) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | AppKey | --appkey | String | &#x2714; | Application key |
  | AppSecret | --appsecret | String | &#x2714; | Application secret |
  | RefreshToken | --refresh | String | &#x2714; | Refresh token |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 Map Of KeyAndValue - serialized JSON response from Dropbox
  </div>
</div>
  
  </TabItem>

  | Parameter | Description |
  |---|---|
  | proxy | InternetProxy or a structure with fields Protocol, Host, Port, User, Password, UseOSAuthentication |
  | timeout | Request execution timeout |
  | adv_response | Formats the response as a complete structure with fields code, body, and headers |
  | retries | Number of send attempts on `5**` status codes or internal client errors |
  | dontwait | Creates a background job and returns its object (for 1C and OneScript only) |

    AppKey       = "oynqxds...";
