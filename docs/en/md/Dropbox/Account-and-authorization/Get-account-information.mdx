# Get account information
 Gets account information



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetAccountInformation(Val Token, Val Account = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | Account | --account | String | &#x2716; | Account ID. Current token account if not filled |
