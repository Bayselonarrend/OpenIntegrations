# Get comment
 Gets comment by ID



`Function GetComment(Val Token, Val ObjectID, Val CommentID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | ObjectID | --object | String | &#x2714; | Identifier of the file or directory where the comment is located |
  | CommentID | --comment | String | &#x2714; | Comment identifier |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from Google

    Result = OPI_GoogleDrive.GetComment(Token, Identifier, CommentID);
