# Get list of directories
 Gets the list of drive directories



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetDirectoriesList(Val Token, Val NameContains = "", Val Detailed = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Token | --token | String | &#x2714; | Token |
  | NameContains | --query | String | &#x2716; | Filter by name |
  | Detailed | --depth | Boolean | &#x2716; | Adds a list of files to the directory fields |
