# Get file description
 Gets a blank description template for uploading a file



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetFileDescription(Val Clear = False) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Clear | --empty | Boolean | &#x2716; | True > structure with empty valuse, False > field descriptions at values |

  
  </TabItem>

  </TabItem>
</Tabs>
  
  Returns:  Map Of KeyAndValue - File description
