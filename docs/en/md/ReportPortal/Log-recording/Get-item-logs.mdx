# Get item logs
 Gets the logs of the selected test item



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetItemLogs(Val URL, Val Token, Val Project, Val ElementID, Val Page = 1) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | ReportPortal server URL |
  | Token | --token | String | &#x2714; | Access token |
  | Project | --proj | String | &#x2714; | Project ID |
  | ElementID | --item | Number | &#x2714; | Test item ID |
  | Page | --page | Number | &#x2716; | List page number |
