# Delete log
 Deletes a log item by ID



`Function DeleteLog(Val URL, Val Token, Val Project, Val LogID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | ReportPortal server URL |
  | Token | --token | String | &#x2714; | Access token |
  | Project | --proj | String | &#x2714; | Project ID |
  | LogID | --id | Number | &#x2714; | Log ID |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from ReportPortal

:::tip
Numeric log ID can be obtained by UUID using the `GetLog` function
:::
