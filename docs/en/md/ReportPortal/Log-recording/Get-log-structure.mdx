# Get log structure
 Gets the parameter structure for log entry



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetLogStructure(Val LaunchID, Val ElementID, Val Time, Val Text = "", Val Level = "info") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | LaunchID | --launch | String | &#x2714; | Launch ID to which the test item belongs |
  | ElementID | --id | String | &#x2714; | Test item ID |
  | Time | --time | Date | &#x2714; | Log entry date and time |
  | Text | --text | String | &#x2716; | Log text information |
  | Level | --level | String | &#x2716; | Information level: error, warn, info, debug, trace, fatal, unknown |
