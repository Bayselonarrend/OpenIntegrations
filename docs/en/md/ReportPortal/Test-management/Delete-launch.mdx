# Delete launch
 Deletes a test session by ID



`Function DeleteLaunch(Val URL, Val Token, Val Project, Val LaunchID) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | ReportPortal server URL |
  | Token | --token | String | &#x2714; | Access token |
  | Project | --proj | String | &#x2714; | Project ID |
  | LaunchID | --id | Number | &#x2714; | Launch ID |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from ReportPortal

:::tip
Numeric launch ID can be obtained by UUID using the `GetLaunch` function
:::
<br/>
