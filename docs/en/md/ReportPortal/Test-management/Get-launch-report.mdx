# Get launch report
 Gets report for the selected test session



`Function GetLaunchReport(Val URL, Val Token, Val Project, Val LaunchID, Val Format = "pdf") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | ReportPortal server URL |
  | Token | --token | String | &#x2714; | Access token |
  | Project | --proj | String | &#x2714; | Project ID |
  | LaunchID | --id | Number | &#x2714; | Launch ID |
  | Format | --format | String | &#x2716; | Report format: pdf, xls, html |

  
  Returns:  BinaryData - report file

:::tip
Numeric launch ID can be obtained by UUID using the `GetLaunch` function
:::
<br/>

    LaunchID = "";
