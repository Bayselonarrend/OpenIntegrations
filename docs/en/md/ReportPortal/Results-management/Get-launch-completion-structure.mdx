# Get launch completion structure
 Gets the parameter structure for completing a test session



`Function GetLaunchCompletionStructure(Val Time, Val Status = "", Val Description = "", Val Attributes = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Time | --end | Date | &#x2714; | Launch end date and time |
  | Status | --status | String | &#x2716; | Completion status: passed, failed, stopped, skipped, interrupted, cancelled |
  | Description | --descr | String | &#x2716; | Launch description. Overwrites the one specified during creation |
  | Attributes | --attr | String | &#x2716; | Launch attributes. Overwrites the one specified during creation |

  
  Returns:  Structure Of KeyAndValue - Fields structure
