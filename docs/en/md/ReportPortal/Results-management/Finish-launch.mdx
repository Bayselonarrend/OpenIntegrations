# Finish launch
 Ends a previously started test execution session



`Function FinishLaunch(Val URL, Val Token, Val Project, Val LaunchID, Val FinishStructure) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | ReportPortal server URL |
  | Token | --token | String | &#x2714; | Access token |
  | Project | --proj | String | &#x2714; | Project ID |
  | LaunchID | --id | String | &#x2714; | Launch ID |
  | FinishStructure | --params | Structure Of KeyAndValue | &#x2714; | Completion parameters. See GetLaunchCompletionStructure |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from ReportPortal
