# Create launch
 Starts a new test execution session



`Function CreateLaunch(Val URL, Val Token, Val Project, Val LaunchStructure) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | URL | --url | String | &#x2714; | ReportPortal server URL |
  | Token | --token | String | &#x2714; | Access token |
  | Project | --proj | String | &#x2714; | Project ID |
  | LaunchStructure | --params | Structure Of KeyAndValue | &#x2714; | Launch parameters. See GetLaunchStructure |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from ReportPortal
