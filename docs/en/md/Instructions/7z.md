# 7z

import LibraryIntro from '@site/src/components/LibraryIntro';

<LibraryIntro module="OPI_7z" cli="7z" use="oint/formats/7z" lang="en"/>

The `OPI_7z` library is for **creating, unpacking and modifying** 7z archives in 1C:Enterprise, OneScript and CLI. This page gives a short overview and typical workflows; see child sections for full method signatures and examples.

## Capabilities

| Task | Method |
|------|--------|
| Pack a directory into 7z | [`ArchiveDirectory`](/docs/7z/Archiving/Archive-directory) |
| Unpack the whole archive to a directory | [`UnarchiveDirectory`](/docs/7z/Archiving/Unarchive-directory) |
| Extract selected files only | [`UnpackFiles`](/docs/7z/Archiving/Unpack-files) |
| Add, replace or delete files | [`ModifyArchive`](/docs/7z/Archiving/Modify-archive) |
| Get archiving settings template | [`GetArchivingSettingsStructure`](/docs/7z/Archiving/Get-archiving-settings-structure) |
| Get archive modification settings template | [`GetArchiveModificationStructure`](/docs/7z/Archiving/Get-archive-modification-structure) |
| Get a tree of files and folders | [`GetFilesList`](/docs/7z/Getting-metadata/Get-files-list) |
| Get metadata and a flat file list | [`GetMetadata`](/docs/7z/Getting-metadata/Get-metadata) |

The archive and source directory can be passed as a **file/folder path on disk** or **binary data**. For password-protected archives, use the `Password` parameter (CLI: `--password`).

## Typical workflow

A common flow:

1. Pack a directory with the desired compression settings тАФ `ArchiveDirectory`.
2. Inspect the archive тАФ `GetFilesList` or `GetMetadata` (with password when the archive is protected).
3. Unpack everything тАФ `UnarchiveDirectory`, or only chosen paths тАФ `UnpackFiles`.

```bsl
SourceDirectory = "C:\Data\project";
ArchivePath     = "C:\Archives\project.7z";

// 1. Create a password-protected archive
Settings = OPI_7z.GetArchivingSettingsStructure(True);
Settings["password"] = "SecretPassword";
Settings["method"]   = "Lzma2";
Settings["level"]    = 9;

Result = OPI_7z.ArchiveDirectory(SourceDirectory, ArchivePath, Settings);

// 2. Inspect contents
FileList = OPI_7z.GetFilesList(ArchivePath, "SecretPassword");
```

## Archiving

`ArchiveDirectory` packs files from the specified directory. The second argument is the save path; if omitted, the archive is returned as `BinaryData`.

```bsl
SourceDirectory = "C:\MyFolder";
ArchivePath     = "C:\Archives\archive.7z";

Result = OPI_7z.ArchiveDirectory(SourceDirectory, ArchivePath);
```

### Compression and encryption settings

`GetArchivingSettingsStructure` returns a structure of optional parameters:

- **password** тАФ password for archive protection
- **method** тАФ compression method: `Lzma2` (default), `Lzma`, `Bzip2`, `Ppmd`, `Copy`
- **level** тАФ compression level from 0 to 9 (default 6)
- **solid** тАФ solid archiving (single-stream compression): `True`/`False` (default `True`)
- **encrypt_header** тАФ header encryption: `True` (default when password is set) / `False`
- **threads** тАФ number of LZMA2 threads (default 1)
- **chunk_size** тАФ chunk size when `threads > 1`
- **dictionary_size** тАФ LZMA2 dictionary size
- **ppmd_order**, **ppmd_memory** тАФ Ppmd method parameters
- **filters** тАФ preprocessing filters: `X86`, `PPC`, `IA64`, `ARM`, `ARM64`, `ARM_THUMB`, `SPARC`, `RISCV`, `DELTA`
- **delta_distance** тАФ distance for the DELTA filter (default 1)

```bsl
Settings = OPI_7z.GetArchivingSettingsStructure(True);
Settings["password"]       = "SecretPassword";
Settings["method"]         = "Lzma2";
Settings["level"]          = 9;
Settings["encrypt_header"] = True;
Settings["threads"]        = 4;

Result = OPI_7z.ArchiveDirectory(SourceDirectory, ArchivePath, Settings);
```

:::tip
For maximum compression, use `Lzma2` with level `9`. For faster archive creation, lower the level or use the `Copy` method.
:::

:::important
When `encrypt_header = True`, file names in the archive are also encrypted and are not visible without the password.
:::

## Unpacking

### Full archive

`UnarchiveDirectory` extracts all entries into the target directory. Pass the password as the third argument when needed.

```bsl
ArchivePath          = "C:\Archives\archive.7z";
DestinationDirectory   = "C:\Unpack";

Result = OPI_7z.UnarchiveDirectory(ArchivePath, DestinationDirectory);

// Password-protected archive
Result = OPI_7z.UnarchiveDirectory(ArchivePath, DestinationDirectory, "MyPassword123");
```

### Selected files

`UnpackFiles` takes an array of **full paths inside the archive** тАФ the same strings you get from listing methods.

```bsl
Paths = New Array;
Paths.Add("readme.txt");
Paths.Add("docs\note.txt");

Result = OPI_7z.UnpackFiles(ArchivePath, Paths, DestinationDirectory, "MyPassword123");
```

:::tip
Before partial unpack, call `GetFilesList` or `GetMetadata` and reuse paths from the response.
:::

## Modifying archives

`ModifyArchive` adds, replaces or deletes files in an existing archive. An on-disk archive is modified **in place**. Unpack and pack passwords go in `Settings` тАФ see [`GetArchiveModificationStructure`](/docs/7z/Archiving/Get-archive-modification-structure).

```bsl
ArchivePath = "C:\Archives\archive.7z";

Additions = New Map;
Additions.Insert("docs/new.txt", "C:\Files\new.txt");

Deletions = New Array;
Deletions.Add("old/obsolete.txt");

Settings = OPI_7z.GetArchiveModificationStructure(True);
Settings["unpack_password"] = "OldPassword";
Settings["password"]          = "NewPassword";

Result = OPI_7z.ModifyArchive(ArchivePath, Additions, Deletions, Settings);
```

## Inspecting contents

**`GetFilesList`** тАФ hierarchical tree of folders and files.

**`GetMetadata`** тАФ archive summary and a **flat** file list with attributes.

```bsl
Password = "MyPassword123";

Tree     = OPI_7z.GetFilesList(ArchivePath, Password);
Metadata = OPI_7z.GetMetadata(ArchivePath, Password);
```

## File path vs binary data

The `Archive` and directory arguments accept a disk path or `BinaryData`.

```bsl
BinaryData = New BinaryData("C:\Archives\archive.7z");

FileList = OPI_7z.GetFilesList(BinaryData, "MyPassword123");
Result   = OPI_7z.UnarchiveDirectory(BinaryData, "C:\Unpack", "MyPassword123");
```

If you **omit** the destination directory when unpacking, the result is a map of paths to binary data. When archiving without a save path, you get `BinaryData` of the archive.

```bsl
// Archive contents as a map, without writing to disk
Description = OPI_7z.UnarchiveDirectory(BinaryData, "", "MyPassword123");

// Archive in memory, without writing to disk
ArchiveInMemory = OPI_7z.ArchiveDirectory(SourceDirectory, "", Settings);
```

:::tip
For large archives, prefer a file path on disk: lower memory use.
:::
