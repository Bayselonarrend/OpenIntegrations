# ReportPortal

This section covers the library for working with the ReportPortal API in 1C:Enterprise, OneScript, and CLI. This page describes all the steps required to get started.

## Getting Started

1. Log in to your ReportPortal instance with your credentials.
2. Navigate to your *Profile* or *User Settings* section.
3. Generate or locate your **API access token**.
4. Copy and securely store the generated token.

The received token should be used when calling library functions or configuring integration settings.

Refer to the documentation of your particular version of ReportPortal for details on permissions and API usage policies.
