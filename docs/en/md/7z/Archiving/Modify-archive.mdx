# Modify archive
 Adds, replaces and deletes files in an existing 7z archive



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function ModifyArchive(Val Archive, Val AddableFiles = Undefined, Val DeletablePaths = Undefined, Val Settings = Undefined) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Archive | --src | String, BinaryData | &#x2714; | Path to the archive on disk or binary data |
  | AddableFiles | --additions | Map Of KeyAndValue | &#x2716; | Key: archive path; value: file path or binary data |
  | DeletablePaths | --deletions | Array Of String | &#x2716; | Paths of files in the archive for deletion |
  | Settings | --settings | Map Of KeyAndValue | &#x2716; | Modification settings. See GetArchiveModificationStructure |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 BinaryData, Map Of KeyAndValue - Result of the operation for the archive on the disk or new binary data of the archive

  </div>
</div>
  
  </TabItem>

  | Parameter | Description |
  |---|---|
  | dontwait | Creates a background job and returns its data (for 1C and OneScript only) |


  </TabItem>
