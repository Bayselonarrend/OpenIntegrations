# Remove product from selection
 Removes a previously added product from the selection



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function RemoveProductFromSelection(Val Product, Val Selection, Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Product | --item | String | &#x2714; | Product ID |
  | Selection | --sel | String | &#x2714; | Selection ID |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization parameters. See GetAuthParameters |
