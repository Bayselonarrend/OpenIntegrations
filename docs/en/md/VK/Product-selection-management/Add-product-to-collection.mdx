# Add product to selection
 Adds a product to the selection



`Function AddProductToCollection(Val ProductsArray, Val Selection, Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | ProductsArray | --items | Array of string, number | &#x2714; | Array of products or product |
  | Selection | --sel | String | &#x2714; | Selection ID |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization JSON or path to .json |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from VK

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Parameters = GetVKParameters();

    Product   = "11831701";
    Selection = "142";

    Result = OPI_VK.AddProductToCollection(Product, Selection, Parameters);
```
