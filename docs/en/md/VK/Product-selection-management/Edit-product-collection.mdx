# Edit product selection
 Edits the properties of a product selection



`Function EditProductCollection(Val Name, Val Selection, Val Image = "", Val Main = False, Val Hidden = False, Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Name | --title | String | &#x2714; | New selection name |
  | Selection | --sel | String | &#x2714; | Selection ID |
  | Image | --picture | String, BinaryData | &#x2716; | New selection image |
  | Main | --main | Boolean | &#x2716; | Main |
  | Hidden | --hidden | Boolean | &#x2716; | Hidden |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization JSON or path to .json |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from VK

<br/>

:::tip
Parameters with Binary data type can also accept file paths on disk and URLs
:::
<br/>


```bsl title="Code example"
    Parameters = GetVKParameters();

    Name      = "EditedCollection";
    Selection = "130";

    Result = OPI_VK.EditProductCollection(Name, Selection, , , , Parameters);

    OPI_TestDataRetrieval.Check_VKTrue(Result);

    OPI_Tools.Pause(5);
```
