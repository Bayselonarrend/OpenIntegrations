# Get selection list
 Gets the list of product selections



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetSelectionList(Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization parameters. See GetAuthParameters |

  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
