# Create token retrieval link
Getting a link for interactive token retrieval (access_token), which is necessary for further actions



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function CreateTokenRetrievalLink(Val App_id) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | app_id | --app | String, Number | &#x2714; | app_id from application settings |

  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 String - URL to go to in the browser
