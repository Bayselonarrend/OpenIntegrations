# Add product property variant
 Adds a variant for an existing property



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function AddProductPropertyVariant(Val Value, Val Property, Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Value | --value | String | &#x2714; | Property value |
  | Property | --prop | String, Number | &#x2714; | Property ID where the variant is added |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization parameters. See GetAuthParameters |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 Map Of KeyAndValue - serialized JSON response from VK
  </div>
</div>

```bsl title="1C:Enterprise/OneScript code example"
    Parameters = GetVKParameters();

    Property    = "4862";
    OptionArray = New Array;
    OptionArray.Add("Yellow");
    OptionArray.Add("Blue");
    OptionArray.Add("Red");

    For Each Value In OptionArray Do

        Result = OPI_VK.AddProductPropertyVariant(Value, Property, Parameters);

    EndDo;
```
