# Edit product property variant
 Edits the value of an existing product property variant



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function EditProductPropertyVariant(Val Value, Val Property, Val Option, Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Value | --value | String | &#x2714; | New property value |
  | Property | --prop | String, Number | &#x2714; | Property ID |
  | Option | --option | String, Number | &#x2714; | Variant ID |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization parameters. See GetAuthParameters |
