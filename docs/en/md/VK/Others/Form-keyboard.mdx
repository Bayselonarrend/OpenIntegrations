# Create keyboard
 Forms a keyboard from an array of buttons



`Function FormKeyboard(Val ButtonArray) Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | ButtonArray | --buttons | Array of String | &#x2714; | Array of button titles |

  
  Returns:  String - Keyboard JSON



```bsl title="1C:Enterprise/OneScript code example"
    ButtonArray = New Array;
    ButtonArray.Add("Button 1");
    ButtonArray.Add("Button 2");

    Result = OPI_VK.FormKeyboard(ButtonArray);
```
