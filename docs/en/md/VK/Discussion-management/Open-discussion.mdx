# Open discussion
 Opens a previously closed discussion



`Function OpenDiscussion(Val DiscussionID, Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | DiscussionID | --topic | String, Number | &#x2714; | Discussion ID |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization JSON or path to .json |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from VK

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Parameters   = GetVKParameters();
    DiscussionID = "52626758";
    Result       = OPI_VK.OpenDiscussion(DiscussionID, Parameters);
```
 



```json title="Result"
{
 "response": 1
}
```
