# Write in discussion
 Adds a message to the discussion on behalf of the group



`Function WriteInDiscussion(Val DiscussionID, Val Text, Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | DiscussionID | --topic | String, Number | &#x2714; | Discussion ID |
  | Text | --text | String | &#x2714; | Message text |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization JSON or path to .json |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from VK

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Parameters   = GetVKParameters();
    DiscussionID = "54106885";
    Message      = "I like yellow more";

    Result = OPI_VK.WriteInDiscussion(DiscussionID, Message, Parameters);
```
