# Delete image
 Deletes an image from the album



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function DeleteImage(Val ImageID, Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | ImageID | --pictureid | String, Number | &#x2714; | Image ID |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization parameters. See GetAuthParameters |
