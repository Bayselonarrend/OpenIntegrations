# Upload video to server
 Uploads video to the group for further use



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function UploadVideoToServer(Val Video, Val Name, Val Description = "", Val Album = "", Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Video | --file | String, BinaryData | &#x2714; | Video file |
  | Name | --title | String | &#x2714; | Video name |
  | Description | --description | String | &#x2716; | Video description |
  | Album | --album | String | &#x2716; | Album ID, if necessary |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization parameters. See GetAuthParameters |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 Map Of KeyAndValue - serialized JSON response from VK
  </div>
</div>
  
  </TabItem>
  <TabItem value="extended" label={<span>Advanced call{' '}<a href="/docs/Start/Advanced-call" target="_blank" rel="noreferrer" title="About advanced call" onClick={(e) => e.stopPropagation()}>?</a></span>}>

  | Parameter | Description |
  |---|---|
  | dontwait | Creates a background job and returns its object (for 1C and OneScript only) |


  </TabItem>
</Tabs>

:::tip
Parameters with Binary data type can also accept file paths on disk and URLs
:::
<br/>


```bsl title="1C:Enterprise/OneScript code example"
    Parameters = GetVKParameters();

    Video       = "https://hut.openintegrations.dev/test_data/video.mp4"; // URL, Binary data or File path
    Name        = "NewVideo";
    Description = "Video description";

    Result = OPI_VK.UploadVideoToServer(Video, Name, Description, , Parameters);
```
