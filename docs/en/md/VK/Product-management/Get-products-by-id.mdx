# Get products by ID
 Gets information about products by array of IDs



`Function GetProductsByID(Val Products, Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Products | --items | String, Array of String | &#x2714; | Array of product IDs |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization JSON or path to .json |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from VK

<br/>



```bsl title="1C:Enterprise/OneScript code example"
    Parameters = GetVKParameters();

    Item1 = "12055345";
    Item2 = "12055347";

    ProductsArray = New Array;
    ProductsArray.Add(Item1);
    ProductsArray.Add(Item2);

    Result = OPI_VK.GetProductsByID(ProductsArray, Parameters);
```
