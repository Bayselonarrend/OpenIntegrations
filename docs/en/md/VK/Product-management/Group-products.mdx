# Group products
 Groups products based on similar sets of properties



`Function GroupProducts(Val ProductsArray, Val ExistingGroup = "", Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | ProductsArray | --items | Array Of String | &#x2714; | Array of product IDs |
  | ExistingGroup | --sellgroup | String | &#x2716; | ID of the existing group, if needed |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization parameters. See GetAuthParameters |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from VK



```bsl title="1C:Enterprise/OneScript code example"
    Parameters = GetVKParameters();

    Item1 = "12369885";
    Item2 = "12369886";

    ProductsArray = New Array;
    ProductsArray.Add(Item1);
    ProductsArray.Add(Item2);

    Result = OPI_VK.GroupProducts(ProductsArray, , Parameters);
```
