# Get product list
 Gets the community's product list



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function GetProductList(Val Selection = "", Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Selection | --sel | String, Number | &#x2716; | Selection ID, if filtering is needed |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization parameters. See GetAuthParameters |

  <div className="return-value-note__title">Returns</div>
