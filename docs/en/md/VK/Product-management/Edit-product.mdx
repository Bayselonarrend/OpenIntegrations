# Edit product
 Edits a previously created product



<Tabs>
  <TabItem value="params" label="Parameters" default>

`Function EditProduct(Val Product, Val ProductDescription, Val Selection = "", Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | Product | --item | Number, String | &#x2714; | Identifier of the product being edited |
  | ProductDescription | --product | Map Of KeyAndValue | &#x2714; | Product description. See GetProductDescription |
  | Selection | --sel | String | &#x2716; | Identifier of the new selection, if needed |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization parameters. See GetAuthParameters |


<div className="return-value-note">
  <div className="return-value-note__title">Returns</div>
  <div className="return-value-note__value">
 Map Of KeyAndValue - serialized JSON response from VK
  </div>
</div>
  
  </TabItem>

  | Parameter | Description |
  |---|---|
  | proxy | InternetProxy or a structure with fields Protocol, Host, Port, User, Password, UseOSAuthentication |
  | timeout | Request execution timeout |
  | adv_response | Formats the response as a complete structure with fields code, body, and headers |

    Parameters = GetVKParameters();

    Product = "12115889";
