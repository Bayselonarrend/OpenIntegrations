# Like
 Likes a post



`Function LikePost(Val PostID, Val WallID = "", Val Parameters = "") Export`

  | Parameter | CLI option | Type | Required | Description |
  |-|-|-|-|-|
  | PostID | --post | String, Number | &#x2714; | Post ID |
  | WallID | --wall | String, Number | &#x2716; | ID of the wall where the post is located |
  | Parameters | --auth | Structure Of String | &#x2716; | Authorization JSON or path to .json |

  
  Returns:  Map Of KeyAndValue - serialized JSON response from VK

<br/>




```bsl title="1C:Enterprise/OneScript code example"
    Parameters = GetVKParameters();
    PostID     = "4822";

    Result = OPI_VK.LikePost(PostID, , Parameters);
```
 



```json title="Result"
{
 "response": {
  "likes": 1
 }
}
```
